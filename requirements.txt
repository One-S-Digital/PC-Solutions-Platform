# PC Solutions Platform v2.0 - System Requirements
# NOTE: Deployment platforms like Render treat this as a Python requirements file.
# Keep the contents comment-only so pip install succeeds without trying to fetch
# non-Python tooling such as Node.js or pnpm.

# Node.js Runtime
# Node.js >= 18.0.0

# Package Manager
# pnpm >= 9.0.0

# Database
# PostgreSQL >= 14.0

# Docker (for containerization)
# Docker >= 20.0.0
# Docker Compose >= 2.0.0

# Development Tools
# Git >= 2.30.0

# Optional: For production deployment
# Nginx >= 1.20.0

# Environment Variables Required:
# - DATABASE_URL
# - CLERK_SECRET_KEY
# - STRIPE_SECRET_KEY
# - CLOUDFLARE_R2_ACCESS_KEY_ID
# - CLOUDFLARE_R2_SECRET_ACCESS_KEY
# - CLOUDFLARE_R2_BUCKET_NAME
# - CLOUDFLARE_R2_ENDPOINT
# - JWT_SECRET
# - MAILTRAP_API_TOKEN (for development)
# - SMTP_HOST, SMTP_PORT, SMTP_USER, SMTP_PASS (for production)

# Installation Instructions:
# 1. Install Node.js 18+ from https://nodejs.org/
# 2. Install pnpm: npm install -g pnpm
# 3. Install PostgreSQL 14+ from https://www.postgresql.org/
# 4. Install Docker from https://www.docker.com/
# 5. Clone repository and run: pnpm install
# 6. Set up environment variables as documented in ENVIRONMENT_SETUP.md
# 7. Run database migrations: cd build && pnpm prisma migrate dev
# 8. Start development: pnpm dev
