# Supported Browsers

Browser compatibility information for the ProCrèche Solutions platform.

---

## Recommended Browsers

The platform is designed to work with modern web browsers. We recommend:

### Desktop

- **Chrome** (latest version)
- **Firefox** (latest version)
- **Safari** (latest version)
- **Edge** (latest version)

### Mobile

- **Safari** (iOS 14+)
- **Chrome** (Android, latest version)

---

## Minimum Requirements

- JavaScript enabled
- Cookies enabled
- Local storage enabled
- Modern browser with ES6+ support

---

## Known Issues

### Internet Explorer

Internet Explorer is **not supported**. Please use a modern browser.

### Older Browser Versions

Browsers older than 2 years may experience issues. We recommend keeping your browser updated.

---

## Features Requiring Modern Browsers

- **Real-time Messaging** - Requires WebSocket support
- **File Uploads** - Requires File API support
- **Notifications** - Requires Notification API support (optional)

---

## Testing

The platform is tested on:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

---

## Mobile Responsiveness

The platform is responsive and works on:
- Desktop (1920x1080 and above)
- Tablet (768px - 1024px)
- Mobile (320px - 767px)

---

## Notes

- Browser extensions may interfere with platform functionality
- Ad blockers may block some features
- Private/Incognito mode may have limitations with local storage

---

**File Reference:**
- `frontend/vite.config.ts` - Build configuration
- Browser compatibility is handled by Vite and React build tools
