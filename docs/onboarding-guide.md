# Organization & Vendor Onboarding Guide

This document details the onboarding process and profile management for organization and vendor roles.

## 1. Common Onboarding Flow

The signup process for all organization and vendor roles follows a consistent multi-step flow:

1.  **Role Selection:** The user first selects their role (Foundation, Product Supplier, or Service Provider).
2.  **Information Entry:** The user provides information relevant to their role in a dynamic form.
3.  **Email Verification:** The user must verify their email address to complete the registration.
4.  **Profile Completion:** After the first login, the user is encouraged to complete their organization's profile in the settings page.

## 2. Foundation (Daycare) Profile

### Signup Form Fields:
- **Organization Name:** The legal name of the daycare.
- **Contact Person:** The name of the primary contact.
- **Email & Password:** For authentication.
- **Phone Number:** The main contact number for the daycare.
- **Canton:** The Swiss canton where the daycare is located.
- **Languages Spoken:** A multi-select field for the languages offered.
- **Capacity:** The total number of children the daycare can accommodate.

### Profile Fields (in Settings):
- All fields from the signup form.
- **Logo Upload:** For the daycare's logo.
- **Cover Image Upload:** For a banner image on their profile page.
- **"About" Text:** A detailed description of the daycare.
- **Pedagogy:** A multi-select field or tag input for the educational approach (e.g., Montessori, Reggio Emilia).
- **VAT Number:** The organization's VAT identification number.

## 3. Product Supplier Profile

### Signup Form Fields:
- **Organization Name:** The name of the supplier company.
- **Contact Person:** The name of the primary contact.
- **Email & Password:** For authentication.
- **Phone Number:** The main contact number for the supplier.
- **Canton:** The primary canton of operation.
- **Product Category:** The main category of products they supply (e.g., "Toys", "Furniture").

### Profile Fields (in Settings):
- All fields from the signup form.
- **Logo Upload:** For the company logo.
- **Direct Order Link:** A URL to their external ordering system, if available.
- **Catalog Upload:** An optional field to upload a PDF product catalog.
- **Minimum Order Quantity:** A numeric field for the minimum order size.

## 4. Service Provider Profile

### Signup Form Fields:
- **Organization Name:** The name of the service provider company.
- **Contact Person:** The name of the primary contact.
- **Email & Password:** For authentication.
- **Phone Number:** The main contact number for the provider.
- **Canton:** The primary canton of operation.
- **Service Type:** The main type of service they provide (e.g., "Cleaning", "IT Support").

### Profile Fields (in Settings):
- All fields from the signup form.
- **Logo Upload:** For the company logo.
- **Service Categories:** A multi-select field for all the categories of services they offer.
- **Delivery Type:** A choice between "On-site", "Remote", or "Hybrid".
- **Booking Link:** A URL to their external booking system, if available.
