# Unterstützte Browser

Dieses Dokument listet die unterstützten Webbrowser und Anforderungen für die Nutzung von ProCrèche Solutions auf.

---

## Empfohlene Browser

Für die beste Erfahrung verwenden Sie einen dieser Browser in der neuesten Version:

| Browser | Mindestversion | Empfohlen |
|---------|----------------|-----------|
| 🌐 **Google Chrome** | 90+ | ✅ Ja |
| 🦊 **Mozilla Firefox** | 88+ | ✅ Ja |
| 🧭 **Apple Safari** | 14+ | ✅ Ja |
| 🔷 **Microsoft Edge** | 90+ | ✅ Ja |

## Nicht unterstützte Browser

Diese Browser werden **nicht** unterstützt:

| Browser | Grund |
|---------|-------|
| ❌ Internet Explorer | Veraltet, fehlende Funktionen |
| ❌ Opera Mini | Eingeschränkte Funktionen |
| ❌ Alte Browser | Sicherheits- und Kompatibilitätsprobleme |

## Technische Anforderungen

### JavaScript

- ✅ JavaScript muss **aktiviert** sein
- Die Plattform funktioniert ohne JavaScript nicht

### Cookies

- ✅ Cookies müssen **erlaubt** sein
- Erforderlich für Anmeldung und Einstellungen

### Bildschirmauflösung

| Gerät | Mindestauflösung |
|-------|------------------|
| Computer | 1024 x 768 px |
| Tablet | 768 x 1024 px |
| Smartphone | 320 x 568 px |

### Internetverbindung

- Stabile Verbindung empfohlen
- Mindestgeschwindigkeit: 1 Mbps

## So aktualisieren Sie Ihren Browser

### Google Chrome

1. Klicken Sie auf **⋮** (Menü)
2. **Hilfe** → **Über Google Chrome**
3. Update erfolgt automatisch

### Mozilla Firefox

1. Klicken Sie auf **☰** (Menü)
2. **Hilfe** → **Über Firefox**
3. Klicken Sie auf **"Aktualisieren"**, falls verfügbar

### Apple Safari

1. Öffnen Sie **Systemeinstellungen** auf dem Mac
2. **Softwareupdate**
3. macOS-Updates installieren

### Microsoft Edge

1. Klicken Sie auf **⋯** (Menü)
2. **Hilfe und Feedback** → **Über Microsoft Edge**
3. Update erfolgt automatisch

## Fehlerbehebung bei Browser-Problemen

### Seite lädt nicht korrekt

1. Seite aktualisieren (F5 oder Strg+R)
2. Browser-Cache leeren
3. Anderen Browser versuchen
4. Internetverbindung überprüfen

### Anzeigeprobleme

1. Seitenzoom überprüfen (100% empfohlen)
2. Cache leeren
3. Erweiterungen deaktivieren, die stören könnten

### Anmeldeprobleme

1. Prüfen, ob Cookies aktiviert sind
2. Cookies der Website löschen
3. Im Inkognito-/Privatmodus versuchen

### Problematische Erweiterungen

Einige Erweiterungen können Probleme verursachen:
- Werbeblocker
- Strenge Datenschutz-Erweiterungen
- VPNs oder Proxys

Versuchen Sie, diese vorübergehend zu deaktivieren, wenn Sie Probleme haben.
