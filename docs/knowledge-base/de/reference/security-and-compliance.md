## DSGVO-Konformität

### Was ist die DSGVO?

Die **Datenschutz-Grundverordnung** (DSGVO) ist eine europäische Verordnung, die die personenbezogenen Daten von Einzelpersonen schützt.

### Unsere Konformität

| Anforderung | Wie wir sie erfüllen |
|-------------|---------------------|
| **Einwilligung** | Ausdrückliche Einwilligung bei der Registrierung |
| **Transparenz** | Klare Datenschutzrichtlinie |
| **Datenzugang** | Sie können Ihre Daten exportieren |
| **Recht auf Vergessenwerden** | Kontolöschung verfügbar |
| **Datenportabilität** | Export von Daten im Standardformat |
| **Schutz** | Technische und organisatorische Maßnahmen |

### Ihre DSGVO-Rechte

Als Benutzer haben Sie das Recht:

1. Auf Ihre personenbezogenen Daten **zuzugreifen**
2. Unrichtige Informationen zu **berichtigen**
3. Ihre Daten **zu löschen** (Recht auf Vergessenwerden)
4. Ihre Daten zu einem anderen Dienst zu **übertragen**
5. Die Verarbeitung Ihrer Daten **einzuschränken**
6. Bestimmten Verarbeitungen zu **widersprechen**
7. Im Falle einer Datenschutzverletzung **informiert zu werden**

## Datenverschlüsselung

### Bei der Übertragung

| Protokoll | Beschreibung |
|-----------|--------------|
| **TLS 1.3** | Neueste Version des Sicherheitsprotokolls |
| **HTTPS** | Alle Verbindungen sind verschlüsselt |
| **HSTS** | Schutz vor Downgrade-Angriffen |

### Im Ruhezustand

| Maßnahme | Beschreibung |
|----------|--------------|
| **AES-256** | Starke Verschlüsselung der gespeicherten Daten |
| **Hashing** | Passwörter gehasht (nicht umkehrbar) |
| **Isolation** | Trennung der Daten zwischen Benutzern |

### Zahlungen

| Maßnahme | Beschreibung |
|----------|--------------|
| **PCI-DSS** | Konformität mit Zahlungsstandards |
| **Tokenisierung** | Kartendaten werden nie direkt gespeichert |
| **Drittanbieter-Prozessor** | Zahlungen von zertifizierten Partnern verwaltet |

## Ihre Datenschutzrechte

### Auf Ihre Daten zugreifen

1. Gehen Sie zu **Einstellungen** → **Datenschutz und Daten**
2. Klicken Sie auf **"Meine Daten exportieren"**
3. Bestätigen Sie die Anfrage
4. Erhalten Sie einen Download-Link per E-Mail

### Was im Export enthalten ist

- Profilinformationen
- Nachrichten (nur Ihre Nachrichten)
- Aktivitätsverlauf
- Einstellungen und Präferenzen

### Ihre Daten löschen

1. Gehen Sie zu **Einstellungen** → **Datenschutz und Daten**
2. Klicken Sie auf **"Mein Konto löschen"**
3. Bestätigen Sie durch Eingabe von "LÖSCHEN"
4. Ihr Konto wird innerhalb von 30 Tagen gelöscht

> ⚠️ **Achtung:** Die Löschung ist nach 30 Tagen unwiderruflich.

### Nach der Löschung aufbewahrte Daten

Einige Daten können aus rechtlichen Gründen aufbewahrt werden:
- Rechnungen und Transaktionen (steuerliche Verpflichtungen)
- Sicherheitsprotokolle (Betrugserkennung)
- Nachrichten (anonymisiert, für Gesprächskontinuität)

## Tipps zur Kontosicherheit

### Sicheres Passwort

| Beste Praxis | Erklärung |
|--------------|-----------|
| ✅ **8+ Zeichen** | Je länger, desto sicherer |
| ✅ **Typen mischen** | Groß-, Kleinbuchstaben, Zahlen, Symbole |
| ✅ **Einzigartig** | Passwörter nicht wiederverwenden |
| ✅ **Manager** | Passwort-Manager verwenden |
| ❌ **Persönliche Infos** | Namen, Geburtsdaten vermeiden |
| ❌ **Wörterbuchbegriffe** | Einfache Wörter vermeiden |

### Ihr Konto schützen

| Aktion | Warum |
|--------|-------|
| 🔐 **Einzigartiges Passwort** | Begrenzt Schaden bei Lecks |
| 📧 **Sichere E-Mail** | Schützen Sie auch Ihre E-Mail |
| 🚪 **Abmelden** | Immer auf gemeinsam genutzten Geräten |
| 🔔 **Benachrichtigungen** | Sicherheitswarnungen aktivieren |
| 🔄 **Updates** | Browser und OS aktuell halten |

### Phishing erkennen

**Vorsicht bei verdächtigen E-Mails:**
- Passwortanfragen per E-Mail (nie legitim)
- Links zu ähnlichen, aber anderen Websites
- Übermäßige Dringlichkeit ("Ihr Konto wird geschlossen")
- Rechtschreibfehler und seltsames Layout

**Im Zweifelsfall:**
1. Nicht auf Links klicken
2. Die Website direkt über Ihren Browser aufrufen
3. Bei Bedarf den Support kontaktieren

## Ein Sicherheitsproblem melden

### Entdeckte Schwachstelle

Wenn Sie eine Sicherheitslücke entdecken:

1. **Nicht ausnutzen**
2. **Uns sofort kontaktieren** über den Support
3. "Sicherheit" als Kategorie wählen
4. Die Schwachstelle detailliert beschreiben
5. Wir werden schnell antworten

### Kompromittiertes Konto

Wenn Sie glauben, dass Ihr Konto kompromittiert wurde:

1. **Sofort Passwort ändern**
2. Profilinformationen überprüfen
3. Letzte Aktivitäten überprüfen
4. Support kontaktieren
5. Auch das E-Mail-Passwort ändern

## Häufig gestellte Fragen

**F: Werden meine Daten an Dritte verkauft?**
A: Nein, niemals. Ihre Daten werden nicht verkauft oder zu Werbezwecken geteilt.

**F: Wo werden meine Daten gehostet?**
A: Unsere Server befinden sich in Europa, DSGVO-konform.

**F: Wie lange werden meine Daten aufbewahrt?**
A: Solange Ihr Konto aktiv ist, plus 30 Tage nach Löschung.

**F: Kann ich eine Kopie meiner Daten anfordern?**
A: Ja, über **Einstellungen** → **Datenschutz** → **Exportieren**.

**F: Was passiert bei einer Datenschutzverletzung?**
A: Wir benachrichtigen Sie innerhalb von 72 Stunden gemäß DSGVO.
