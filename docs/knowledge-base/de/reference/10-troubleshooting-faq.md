## Kontoprobleme

### Ich erhalte keine Verifizierungs-E-Mail

**Mögliche Ursachen:**
- E-Mail ist in Ihrem Spam-Ordner
- E-Mail-Adresse falsch eingegeben
- Vorübergehendes Serverproblem

**Lösungen:**
1. Überprüfen Sie Ihren **Spam-/Junk-Ordner**
2. Warten Sie 30 Sekunden und klicken Sie auf **"Code erneut senden"**
3. Überprüfen Sie die Schreibweise Ihrer E-Mail-Adresse
4. Versuchen Sie es mit einer anderen E-Mail-Adresse
5. Kontaktieren Sie den Support, wenn das Problem weiterhin besteht

---

### Mein Verifizierungscode funktioniert nicht

**Mögliche Ursachen:**
- Code abgelaufen (gültig ~10 Minuten)
- Code falsch eingegeben
- Sie haben einen neuen Code angefordert

**Lösungen:**
1. Klicken Sie auf **"Code erneut senden"**, um einen neuen Code zu erhalten
2. Verwenden Sie den neuesten Code
3. Überprüfen Sie, ob Sie den richtigen Code eingeben (6 Ziffern)
4. Kopieren Sie den Code aus der E-Mail, falls möglich

---

### Ich kann mich nicht anmelden

**Mögliche Ursachen:**
- Falsches Passwort
- Falsche E-Mail
- Konto nicht verifiziert

**Lösungen:**
1. Überprüfen Sie, ob die E-Mail-Adresse korrekt ist
2. Verwenden Sie **"Passwort vergessen"** zum Zurücksetzen
3. Überprüfen Sie, ob Sie die E-Mail-Verifizierung abgeschlossen haben
4. Versuchen Sie, den Browser-Cache zu leeren
5. Versuchen Sie einen anderen Browser

---

### Ich habe mein Passwort vergessen

**Lösung:**
1. Gehen Sie zur Anmeldeseite
2. Klicken Sie auf **"Passwort vergessen"**
3. Geben Sie Ihre E-Mail-Adresse ein
4. Überprüfen Sie Ihren Posteingang
5. Klicken Sie auf den Zurücksetzungslink
6. Erstellen Sie ein neues Passwort
7. Melden Sie sich mit dem neuen Passwort an

---

### Ich möchte meine E-Mail-Adresse ändern

**Schritte:**
1. Melden Sie sich bei Ihrem Konto an
2. Gehen Sie zu **Einstellungen** → **Kontosicherheit**
3. Finden Sie **"E-Mail-Adresse ändern"**
4. Geben Sie Ihre neue E-Mail-Adresse ein
5. Ein Verifizierungscode wird gesendet
6. Geben Sie den Code zur Bestätigung ein

---

### Ich möchte mein Konto löschen

**Schritte:**
1. Gehen Sie zu **Einstellungen** → **Datenschutz und Daten**
2. Klicken Sie auf **"Mein Konto löschen"**
3. Lesen Sie die Informationen darüber, was gelöscht wird
4. Bestätigen Sie Ihre Entscheidung

> ⚠️ **Achtung:** Die Löschung ist nach 30 Tagen endgültig und unwiderruflich.

## Abrechnungsprobleme

### Meine Karte wurde abgelehnt

**Mögliche Ursachen:**
- Unzureichende Deckung
- Karte abgelaufen
- Transaktionslimit erreicht
- Blockierung durch die Bank

**Lösungen:**
1. Überprüfen Sie Ihren Kontostand
2. Überprüfen Sie das Ablaufdatum der Karte
3. Kontaktieren Sie Ihre Bank zur Freischaltung
4. Versuchen Sie eine andere Karte
5. Kontaktieren Sie den Support für alternative Optionen

---

### Ich habe meine Rechnung nicht erhalten

**Lösungen:**
1. Überprüfen Sie Ihren Spam-Ordner
2. Gehen Sie zu **Einstellungen** → **Abrechnung** → **Verlauf**
3. Laden Sie die Rechnung von Ihrem Konto herunter
4. Kontaktieren Sie den Support, wenn sie nicht angezeigt wird

---

### Ich möchte eine Erstattung

**Prozess:**
1. Erstellen Sie ein Support-Ticket
2. Wählen Sie "Abrechnung" als Kategorie
3. Erklären Sie Ihre Situation
4. Das Team wird Ihre Anfrage prüfen
5. Sie erhalten innerhalb von 2-3 Tagen eine Antwort

**Erstattungsrichtlinie:**
- Erstattungen werden von Fall zu Fall bewertet
- Anteilige Erstattung möglich
- Keine automatische Erstattung

---

### Mein Abonnement ist nicht aktiv

**Mögliche Ursachen:**
- Zahlung fehlgeschlagen
- Abonnement abgelaufen
- Verarbeitung läuft

**Lösungen:**
1. Überprüfen Sie **Einstellungen** → **Abrechnung**
2. Überprüfen Sie den Zahlungsstatus
3. Aktualisieren Sie Ihre Zahlungsmethode
4. Warten Sie 24 Stunden auf die Verarbeitung
5. Kontaktieren Sie den Support, wenn das Problem weiterhin besteht

## Funktionsspezifische Fragen

### Stiftungen

#### Eltern sehen meine Kita nicht

**Mögliche Ursachen:**
- Unvollständiges Profil
- Kein aktives Abonnement
- Profil verborgen

**Lösungen:**
1. Vervollständigen Sie Ihr Profil zu 100%
2. Überprüfen Sie Ihr Abonnement
3. Überprüfen Sie in **Einstellungen** → **Datenschutz**, ob Sie sichtbar sind

---

#### Ich erhalte keine Eltern-Leads

**Mögliche Ursachen:**
- Abonnement ohne Leads
- Unattraktives Profil
- Wenig aktive Region

**Lösungen:**
1. Überprüfen Sie, ob Ihr Plan Leads beinhaltet
2. Verbessern Sie Ihr Profil (Fotos, Beschreibung)
3. Erweitern Sie Ihre bedienten Regionen

---

### Lieferanten/Dienstleister

#### Meine Produkte/Dienstleistungen werden nicht angezeigt

**Mögliche Ursachen:**
- Status "Inaktiv" oder "Entwurf"
- Kein aktives Abonnement
- Profil verborgen

**Lösungen:**
1. Überprüfen Sie den Status jedes Produkts/jeder Dienstleistung
2. Stellen Sie sicher, dass sie "Aktiv" sind
3. Überprüfen Sie Ihr Abonnement

---

### Erzieher

#### Mein Profil erscheint nicht in den Suchen

**Mögliche Ursachen:**
- Unvollständiges Profil
- Profil verborgen
- Verfügbarkeit nicht festgelegt

**Lösungen:**
1. Vervollständigen Sie Ihr Profil zu 100%
2. Überprüfen Sie **Einstellungen** → **Datenschutz**
3. Legen Sie Ihre Verfügbarkeit fest
4. Fügen Sie ein professionelles Foto hinzu

---

### Eltern

#### Ich finde keine Kitas in meiner Region

**Lösungen:**
1. Erweitern Sie Ihre Suchzone
2. Reduzieren Sie Ihre Filterkriterien
3. Überprüfen Sie, ob Kitas in Ihrer Region registriert sind

---

## Technische Probleme

### Die Seite lädt nicht

**Lösungen:**
1. Seite aktualisieren (F5 oder Strg+R)
2. Browser-Cache leeren
3. Anderen Browser versuchen
4. Internetverbindung überprüfen
5. Ein paar Minuten warten und erneut versuchen

---

### Die Website ist langsam

**Mögliche Ursachen:**
- Langsame Internetverbindung
- Browser-Cache voll
- Server überlastet

**Lösungen:**
1. Browser-Cache leeren
2. Internetverbindung überprüfen
3. Andere Tabs/Anwendungen schließen
4. Zu einem anderen Zeitpunkt versuchen

---

### Ich sehe einen Fehler

**Was zu tun ist:**
1. Fehlermeldung notieren
2. Screenshot machen
3. Seite aktualisieren
4. Wenn das Problem weiterhin besteht, Support-Ticket erstellen

**Nützliche Informationen:**
- Genaue Fehlermeldung
- Seite, auf der der Fehler aufgetreten ist
- Aktionen vor dem Fehler
- Browser und Betriebssystem

---

### Bilder werden nicht angezeigt

**Lösungen:**
1. Seite aktualisieren
2. Browser-Cache leeren
3. Verbindung überprüfen
4. Anderen Browser versuchen

---

### Datei-Upload schlägt fehl

**Mögliche Ursachen:**
- Datei zu groß
- Format nicht unterstützt
- Instabile Verbindung

**Lösungen:**
1. Dateigröße überprüfen (max 10 MB)
2. Format überprüfen (PDF, JPG, PNG, etc.)
3. Datei bei Bedarf komprimieren
4. Stabile Verbindung verwenden
5. Erneut versuchen

## Browser-Cache leeren

### Chrome

1. Klicken Sie auf ⋮ (Menü)
2. Einstellungen → Datenschutz → Browserdaten löschen
3. Wählen Sie "Bilder und Dateien im Cache"
4. Klicken Sie auf "Daten löschen"

### Firefox

1. Klicken Sie auf ☰ (Menü)
2. Einstellungen → Datenschutz & Sicherheit
3. Cookies und Website-Daten → Daten entfernen
4. "Zwischengespeicherte Webinhalte" auswählen
5. Auf "Leeren" klicken

### Safari

1. Safari → Einstellungen → Erweitert
2. "Entwickler-Menü in der Menüleiste anzeigen" aktivieren
3. Entwickler → Caches leeren

### Edge

1. Klicken Sie auf ⋯ (Menü)
2. Einstellungen → Datenschutz → Browserdaten löschen
3. "Zwischengespeicherte Bilder und Dateien" auswählen
4. Auf "Jetzt löschen" klicken

## Support kontaktieren

### Ticket erstellen

1. Klicken Sie auf **Support** in der Seitenleiste
2. Klicken Sie auf **"Ticket erstellen"**
3. Ausfüllen:
   - **Betreff** - Kurze Problembeschreibung
   - **Kategorie** - Allgemein, Technisch, Abrechnung
   - **Priorität** - Niedrig, Mittel, Hoch, Dringend
   - **Beschreibung** - Detaillierte Erklärung
4. Screenshots anhängen, falls hilfreich
5. Auf **"Absenden"** klicken

### Nützliche Informationen

- Ihre Rolle auf der Plattform
- Schritte zur Reproduktion des Problems
- Fehlermeldung (falls vorhanden)
- Browser und Version
- Betriebssystem
- Screenshots

### Antwortzeiten

| Priorität | Antwortzeit |
|-----------|-------------|
| Dringend | < 4 Stunden |
| Hoch | < 24 Stunden |
| Mittel | 1-2 Tage |
| Niedrig | 2-5 Tage |

## Allgemeine FAQ

### Konto

**F: Kann ich mehrere Konten haben?**
A: Nein, nur ein Konto pro E-Mail-Adresse ist erlaubt.

**F: Kann ich meine Rolle ändern?**
A: Kontaktieren Sie den Support, um eine Rollenänderung zu besprechen.

**F: Sind meine Daten sicher?**
A: Ja, wir verwenden Verschlüsselung und folgen Sicherheitsstandards. Siehe unsere Datenschutzrichtlinie.

---

### Nutzung

**F: Ist die Plattform auf Mobilgeräten verfügbar?**
A: Ja, die Website ist responsive und funktioniert auf allen Geräten.

**F: Gibt es eine mobile App?**
A: Derzeit nicht, aber die mobile Website bietet ein ähnliches Erlebnis.

**F: In welchen Sprachen ist die Plattform verfügbar?**
A: Englisch, Französisch und Deutsch.
