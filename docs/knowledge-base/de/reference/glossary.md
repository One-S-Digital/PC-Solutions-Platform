# Glossar

Dieses Glossar definiert häufig verwendete Begriffe auf ProCrèche Solutions.

---

## Allgemeine Begriffe

| Begriff | Definition |
|---------|------------|
| **Dashboard** | Personalisierte Startseite mit wichtigen Metriken und aktuellen Aktivitäten |
| **Profil** | Öffentliche Informationen über einen Benutzer oder eine Organisation |
| **Lead** | Anfrage eines Elternteils an eine Kita |
| **Marktplatz** | Bereich, in dem Stiftungen Produkte/Dienstleistungen finden und bestellen können |
| **Abonnement** | Kostenpflichtiger Plan, der Zugang zu Premium-Funktionen ermöglicht |
| **Kanton** | Verwaltungseinheit der Schweiz |

---

## Benutzerrollen

| Rolle | Beschreibung |
|-------|--------------|
| **Stiftung** | Kinderbetreuungsorganisation (Kita, Krippe, Kindergarten) |
| **Produktlieferant** | Unternehmen, das Produkte an Kitas verkauft |
| **Dienstleister** | Unternehmen, das Dienstleistungen für Kitas anbietet |
| **Erzieher** | Fachkraft der frühkindlichen Bildung auf Jobsuche |
| **Eltern** | Person, die Kinderbetreuung für ihr Kind sucht |
| **Administrator** | Benutzer mit Plattformverwaltungsrechten |

---

## Stiftungs-Begriffe

| Begriff | Definition |
|---------|------------|
| **Kapazität** | Maximale Anzahl der betreuten Kinder |
| **Pädagogik** | Pädagogischer Ansatz (Montessori, Reggio Emilia, etc.) |
| **Eltern-Lead** | Platzierungsanfrage eines Elternteils |
| **Stellenangebot** | Offene Position zur Besetzung |
| **Bewerbung** | Einreichung eines Erziehers für eine Stelle |
| **E-Learning** | Online-Schulungen für das Personal |
| **HR-Verfahren** | Personaldokumente und Vorlagen |
| **Kantonale Richtlinien** | Kantonsspezifische Vorschriften |

---

## Lieferanten-/Dienstleister-Begriffe

| Begriff | Definition |
|---------|------------|
| **Produkt** | Physischer Artikel, der an Kitas verkauft wird |
| **Dienstleistung** | Service, der für Kitas angeboten wird |
| **Bestellung** | Produktkauf durch eine Stiftung |
| **Serviceanfrage** | Dienstleistungsanfrage durch eine Stiftung |
| **Anfrage** | Frage einer Stiftung zu Produkten/Dienstleistungen |
| **Promo-Code** | Rabattcode für Kunden |
| **SKU** | Interne Produktreferenz |
| **Katalog** | Vollständige Liste der Produkte/Dienstleistungen |
| **Buchungslink** | URL zur Terminvereinbarung |

---

## Status

### Allgemeine Status

| Status | Beschreibung |
|--------|--------------|
| **Aktiv** | Element sichtbar und betriebsbereit |
| **Inaktiv** | Element verborgen oder deaktiviert |
| **Entwurf** | Element nicht veröffentlicht, in Erstellung |
| **Ausstehend** | Wartet auf Aktion oder Verarbeitung |
| **Abgeschlossen** | Erfolgreich beendet |
| **Storniert** | Vom Benutzer abgebrochen |

### Bestellstatus

| Status | Beschreibung |
|--------|--------------|
| **Eingereicht** | Bestellung gesendet, wartet auf Bestätigung |
| **Akzeptiert** | Vom Lieferanten bestätigt |
| **In Bearbeitung** | Wird vorbereitet |
| **Versendet** | An den Kunden gesendet |
| **Geliefert** | Vom Kunden erhalten |

### Bewerbungsstatus

| Status | Beschreibung |
|--------|--------------|
| **Eingereicht** | Bewerbung gesendet |
| **Angesehen** | Vom Arbeitgeber gesehen |
| **In Prüfung** | Wird bewertet |
| **Vorstellungsgespräch** | Zum Gespräch eingeladen |
| **Akzeptiert** | Angebot gemacht oder Stelle erhalten |
| **Abgelehnt** | Nicht in Betracht gezogen |

### Lead-Status

| Status | Beschreibung |
|--------|--------------|
| **Neu** | Gerade erhaltener Lead |
| **In Bearbeitung** | Wird bearbeitet |
| **Kontaktiert** | Antwort an Elternteil gesendet |
| **Konvertiert** | Kind angemeldet |
| **Geschlossen** | Lead ohne Konversion beendet |

## Abrechnungsbegriffe

| Begriff | Definition |
|---------|------------|
| **Plan** | Abonnementstufe (Basic, Essential, Professional, Enterprise) |
| **Abrechnungszyklus** | Zeitraum zwischen zwei Zahlungen (monatlich/jährlich) |
| **Verlängerung** | Automatische Verlängerung des Abonnements |
| **Anteilig** | Proportionale Berechnung für Teilzeiträume |
| **Rechnung** | Dokument mit Zahlungsdetails |
| **Zahlungsmethode** | Mittel zur Zahlung (Karte, Überweisung) |

---

## Technische Begriffe

| Begriff | Definition |
|---------|------------|
| **Cache** | Temporär gespeicherte Daten zur Beschleunigung des Ladens |
| **Cookie** | Kleine vom Browser gespeicherte Datei |
| **DSGVO** | Datenschutz-Grundverordnung |
| **Authentifizierung** | Überprüfung der Benutzeridentität |
| **E-Mail-Verifizierung** | Bestätigung des Zugriffs auf eine E-Mail-Adresse |
| **OAuth** | Anmeldung über einen Drittanbieterdienst (Google) |
| **API** | Programmierschnittstelle für Integrationen |

---

## Nachrichtenbegriffe

| Begriff | Definition |
|---------|------------|
| **Gespräch** | Nachrichtenverlauf zwischen Benutzern |

| **Anhang** | An eine Nachricht angehängte Datei |
| **Benachrichtigung** | Warnung über ein Ereignis |
