## Kontosicherheit

### Persönliche Informationen

Aktualisieren Sie Ihre Grundinformationen.

#### Verfügbare Felder

| Feld | Beschreibung | Änderbar |
|------|--------------|----------|
| **Vollständiger Name** | Ihr Name | ✅ Ja |
| **Telefonnummer** | Kontakt | ✅ Ja |
| **E-Mail** | Über E-Mail-Änderung | Siehe unten |

#### So ändern

1. Gehen Sie zu **Einstellungen** → **Kontosicherheit**
2. Ändern Sie die gewünschten Felder
3. Klicken Sie auf **"Änderungen speichern"**

---

### Passwort ändern

#### Schritte

1. Geben Sie Ihr **aktuelles Passwort** ein
2. Geben Sie Ihr **neues Passwort** ein (mind. 8 Zeichen)
3. **Bestätigen** Sie das neue Passwort
4. Klicken Sie auf **"Passwort aktualisieren"**

#### Passwortanforderungen

- ✅ Mindestens 8 Zeichen
- ✅ Kein häufig verwendetes Passwort
- ✅ Darf nicht in einem Datenleck kompromittiert sein

---

### E-Mail-Adresse ändern

#### Prozess

1. Geben Sie Ihre **neue E-Mail-Adresse** ein
2. Klicken Sie auf **"Verifizierungscode senden"**
3. Überprüfen Sie Ihre **neue E-Mail** auf den Code
4. Geben Sie den **6-stelligen Code** ein
5. Klicken Sie auf **"Verifizieren und E-Mail ändern"**

> ⚠️ **Wichtig:** Ihre neue Adresse muss verifiziert werden, bevor die Änderung wirksam wird.

## Abrechnung und Abonnement

*Verfügbar für: Stiftungen, Lieferanten, Dienstleister*

### Übersicht

Zeigen Sie Ihren aktuellen Plan an und verwalten Sie die Abrechnung.

#### Angezeigte Informationen

| Element | Beschreibung |
|---------|--------------|
| **Aktueller Plan** | Name und Stufe Ihres Abonnements |
| **Status** | Aktiv, Inaktiv, Ausstehend, Storniert |
| **Preis** | Monatliche oder jährliche Kosten |
| **Nächste Verlängerung** | Datum der nächsten Zahlung |
| **Zahlungsmethode** | Registrierte Karte |

### Verfügbare Aktionen

| Aktion | Beschreibung |
|--------|--------------|
| **Änderung anfordern** | Upgrade oder Downgrade |
| **Zahlung verwalten** | Karte hinzufügen/ändern |
| **Rechnungen anzeigen** | Abrechnungsverlauf |
| **Kündigung anfordern** | Abonnement kündigen |

## Promo-Code-Manager

*Verfügbar für: Lieferanten, Dienstleister*

### Auf Promo-Codes zugreifen

**Einstellungen** → **Promo-Code-Manager**

### Promo-Code erstellen

#### Felder

| Feld | Beschreibung | Erforderlich |
|------|--------------|--------------|
| **Code** | Der Textcode (z.B. PROMO20) | ✅ Ja |
| **Rabatt** | Was der Kunde erhält | ✅ Ja |
| **Beschreibung** | Bedingungen | ❌ Nein |
| **Auf Profil anzeigen** | Öffentliche Sichtbarkeit | ✅ Ja (Standard) |

### Bestehende Codes verwalten

| Aktion | Wie |
|--------|-----|
| **Bearbeiten** | Auf "Bearbeiten" beim Code klicken |
| **Löschen** | Auf "Löschen" klicken |
| **Verbergen** | "Auf Profil anzeigen" deaktivieren |

## Benachrichtigungseinstellungen

### Benachrichtigungstypen

#### E-Mails

| Einstellung | Beschreibung | Rollen |
|-------------|--------------|--------|
| **Neue Nachrichten** | E-Mail bei neuer Nachricht | Alle |
| **Neue Bestellungen** | E-Mail bei neuen Bestellungen | Lieferanten |
| **Neue Anfragen** | E-Mail bei neuen Anfragen | Dienstleister |
| **Neue Leads** | E-Mail bei neuen Eltern-Leads | Stiftungen |
| **Neue Bewerbungen** | E-Mail bei erhaltenen Bewerbungen | Stiftungen |
| **Bewerbungs-Updates** | E-Mail über Ihre Bewerbungen | Erzieher |
| **Promo-Code-Nutzung** | Wenn Ihre Codes verwendet werden | Lieferanten, Dienstleister |

### Digest-Häufigkeit

Wählen Sie, wie Sie Zusammenfassungen erhalten:

| Option | Beschreibung |
|--------|--------------|
| **Sofort** | Benachrichtigung bei jedem Ereignis |
| **Täglich** | Einmal täglich Zusammenfassung |
| **Wöchentlich** | Einmal wöchentlich Zusammenfassung |
| **Keine** | Kein Digest (nur einzelne Benachrichtigungen) |

### Benachrichtigungen konfigurieren

1. Gehen Sie zu **Einstellungen** → **Benachrichtigungseinstellungen**
2. Aktivieren/deaktivieren Sie jeden Typ mit den Schaltern
3. Wählen Sie die Digest-Häufigkeit
4. Änderungen werden automatisch gespeichert

## Datenschutz und Daten

### Profilsichtbarkeit

Kontrollieren Sie, wer Ihr Profil sehen kann.

| Einstellung | Optionen | Beschreibung |
|-------------|----------|--------------|
| **Öffentlich verbergen** | An/Aus | Verbirgt Ihr Profil aus Suchen |

### Datenverwaltung

#### Ihre Daten exportieren

1. Klicken Sie auf **"Meine Daten exportieren"**
2. Bestätigen Sie die Anfrage
3. Erhalten Sie eine E-Mail mit dem Download-Link
4. Innerhalb von 48 Stunden herunterladen

Exportierte Daten umfassen:
- Profilinformationen
- Nachrichten
- Aktivitätsverlauf
- Einstellungen

#### Ihr Konto löschen (DSGVO)

1. Klicken Sie auf **"Mein Konto löschen"**
2. Lesen Sie die wichtigen Informationen
3. Tippen Sie "LÖSCHEN" zur Bestätigung
4. Endgültig bestätigen

> ⚠️ **Achtung:**
> - Löschung ist nach 30 Tagen endgültig
> - Alle Ihre Daten werden gelöscht
> - Gespräche werden aufbewahrt (Nachrichten anonymisiert)

## Kontakt und Buchung

*Verfügbar für: Lieferanten, Dienstleister*

### Kontaktpräferenzen

| Einstellung | Beschreibung |
|-------------|--------------|
| **Kontakt-E-Mail** | Öffentlich angezeigte E-Mail |
| **Kontakttelefon** | Öffentlich angezeigte Nummer |
| **Kontaktzeiten** | Wann Sie erreichbar sind |

### Buchungslink

Ermöglichen Sie Kunden, direkt Termine zu buchen.

#### Konfiguration

1. Gehen Sie zu **Einstellungen** → **Kontakt und Buchung**
2. Finden Sie **"Buchungslink"**
3. Fügen Sie die URL Ihres Planungstools ein:
   - Calendly
   - Cal.com
   - Microsoft Bookings
   - Andere
4. Fügen Sie eine Beschreibung hinzu (z.B. "Beratung buchen")
5. Speichern

#### So funktioniert es

- Ein "Buchen"-Button erscheint auf Ihrem Profil
- Kunden klicken und werden zu Ihrem Kalender weitergeleitet
- Sie wählen einen verfügbaren Zeitslot
- Sie erhalten eine Buchungsbenachrichtigung

- ✅ Professionelles Profilbild
- ✅ Aktuelle Kontaktinformationen
- ✅ Angemessene Sichtbarkeit

- ✅ Wichtige Benachrichtigungen aktivieren
- ✅ Digest verwenden, um Spam zu vermeiden
- ✅ Irrelevantes deaktivieren
- ✅ Einstellungen regelmäßig überprüfen
