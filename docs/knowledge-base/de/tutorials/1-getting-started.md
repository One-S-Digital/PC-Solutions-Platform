## Plattformübersicht

### Was ist ProCrèche Solutions?

ProCrèche Solutions ist eine All-in-One-Plattform, die das Schweizer Kinderbetreuungs-Ökosystem verbindet:

- **Kindertagesstätten (Stiftungen)** finden Produkte, Dienstleistungen, Personal und verwalten Elternanfragen
- **Produktlieferanten** verkaufen Kinderbetreuungsprodukte an Kitas
- **Dienstleister** bieten professionelle Dienstleistungen für Kitas an
- **Erzieher** finden Stellenangebote in Kindertagesstätten
- **Eltern** finden qualitativ hochwertige Kinderbetreuung für ihre Kinder

### Unterstützte Sprachen

Die Plattform ist in drei Sprachen verfügbar:
- 🇬🇧 **English** (Englisch)
- 🇫🇷 **Français** (Französisch)
- 🇩🇪 **Deutsch**

Sie können die Sprache jederzeit über den Sprachauswähler in der Navigationsleiste ändern.

## Konto erstellen

### Schritt 1: Zur Registrierungsseite navigieren

1. Besuchen Sie die ProCrèche Solutions Website
2. Klicken Sie auf **"Registrieren"** in der oberen Navigationsleiste
3. Sie werden zur Registrierungsseite weitergeleitet

### Schritt 2: Wählen Sie Ihre Rolle

Wählen Sie den Kontotyp, der Ihren Bedürfnissen entspricht:

| Rolle | Für wen? |
|-------|----------|
| 🏫 **Stiftung** | Kindertagesstätten, Kinderbetreuungsorganisationen |
| 📦 **Produktlieferant** | Unternehmen, die Produkte an Kitas verkaufen |
| 🔧 **Dienstleister** | Unternehmen, die Dienstleistungen für Kitas anbieten |
| 👩‍🏫 **Erzieher** | Fachkräfte der frühkindlichen Bildung auf Jobsuche |
| 👨‍👩‍👧 **Eltern** | Eltern, die Kinderbetreuung suchen |

Klicken Sie auf Ihre Rolle, um zum Registrierungsformular zu gelangen.

### Schritt 3: Füllen Sie Ihre Informationen aus

Die erforderlichen Informationen variieren je nach Rolle:

#### Für Stiftungen, Lieferanten und Dienstleister:

| Feld | Beschreibung | Erforderlich |
|------|--------------|--------------|
| Organisationsname | Name Ihres Unternehmens oder Ihrer Kita | ✅ Ja |
| Kontaktperson | Ihr vollständiger Name | ✅ Ja |
| E-Mail | Ihre E-Mail-Adresse (für die Anmeldung) | ✅ Ja |
| Passwort | Erstellen Sie ein sicheres Passwort (mind. 8 Zeichen) | ✅ Ja |
| Passwort bestätigen | Passwort erneut eingeben | ✅ Ja |
| Telefonnummer | Geschäftliche Kontaktnummer | ✅ Ja |
| Kanton/Region | Ihr Standort in der Schweiz | ✅ Ja |
| Kapazität | Anzahl der Kinder (nur Stiftungen) | ✅ Ja |
| Kategorie | Produktkategorie (nur Lieferanten) | ✅ Ja |
| Dienstleistungstyp | Art der Dienstleistung (nur Dienstleister) | ✅ Ja |

#### Für Erzieher:

| Feld | Beschreibung | Erforderlich |
|------|--------------|--------------|
| Vollständiger Name | Ihr Vor- und Nachname | ✅ Ja |
| E-Mail | Ihre E-Mail-Adresse | ✅ Ja |
| Passwort | Erstellen Sie ein sicheres Passwort | ✅ Ja |
| Passwort bestätigen | Passwort erneut eingeben | ✅ Ja |

#### Für Eltern:

| Feld | Beschreibung | Erforderlich |
|------|--------------|--------------|
| Elternname | Ihr vollständiger Name | ✅ Ja |
| E-Mail | Ihre E-Mail-Adresse | ✅ Ja |
| Passwort | Erstellen Sie ein sicheres Passwort | ✅ Ja |
| Passwort bestätigen | Passwort erneut eingeben | ✅ Ja |
| Alter des Kindes | Alter Ihres Kindes | ✅ Ja |
| Gewünschtes Startdatum | Wann Sie Betreuung benötigen | ✅ Ja |

### Schritt 4: Bedingungen akzeptieren und CAPTCHA ausfüllen

1. ☑️ Aktivieren Sie das Kontrollkästchen, um die **Nutzungsbedingungen** zu akzeptieren
2. Füllen Sie die **CAPTCHA-Verifizierung** aus, um zu beweisen, dass Sie ein Mensch sind
3. Klicken Sie auf **"Konto erstellen"**

### Schritt 5: E-Mail verifizieren

Nach dem Klicken auf "Konto erstellen" sehen Sie den **E-Mail-Verifizierung**-Bildschirm.

#### Was passiert:

1. Ein **6-stelliger Verifizierungscode** wird an Ihre E-Mail-Adresse gesendet
2. Überprüfen Sie Ihren Posteingang (und den Spam-/Junk-Ordner falls nötig)
3. Die E-Mail kommt von ProCrèche Solutions mit dem Betreff: "E-Mail verifizieren"

#### E-Mail-Verifizierungsbildschirm:

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│              ✉️ E-Mail verifizieren                 │
│                                                     │
│  Wir haben einen Verifizierungscode gesendet an:   │
│  ihre.email@beispiel.com                            │
│                                                     │
│  Bitte geben Sie den Code unten ein:               │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │              [0] [0] [0] [0] [0] [0]        │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │            [ E-Mail verifizieren ]           │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  Code nicht erhalten?                              │
│  [Code erneut senden] (in 30s verfügbar)           │
│                                                     │
└─────────────────────────────────────────────────────┘
```

#### So verifizieren Sie Ihre E-Mail:

1. Öffnen Sie Ihre E-Mail und finden Sie die Verifizierungs-E-Mail
2. Kopieren Sie den **6-stelligen Code** aus der E-Mail
3. Geben Sie den Code in das Verifizierungsfeld ein
4. Klicken Sie auf **"E-Mail verifizieren"**

#### Code erneut senden:

Wenn Sie die E-Mail nicht erhalten haben:

1. Warten Sie 30 Sekunden (ein Countdown wird angezeigt)
2. Klicken Sie auf **"Code erneut senden"**
3. Überprüfen Sie Ihre E-Mail erneut
4. Schauen Sie auch in Ihrem **Spam-/Junk-Ordner** nach

#### Code abgelaufen?

Verifizierungscodes laufen nach einigen Minuten ab. Wenn Ihr Code abgelaufen ist:

1. Klicken Sie auf **"Code erneut senden"**, um einen neuen zu erhalten
2. Verwenden Sie den neuen Code zur Verifizierung

> 💡 **Tipp:** Der Code ist normalerweise 10 Minuten gültig. Geben Sie ihn ein, sobald Sie ihn erhalten.

### Schritt 6: Kontoeinrichtung abgeschlossen!

Nach erfolgreicher Verifizierung:

1. Ihr Konto ist erstellt
2. Sie sehen eine **"Erfolg!"**-Nachricht
3. Klicken Sie auf **"Zum Dashboard"**, um auf Ihr Konto zuzugreifen

> ⏳ **Bitte warten:** Das System benötigt einige Sekunden, um Ihr Konto nach der Verifizierung einzurichten. Sie sehen einen Ladeindikator.

## Alternative: Mit Google registrieren

Sie können auch ein Konto mit Ihrem Google-Konto erstellen:

### Schritte:

1. Klicken Sie auf der Registrierungsseite auf **"Mit Google fortfahren"**
2. Wählen Sie Ihr Google-Konto
3. Erteilen Sie ProCrèche Solutions die Berechtigung
4. Sie werden weitergeleitet, um Ihr Profil zu vervollständigen

### Google-Registrierungsprofil vervollständigen:

Nach der Google-Authentifizierung müssen Sie:

1. **Ihre Rolle auswählen** (Stiftung, Lieferant, etc.)
2. **Erforderliche Informationen** für Ihre Rolle ausfüllen
3. Nutzungsbedingungen akzeptieren
4. CAPTCHA ausfüllen
5. Auf **"Profil vervollständigen"** klicken

> 📧 **Hinweis:** Ihre Google-E-Mail-Adresse wird verwendet und kann in diesem Schritt nicht geändert werden.

## Anmelden

### Standard-Anmeldung

1. Gehen Sie zur Anmeldeseite
2. Geben Sie Ihre **E-Mail-Adresse** ein
3. Geben Sie Ihr **Passwort** ein
4. Klicken Sie auf **"Anmelden"**

### Mit Google anmelden

1. Klicken Sie auf **"Mit Google fortfahren"**
2. Wählen Sie Ihr Google-Konto
3. Sie werden automatisch angemeldet

### Angemeldet bleiben

Ihre Sitzung bleibt aktiv, während Sie auf der Plattform navigieren. Aus Sicherheitsgründen werden Sie nach einer Inaktivitätsperiode automatisch abgemeldet.

## Passwort vergessen

### Schritt 1: Passwort-Zurücksetzung anfordern

1. Klicken Sie auf der Anmeldeseite auf **"Passwort vergessen?"**
2. Geben Sie Ihre E-Mail-Adresse ein
3. Klicken Sie auf **"Zurücksetzungslink senden"**

### Schritt 2: E-Mail überprüfen

1. Suchen Sie nach einer E-Mail von ProCrèche Solutions
2. Betreff: "Passwort zurücksetzen"
3. Klicken Sie auf den **Zurücksetzungslink** in der E-Mail

### Schritt 3: Neues Passwort erstellen

1. Geben Sie Ihr **neues Passwort** ein (mindestens 8 Zeichen)
2. **Bestätigen** Sie Ihr neues Passwort
3. Klicken Sie auf **"Passwort zurücksetzen"**

### Schritt 4: Anmelden

1. Sie werden zur Anmeldeseite weitergeleitet
2. Melden Sie sich mit Ihrem neuen Passwort an

> 💡 **Tipp:** Zurücksetzungslinks laufen nach kurzer Zeit ab. Fordern Sie einen neuen an, wenn Ihrer abgelaufen ist.

## Kontosicherheit

### Passwortanforderungen

Ihr Passwort muss:
- Mindestens **8 Zeichen** lang sein
- Kein häufig verwendetes Passwort sein
- Nicht in einem Datenleck kompromittiert sein

### Passwort ändern

1. Gehen Sie zu **Einstellungen** → **Kontosicherheit**
2. Geben Sie Ihr **aktuelles Passwort** ein
3. Geben Sie Ihr **neues Passwort** ein
4. Bestätigen Sie Ihr neues Passwort
5. Klicken Sie auf **"Passwort aktualisieren"**

### E-Mail-Adresse ändern

1. Gehen Sie zu **Einstellungen** → **Kontosicherheit**
2. Scrollen Sie zu **"E-Mail-Adresse ändern"**
3. Geben Sie Ihre neue E-Mail-Adresse ein
4. Klicken Sie auf **"Verifizierungscode senden"**
5. Überprüfen Sie Ihre **neue E-Mail** auf einen Verifizierungscode
6. Geben Sie den 6-stelligen Code ein
7. Klicken Sie auf **"Verifizieren und E-Mail ändern"**

> ⚠️ **Wichtig:** Sie müssen Ihre neue E-Mail-Adresse verifizieren, bevor die Änderung wirksam wird.

### Beste Praktiken

- ✅ Verwenden Sie ein einzigartiges Passwort für Ihr Konto
- ✅ Teilen Sie Ihr Passwort nicht mit anderen
- ✅ Melden Sie sich ab, wenn Sie gemeinsam genutzte Computer verwenden
- ✅ Halten Sie Ihre E-Mail-Adresse aktuell

## Profil vervollständigen

Nach der Registrierung empfehlen wir, Ihr Profil für eine bessere Erfahrung zu vervollständigen.

### Warum Ihr Profil vervollständigen?

- ✅ **Stiftungen:** In Elternsuchen erscheinen, Kandidaten anziehen
- ✅ **Lieferanten/Dienstleister:** Von potenziellen Kunden entdeckt werden
- ✅ **Erzieher:** Bei Arbeitgebern hervorstechen
- ✅ **Eltern:** Bessere Kita-Matches erhalten

### So vervollständigen Sie Ihr Profil

1. Klicken Sie auf Ihr **Profilbild** (oben rechts)
2. Wählen Sie **"Profil bearbeiten"**
3. Füllen Sie alle Abschnitte aus
4. Laden Sie ein Profilbild hoch
5. Klicken Sie auf **"Speichern"**

> 📖 **Mehr erfahren:** Lesen Sie Ihren rollenspezifischen Guide für detaillierte Profilanweisungen.

## Fehlerbehebung

### Häufige Probleme

| Problem | Lösung |
|---------|--------|
| Verifizierungs-E-Mail nicht erhalten | Spam-Ordner prüfen, 30s warten und erneut senden |
| Verifizierungscode abgelaufen | Klicken Sie auf "Code erneut senden" für einen neuen |
| Anmeldung nicht möglich | E-Mail/Passwort überprüfen, "Passwort vergessen" verwenden |
| Profil lädt nicht | Browser-Cache leeren, Seite aktualisieren |
| Google-Anmeldung funktioniert nicht | Standard-E-Mail/Passwort-Anmeldung versuchen |

Für weitere Hilfe siehe unseren [Fehlerbehebungs-Guide](../reference/10-troubleshooting-faq.md).
