## Promo-Codes

### Einen Promo-Code erstellen

1. Klicken Sie auf **"Neuen Code hinzufügen"**
2. Füllen Sie das Formular aus:
   - **Code**: Der Code, den Kunden verwenden
   - **Rabatt**: Beschreibung des Rabatts ("20% Rabatt", "Kostenloser Versand")
   - **Beschreibung**: AGB (optional)
   - **Im Profil anzeigen**: Sichtbarkeit ein/aus

3. Klicken Sie auf **"Speichern"**
