## Abonnement-Anforderungen

Produktlieferanten benötigen ein aktives Abonnement, um auf die Plattformfunktionen zuzugreifen.

**In Ihrem Plan enthalten:**
- ✅ Produktverwaltung
- ✅ Bestellungsverwaltung
- ✅ Anfragenverwaltung
- ✅ Analytik
- ✅ Promo-Codes
- ✅ Nachrichten
- ✅ Team-Management

**Immer verfügbar (ohne Abonnement):** Einstellungen, Profilverwaltung, Support

## Dashboard-Übersicht

Ihr Dashboard gibt Ihnen einen umfassenden Überblick über Ihre Geschäftsaktivitäten. Greifen Sie darauf zu, indem Sie auf **Dashboard** in der Seitenleiste klicken.

### Verkaufsübersicht-Widget

Sehen Sie Ihre Verkäufe auf einen Blick:

| Metrik | Was sie zeigt |
|--------|---------------|
| 💰 **Gesamtumsatz** | Ihre Einnahmen über Zeit |
| 📦 **Gesamte Bestellungen** | Anzahl der erhaltenen Bestellungen |
| ✅ **Erfüllungsrate** | Prozentsatz der gelieferten Bestellungen |
| ⏳ **Ausstehende Bestellungen** | Bestellungen, die Aktion erfordern |

### Anfragen-Widget

Kundenanfragen verfolgen:
- Erhaltene Anfragen insgesamt
- Ausstehende Anfragen, die Antwort benötigen
- Antwortrate

### Produktzusammenfassung

Schnellübersicht Ihres Inventars:
- Gelistete Produkte insgesamt
- Aktive Produkte
- Nicht vorrätige Artikel

## Organisationsprofil

Ihr Firmenprofil ist für Stiftungen sichtbar, die den Marktplatz durchsuchen. Ein vollständiges Profil schafft Vertrauen und zieht Kunden an.

### Auf Ihr Profil zugreifen

1. Klicken Sie auf **Organisationsprofil** in der Seitenleiste, ODER
2. Gehen Sie zu **Einstellungen** → **Profil bearbeiten**

### Profilinformationen

#### Grundlegende Geschäftsdetails

| Feld | Beschreibung | Tipps |
|------|--------------|-------|
| **Firmenname** | Ihr Unternehmensname | Verwenden Sie Ihren offiziellen Handelsnamen |
| **Kontaktperson** | Hauptansprechpartner | Vertrieb oder Kundenbetreuer |
| **Telefonnummer** | Geschäftstelefon | Landesvorwahl einbeziehen |
| **Kontakt-E-Mail** | Öffentliche E-Mail | Für Kundenanfragen |
| **Kanton/Region** | Ihr Standort | Wo Sie ansässig sind |
| **Stadt** | Ihre Stadt | Hilft Kunden, lokale Lieferanten zu finden |

#### Geschäftsdetails

| Feld | Beschreibung | Tipps |
|------|--------------|-------|
| **Beschreibung** | Über Ihr Unternehmen | Beschreiben Sie Ihre Produkte und Werte |
| **Produktkategorien** | Verkaufte Produktarten | Alle relevanten Kategorien auswählen |
| **Belieferte Regionen** | Wohin Sie liefern | Mehrere Kantone auswählen |
| **Sprachen** | Unterstützte Sprachen | Für Kundenkommunikation |
| **USt-IdNr.** | Steueridentifikation | Erforderlich für Abrechnung |
| **Mindestbestellmenge** | Mindestbestellgröße | Festlegen, falls zutreffend |
| **Direktbestelllink** | Externe Bestell-URL | Link zu Ihrem eigenen Shop (optional) |
| **Katalog-URL** | Link zum Produktkatalog | PDF oder Online-Katalog |

## Produktkatalog

Verwalten Sie Ihre Produkte, die auf dem Marktplatz erscheinen.

### Auf Produkte zugreifen

Klicken Sie auf **Produktkatalog** in der Seitenleiste.

### Produkt erstellen

1. Klicken Sie auf **"Produkt erstellen"**
2. Füllen Sie die Produktdetails aus:

#### Grundlegende Informationen

| Feld | Beschreibung | Erforderlich |
|------|--------------|--------------|
| **Titel** | Produktname | ✅ Ja |
| **Untertitel** | Kurzer Slogan | ❌ Optional |
| **Beschreibung** | Vollständige Produktbeschreibung | ✅ Ja |
| **Preis** | Produktpreis | ✅ Ja |
| **Währung** | CHF, EUR, etc. | ✅ Ja (Standard: CHF) |

#### Inventardetails

| Feld | Beschreibung |
|------|--------------|
| **SKU** | Interner Produktcode |
| **Mengeneinheit** | Stück, Pack, Karton, etc. |
| **Min. Bestellmenge** | Mindesteinheiten pro Bestellung |
| **Lagerstatus** | Auf Lager, Niedriger Bestand, Nicht vorrätig |
| **Lieferzeit** | Tage bis zur Lieferung |

3. Produktstatus festlegen:

| Status | Sichtbarkeit |
|--------|--------------|
| **Aktiv** | Sichtbar auf Marktplatz |
| **Inaktiv** | Vom Marktplatz verborgen |
| **Entwurf** | Noch nicht veröffentlicht |
| **Nicht vorrätig** | Als nicht verfügbar angezeigt |

4. Klicken Sie auf **"Produkt speichern"**

## Bestellungsverwaltung

Verwalten Sie Bestellungen von Stiftungen.

### Auf Bestellungen zugreifen

Klicken Sie auf **Bestellungen** in der Seitenleiste.

### Bestellstatus

| Status | Bedeutung | Ihre Aktion |
|--------|-----------|-------------|
| 📝 **Ausstehend** | Neue Bestellung erhalten | Prüfen und akzeptieren |
| ✅ **Akzeptiert** | Sie haben bestätigt | Für Versand vorbereiten |
| 🔄 **In Bearbeitung** | Wird vorbereitet | Aktualisieren, wenn fertig |
| 🚚 **Versendet** | An Kunden gesendet | Tracking-Info bereitstellen |
| ✅ **Geliefert** | Erfolgreich zugestellt | Abgeschlossen |
| ❌ **Storniert** | Bestellung storniert | Keine Aktion erforderlich |

### Bestellstatus aktualisieren

1. Bestellung öffnen
2. Auf **"Status aktualisieren"** klicken
3. Neuen Status auswählen
4. Notizen hinzufügen (optional)
5. Speichern

## Anfragenverwaltung

Verwalten Sie Fragen und Anfragen von potenziellen Kunden.

### Was sind Anfragen?

Anfragen sind Nachrichten von Stiftungen, die mehr über Ihre Produkte erfahren möchten, bevor sie bestellen.

### Anfragen-Status

| Status | Bedeutung | Ihre Aktion |
|--------|-----------|-------------|
| 🆕 **Neu** | Gerade erhalten | Prüfen und antworten |
| ⏳ **Ausstehend** | Wartet auf Ihre Antwort | Schnell antworten |
| 📞 **Kontaktiert** | Sie haben geantwortet | Bei Bedarf nachfassen |
| 💰 **Angebot gesendet** | Angebot bereitgestellt | Auf Entscheidung warten |
| ✅ **Abgeschlossen** | In Bestellung umgewandelt | Fertig |
| ❌ **Abgelehnt** | Sie haben abgelehnt | Keine Aktion |

## Promo-Codes

Erstellen Sie Aktionscodes, um Kunden anzuziehen. Promo-Codes werden auf Ihrem öffentlichen Profil angezeigt.

### Auf Promo-Codes zugreifen

Gehen Sie zu **Einstellungen** → **Promo-Code-Manager**

### Promo-Code erstellen

1. Klicken Sie auf **"Neuen Code hinzufügen"**
2. Ein Modalfenster öffnet sich mit diesen Feldern:

```
┌─────────────────────────────────────────────────────┐
│         Neuen Promo-Code hinzufügen             [X] │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Code *                                             │
│  ┌─────────────────────────────────────────────┐   │
│  │  PROMO20                                     │   │
│  └─────────────────────────────────────────────┘   │
│  Der Code, den Kunden verwenden werden             │
│                                                     │
│  Rabatt *                                           │
│  ┌─────────────────────────────────────────────┐   │
│  │  20% Rabatt auf Ihre erste Bestellung       │   │
│  └─────────────────────────────────────────────┘   │
│  Beschreiben Sie den Rabatt                        │
│                                                     │
│  Beschreibung (Optional)                            │
│  ┌─────────────────────────────────────────────┐   │
│  │  Nur für Neukunden gültig.                  │   │
│  │  Nicht mit anderen Angeboten kombinierbar.  │   │
│  └─────────────────────────────────────────────┘   │
│  Optionale Details zu Bedingungen                  │
│                                                     │
│  ☑ Auf Profil anzeigen                             │
│                                                     │
│         [Abbrechen]              [Speichern]       │
└─────────────────────────────────────────────────────┘
```

#### Promo-Code-Felder

| Feld | Beschreibung | Erforderlich | Beispiel |
|------|--------------|--------------|----------|
| **Code** | Der Code, den Kunden verwenden | ✅ Ja | `PROMO20`, `VERSANDFREI` |
| **Rabatt** | Was der Kunde erhält | ✅ Ja | "20% Rabatt", "Kostenloser Versand" |
| **Beschreibung** | Bedingungen | ❌ Optional | "Gültig für Bestellungen > CHF 100" |
| **Auf Profil anzeigen** | Öffentlich anzeigen | ✅ Ja (Standard: aktiviert) | An/Aus |

3. Klicken Sie auf **"Speichern"**

### Promo-Codes verwalten

| Aktion | Wie |
|--------|-----|
| **Bearbeiten** | Auf "Bearbeiten" beim Code klicken |
| **Löschen** | Auf "Löschen" klicken |
| **Verbergen** | "Auf Profil anzeigen" deaktivieren |

### Wie Kunden Promo-Codes sehen

Wenn Stiftungen Ihr Profil anzeigen, sehen sie Ihre aktiven Promo-Codes:

```

│  │  PROMO20                     Code kopieren [📋]│ │
│  │  20% Rabatt erste Bestellung                  │ │
│  │  Nur Neukunden                                │ │
│  └───────────────────────────────────────────────┘ │
│                                                     │
│  ┌───────────────────────────────────────────────┐ │
│  │  VERSANDFREI                 Code kopieren [📋]│ │
│  │  Kostenloser Versand > CHF 100                │ │
│  └───────────────────────────────────────────────┘ │
│                                                     │
```

## Analytik

Verfolgen Sie Ihre Geschäftsleistung mit detaillierten Analysen.

### Auf Analysen zugreifen

Klicken Sie auf **Analytik** in der Seitenleiste.

### Verfügbare Berichte

#### Verkaufsanalysen
- Gesamtumsatz
- Umsatztrends über Zeit
- Durchschnittlicher Bestellwert
- Meistverkaufte Produkte

#### Bestellungsanalysen
- Bestellungen nach Status
- Bestellvolumen-Trends
- Erfüllungsrate
- Bearbeitungszeit

#### Anfrageanalysen
- Anfragen insgesamt
- Antwortrate
- Konversionsrate
- Anfragequellen

### Daten exportieren

Klicken Sie auf **"CSV exportieren"**, um Berichte herunterzuladen.

## Rabattkündigungen

### Was sind Rabattkündigungen?

Rabattkündigungen sind spezielle Vereinbarungen, bei denen Aktionspreise oder Rabatte formell beendet werden.

### Wie es funktioniert

1. Rabattkündigungen werden über das Admin-System verwaltet
2. Sie werden im Voraus über jede Kündigung informiert
3. Standardpreise gelten nach der Kündigung wieder
4. Rabatthistorie wird aufbewahrt
