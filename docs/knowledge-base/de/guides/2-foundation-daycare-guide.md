## Dashboard-Übersicht

Ihr Dashboard ist Ihre Homebase. Greifen Sie darauf zu, indem Sie auf **Dashboard** in der Seitenleiste klicken.

### Schnellstatistik-Karten

Oben auf Ihrem Dashboard sehen Sie wichtige Metriken auf einen Blick:

| Statistik | Was sie zeigt |
|-----------|---------------|
| 📋 **Gesamte Stellenangebote** | Anzahl der veröffentlichten Stellen |
| ⏳ **Ausstehende Bewerbungen** | Zu prüfende Bewerbungen |
| 📝 **Gesamte Bewerbungen** | Alle erhaltenen Bewerbungen |
| 🛒 **Gesamte Bestellungen** | Auf dem Marktplatz getätigte Bestellungen |
| 🔧 **Serviceanfragen** | Angeforderte Dienstleistungen |
| ✅ **Aktive Stellen** | Derzeit veröffentlichte Stellenangebote |

### Letzte Aktivitäten

Unter den Statistiken sehen Sie Ihre neuesten Aktivitäten:
- Neue erhaltene Eltern-Leads
- Eingegangene Bewerbungen
- Bestellungsaktualisierungen
- Serviceanfragen-Updates
- Neue Nachrichten

### Kalenderübersicht

Kalenderevents anzeigen und verwalten:
- Bevorstehende Termine
- Vorstellungsgespräche
- Service-Termine
- Besprechungen
- Erinnerungen

### Kalenderevent erstellen

1. Gehen Sie zum **Dashboard**
2. Finden Sie den Kalenderbereich
3. Klicken Sie auf **"Event hinzufügen"**
4. Füllen Sie aus:
   - **Titel** - Event-Name
   - **Typ** - Termin, Interview, Service, Meeting oder Erinnerung
   - **Datum und Zeit** - Wann es stattfindet
   - **Beschreibung** - Zusätzliche Details (optional)
   - **Ort** - Wo es stattfindet (optional)
5. Klicken Sie auf **Speichern**

## Organisationsprofil

Ihr Organisationsprofil ist für Eltern, Erzieher und Lieferanten sichtbar. Halten Sie es aktuell, um die richtigen Verbindungen anzuziehen.

### Auf Ihr Profil zugreifen

1. Klicken Sie auf **Organisationsprofil** in der Seitenleiste, ODER
2. Gehen Sie zu **Einstellungen** → **Profil bearbeiten**

### Profilinformationen

#### Grundlegende Informationen

| Feld | Beschreibung | Tipps |
|------|--------------|-------|
| **Organisationsname** | Name Ihrer Kita/Stiftung | Verwenden Sie Ihren offiziellen Namen |
| **Kontaktperson** | Hauptansprechpartner | Normalerweise Leiter oder Manager |
| **Telefonnummer** | Geschäftstelefon | Landesvorwahl einbeziehen |
| **Kontakt-E-Mail** | Öffentliche Kontakt-E-Mail | Kann von der Anmelde-E-Mail abweichen |
| **Kanton/Region** | Ihr Standort | Aus der Liste auswählen |
| **Stadt** | Ihre Stadt | Hilft Eltern, Sie zu finden |
| **Adresse** | Postanschrift | Für Servicelieferungen |

#### Detaillierte Informationen

| Feld | Beschreibung | Tipps |
|------|--------------|-------|
| **Über/Beschreibung** | Beschreiben Sie Ihre Kita | Heben Sie Ihre Philosophie hervor |
| **Kapazität** | Anzahl der betreuten Kinder | Lizenzierte Gesamtkapazität eingeben |
| **Gesprochene Sprachen** | Angebotene Sprachen | Alle zutreffenden auswählen |
| **Pädagogik** | Pädagogische Ansätze | Montessori, Reggio Emilia, etc. |
| **USt-IdNr.** | Für die Abrechnung (falls zutreffend) | Optional |
| **Betreute Regionen** | Gebiete, die Sie bedienen | Mehrere Kantone auswählen |

#### Visuelle Elemente

| Element | Empfehlung |
|---------|------------|
| **Logo** | Quadratisches Bild, min 200x200px, erscheint in Listen |
| **Titelbild** | Breites Banner, min 1200x400px, erscheint auf Profilseite |

### Profil speichern

1. Nehmen Sie Ihre Änderungen vor
2. Klicken Sie unten auf **"Änderungen speichern"**
3. Warten Sie auf die Bestätigungsmeldung

> 💡 **Tipp:** Vollständige Profile erscheinen höher in den Suchergebnissen!

## Eltern-Leads (Anfragen)

Eltern-Leads sind Anfragen von Eltern, die Kinderbetreuung suchen. Schnelle Antworten erhöhen die Anmeldungen.

> 📋 **Erforderlich:** Essential-Plan oder höher

### Lead-Posteingang verstehen

Greifen Sie auf Ihre Leads zu, indem Sie auf **Leads** in der Seitenleiste klicken.

#### Lead-Informationen

Jeder Lead enthält:
- **Elterndetails** - Name, E-Mail, Telefon
- **Kinderdetails** - Name, Alter
- **Präferenzen** - Standort, Startdatum, Sprachen
- **Besondere Bedürfnisse** - Spezielle Anforderungen
- **Nachricht** - Persönliche Notiz des Elternteils

#### Lead-Status

| Status | Bedeutung | Erforderliche Aktion |
|--------|-----------|---------------------|
| 🆕 **Neu** | Gerade erhalten | Prüfen und antworten |
| 🔄 **In Bearbeitung** | Sie arbeiten daran | Weiteres Follow-up |
| ✅ **Konvertiert** | Erfolgreich angemeldet | Abgeschlossen |

### Auf einen Lead antworten

1. Klicken Sie auf einen Lead, um Details zu öffnen
2. Prüfen Sie die Informationen und Bedürfnisse des Elternteils
3. Klicken Sie auf **"Antworten"**
4. Wählen Sie Ihre Antwort:
   
   | Antwort | Wann verwenden |
   |---------|----------------|
   | **Interessiert** | Sie möchten mit dieser Familie fortfahren |
   | **Nicht interessiert** | Keine Übereinstimmung (volle Kapazität, etc.) |
   | **Mehr Infos benötigt** | Sie benötigen zusätzliche Informationen |
   | **Angemeldet** | Kind wurde angemeldet |

5. Fügen Sie eine persönliche Nachricht hinzu (empfohlen)
6. Klicken Sie auf **Antwort senden**

## Marktplatz-Zugang

Durchsuchen und kaufen Sie Produkte und Dienstleistungen von verifizierten Anbietern.

### Produkte durchsuchen

1. Klicken Sie auf **Marktplatz** in der Seitenleiste
2. Wählen Sie den Tab **Produkte**
3. Verwenden Sie Filter:
   - **Kategorie** - Spielzeug, Möbel, Lebensmittel, etc.
   - **Lieferant** - Nach bestimmtem Lieferanten filtern
   - **Suche** - Bestimmte Produkte finden
4. Klicken Sie auf ein Produkt für Details

### Dienstleistungen durchsuchen

1. Im Marktplatz, wählen Sie den Tab **Dienstleistungen**
2. Filtern nach:
   - **Dienstleistungstyp** - Reinigung, Schulung, etc.
   - **Anbieter** - Nach Lieferant filtern
   - **Lieferart** - Vor Ort, Remote, Hybrid
3. Klicken Sie auf eine Dienstleistung für Details

### Bestellungen aufgeben

Für Produktbestellungen:

1. Finden Sie das gewünschte Produkt
2. Klicken Sie auf **"In den Warenkorb"** oder **"Jetzt bestellen"**
3. Menge festlegen
4. Warenkorb überprüfen
5. Klicken Sie auf **"Bestellung aufgeben"**
6. Bestelldetails bestätigen
7. Bestellung abschließen

### Dienstleistungen anfragen

Für Dienstleistungsanfragen:

1. Finden Sie die gewünschte Dienstleistung
2. Klicken Sie auf **"Dienstleistung anfragen"**
3. Füllen Sie die Anforderungsdetails aus:
   - Bevorzugte Termine
   - Spezifische Anforderungen
   - Zusätzliche Hinweise
4. Anfrage absenden
5. Auf Antwort des Anbieters warten

## Rekrutierungsmodul

Veröffentlichen Sie Stellenangebote und finden Sie qualifizierte Erzieher für Ihre Kita.

> 📋 **Erforderlich:** Professional-Plan oder höher

### Auf Rekrutierung zugreifen

Klicken Sie auf **Rekrutierung** in der Seitenleiste.

### Stellenangebot erstellen

1. Gehen Sie zu **Rekrutierung** → **Stellenangebote**
2. Klicken Sie auf **"Stelle erstellen"**
3. Füllen Sie die Details aus:

| Feld | Beschreibung | Tipps |
|------|--------------|-------|
| **Titel** | Stellenbezeichnung | Seien Sie spezifisch: "Leitende Kita-Erzieherin" |
| **Beschreibung** | Vollständige Stellenbeschreibung | Tägliche Aufgaben einbeziehen |
| **Anforderungen** | Was Bewerber benötigen | Pflichtqualifikationen auflisten |
| **Leistungen** | Was Sie bieten | Vorteile hervorheben |
| **Verantwortlichkeiten** | Tägliche Aufgaben | Detailliert sein |
| **Qualifikationen** | Gewünschter Hintergrund | Ausbildung, Zertifikate |
| **Standort** | Arbeitsort | Stadt/Kanton |
| **Gehaltsrahmen** | Angebotene Vergütung | Transparent sein |
| **Vertragsart** | Beschäftigungsart | Vollzeit, Teilzeit, etc. |
| **Startdatum** | Wann die Stelle beginnt | Sofort oder bestimmtes Datum |

4. Status wählen:
   - **Entwurf** - Speichern ohne Veröffentlichung
   - **Veröffentlicht** - Für Kandidaten sichtbar machen
5. Klicken Sie auf **Speichern**

### Vertragsarten

| Typ | Beschreibung |
|-----|--------------|
| **Vollzeit** | Standard-Vollzeitstelle |
| **Teilzeit** | Reduzierte Arbeitszeit |
| **Unbefristet** | Unbefristeter Vertrag (permanent) |
| **Befristet** | Befristeter Vertrag |
| **Praktikum** | Ausbildungsposition |
| **Vertretung** | Vorübergehende Vertretungsposition |
| **Temporär** | Kurzfristiger Einsatz |
| **Freiberuflich** | Selbstständige Arbeit |

### Bewerbungen verwalten

Bewerbungsstatus aktualisieren:

| Status | Wann verwenden |
|--------|----------------|
| **Ausstehend** | Gerade erhalten, noch nicht geprüft |
| **Geprüft** | Sie haben sie angesehen |
| **Akzeptiert** | Weitermachen/Stelle anbieten |
| **Abgelehnt** | Nicht in Betracht gezogen |

## Support und Hilfe

### Support-Ticket erstellen

1. Klicken Sie auf **Support** in der Seitenleiste
2. Klicken Sie auf **"Ticket erstellen"**
3. Füllen Sie aus:
   - **Betreff** - Kurze Beschreibung
   - **Kategorie** - Allgemein, Technisch, Abrechnung, Feature-Anfrage
   - **Priorität** - Niedrig, Mittel, Hoch, Dringend
   - **Nachricht** - Detaillierte Erklärung
   - **Anhänge** - Screenshots oder Dateien (optional)
4. Klicken Sie auf **Absenden**

### Ticketstatus

| Status | Bedeutung |
|--------|-----------|
| **Offen** | Eingereicht, wartet auf Antwort |
| **In Bearbeitung** | Wird bearbeitet |
| **Gelöst** | Problem behoben |
| **Geschlossen** | Ticket abgeschlossen |
