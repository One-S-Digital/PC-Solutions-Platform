## Kostenlos vs. Kostenpflichtig

### Was immer kostenlos ist

Alle Benutzer haben Zugang zu:

- ✅ Kontoerstellung und -verwaltung
- ✅ Grundlegende Profilverwaltung
- ✅ Nachrichten
- ✅ Grundlegender Support
- ✅ Einstellungen und Präferenzen
- ✅ Benachrichtigungen

### Kostenlose Rollen

| Rolle | Kosten |
|-------|--------|
| 👨‍👩‍👧 **Eltern** | Immer kostenlos |
| 👩‍🏫 **Erzieher** | Immer kostenlos |

### Rollen mit Abonnement

| Rolle | Abonnement erforderlich für |
|-------|----------------------------|
| 🏫 **Stiftung** | Leads, Rekrutierung, HR, E-Learning |
| 📦 **Lieferant** | Produktverwaltung, Bestellungen |
| 🔧 **Dienstleister** | Dienstleistungsverwaltung, Anfragen |

2. Klicken Sie auf **Abrechnung und Abonnement**

### Angezeigte Informationen

| Element | Beschreibung |
|---------|--------------|
| **Aktueller Plan** | Name Ihres Plans |
| **Preis** | Monatliche oder jährliche Kosten |
| **Status** | Aktiv, Inaktiv, Ausstehend |
| **Verlängerung** | Datum der nächsten Zahlung |
| **Zahlungsmethode** | Registrierte Karte |

### Plan wechseln

#### Upgrade

1. Gehen Sie zu **Einstellungen** → **Abrechnung**
2. Klicken Sie auf **"Planänderung anfordern"**
3. Wählen Sie den gewünschten neuen Plan
4. Bestätigen Sie Ihre Anfrage
5. Das Team wird Sie kontaktieren

#### Downgrade

1. Gleicher Prozess wie beim Upgrade
2. Wählen Sie einen niedrigeren Plan
3. Bestätigen
4. Tritt am Ende des aktuellen Zeitraums in Kraft

> ⚠️ **Hinweis:** Ein Downgrade kann zum Verlust des Zugangs zu bestimmten Funktionen führen.

## Zahlungsmethoden

### Akzeptierte Methoden

| Methode | Unterstützt |
|---------|-------------|
| 💳 Visa | ✅ |
| 💳 Mastercard | ✅ |
| 💳 American Express | ✅ |
| 🏦 Banküberweisung | ✅ (Enterprise) |
| 📄 Rechnung | ✅ (Enterprise) |

### Zahlungsmethode hinzufügen

1. Gehen Sie zu **Einstellungen** → **Abrechnung**
2. Klicken Sie auf **"Zahlung verwalten"**
3. Klicken Sie auf **"Karte hinzufügen"**
4. Geben Sie die Kartendetails ein:
   - Kartennummer
   - Ablaufdatum
   - CVV
   - Name auf der Karte
5. Klicken Sie auf **"Speichern"**

### Zahlungsmethode ändern

1. Zahlungseinstellungen aufrufen
2. Auf **"Bearbeiten"** neben der Karte klicken
3. Informationen aktualisieren
4. Speichern

### Zahlungsmethode entfernen

1. Zahlungseinstellungen aufrufen
2. Auf **"Löschen"** neben der Karte klicken
3. Löschung bestätigen

> ⚠️ **Hinweis:** Sie müssen mindestens eine aktive Zahlungsmethode haben, wenn Sie ein laufendes Abonnement haben.

### Zahlungssicherheit

- 🔒 SSL/TLS-Verschlüsselung
- 🏦 PCI-DSS-Konformität
- 🛡️ Kartendaten-Tokenisierung
- 📧 Zahlungsbenachrichtigungen

### Rechnung herunterladen

1. Rechnung in der Liste finden
2. Auf **"Herunterladen"** oder PDF-Symbol klicken
3. PDF-Datei wird heruntergeladen
4. Für Ihre Buchhaltung verwenden

## Jährliche vs. monatliche Abrechnung

### Vorteile der Jahresabrechnung

| Aspekt | Monatlich | Jährlich |
|--------|-----------|----------|
| **Zahlung** | Jeden Monat | Einmal pro Jahr |
| **Preis** | Standardpreis | Rabatt (~10-20%) |
| **Bindung** | Monat für Monat | 12 Monate |
| **Flexibilität** | Flexibler | Weniger flexibel |

### Wie wechseln

1. Support kontaktieren oder
2. Über **Einstellungen** → **Abrechnung** anfordern
3. Das Team wird Sie mit Optionen kontaktieren

### Was beim Wechsel passiert

- **Monatlich → Jährlich**: Anteilig berechnet, Differenz in Rechnung gestellt
- **Jährlich → Monatlich**: Tritt am Ende des Jahres in Kraft

## Abonnementanfragen

### Wie das System funktioniert

Abonnementänderungen laufen über ein Anfragensystem:

1. Sie reichen eine Anfrage ein
2. Das Team prüft Ihre Anfrage
3. Sie erhalten eine Bestätigung
4. Die Änderung wird angewendet

### Anfrage einreichen

1. Gehen Sie zu **Einstellungen** → **Abrechnung**
2. Klicken Sie auf **"Änderung anfordern"**
3. Wählen Sie den Anfragetyp:
   - Upgrade
   - Downgrade
   - Zyklusänderung
   - Kündigung
4. Fügen Sie bei Bedarf Kommentare hinzu
5. Absenden

### Ihre Anfragen verfolgen

- Laufende Anfragen werden angezeigt
- Sie erhalten Update-Benachrichtigungen
- Kontaktieren Sie den Support bei Fragen

## Kündigung

### Wie kündigen

1. Gehen Sie zu **Einstellungen** → **Abrechnung**
2. Klicken Sie auf **"Kündigung anfordern"**
3. Wählen Sie einen Grund (optional)
4. Bestätigen Sie Ihre Anfrage

### Was nach der Kündigung passiert

| Element | Verhalten |
|---------|-----------|
| **Zugang** | Bis zum Ende des bezahlten Zeitraums |
| **Daten** | 30 Tage aufbewahrt |
| **Konto** | Wechselt zum kostenlosen Plan |
| **Funktionen** | Zugang zu Basisfunktionen |

### Reaktivierung

Sie können Ihr Abonnement reaktivieren:

1. Gehen Sie zu **Einstellungen** → **Abrechnung**
2. Klicken Sie auf **"Abonnement reaktivieren"**
3. Wählen Sie einen Plan
4. Fahren Sie mit der Zahlung fort

> 💡 **Tipp:** Wenn Sie versehentlich kündigen, kontaktieren Sie schnell den Support.

### Erstattungen

- Erstattungen werden von Fall zu Fall bewertet
- Kontaktieren Sie den Support für jede Anfrage
- Anteilige Teilerstattungen sind möglich

## Häufig gestellte Fragen

### Abrechnung

**F: Wann werde ich belastet?**
A: Am gleichen Tag jeden Monat (oder Jahr) ab Ihrem Abonnementstart.

**F: Kann ich den Plan mitten im Monat wechseln?**
A: Ja, Änderungen werden anteilig berechnet.

**F: Was passiert, wenn meine Zahlung fehlschlägt?**
A: Sie erhalten eine Benachrichtigung. Sie haben in der Regel einige Tage Zeit, um Ihre Zahlungsmethode zu aktualisieren, bevor die Sperrung erfolgt.

### Pläne

**F: Kann ich einen höheren Plan testen?**
A: Kontaktieren Sie den Support, um eine Testphase zu besprechen.

**F: Gibt es versteckte Gebühren?**
A: Nein, die angezeigten Preise beinhalten alle Planfunktionen.

### Kündigung

**F: Kann ich jederzeit kündigen?**
A: Ja, aber der Zugang bleibt bis zum Ende des bezahlten Zeitraums bestehen.

**F: Werden meine Daten nach der Kündigung gelöscht?**
A: Daten werden 30 Tage aufbewahrt und dann gelöscht, sofern nicht anders gewünscht.
