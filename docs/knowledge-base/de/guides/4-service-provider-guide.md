## Organisationsprofil

Ihr Firmenprofil ist für Stiftungen sichtbar, die den Marktplatz durchsuchen. Ein vollständiges Profil generiert mehr Leads.

### Auf Ihr Profil zugreifen

1. Klicken Sie auf **Organisationsprofil** in der Seitenleiste, ODER
2. Gehen Sie zu **Einstellungen** → **Profil bearbeiten**

### Profilinformationen

#### Grundlegende Geschäftsdetails

| Feld | Beschreibung | Tipps |
|------|--------------|-------|
| **Firmenname** | Ihr Unternehmensname | Verwenden Sie Ihren offiziellen Namen |
| **Kontaktperson** | Hauptansprechpartner | Manager oder Kundenbetreuer |
| **Telefonnummer** | Geschäftstelefon | Landesvorwahl einbeziehen |
| **Kontakt-E-Mail** | Öffentliche E-Mail | Für Kundenanfragen |
| **Kanton/Region** | Ihr Standort | Wo Sie ansässig sind |
| **Stadt** | Ihre Stadt | Hilft Kunden, Sie zu finden |

#### Dienstleistungsdetails

| Feld | Beschreibung | Tipps |
|------|--------------|-------|
| **Beschreibung** | Über Ihre Dienstleistungen | Beschreiben Sie Ihre Expertise |
| **Dienstleistungstypen** | Angebotene Dienstleistungen | Alle Kategorien auswählen |
| **Lieferart** | Wie Sie liefern | Vor Ort, Remote, Hybrid |
| **Belieferte Regionen** | Wo Sie arbeiten | Abgedeckte Kantone |
| **Sprachen** | Unterstützte Sprachen | Für Kommunikation |
| **Buchungslink** | Terminbuchungs-URL | Calendly-Link, etc. |

## Dienstleistungskatalog

Verwalten Sie Ihre Dienstleistungsangebote, die auf dem Marktplatz erscheinen.

### Auf Dienstleistungen zugreifen

Klicken Sie auf **Dienstleistungskatalog** in der Seitenleiste.

### Dienstleistung erstellen

1. Klicken Sie auf **"Dienstleistung erstellen"**
2. Füllen Sie die Dienstleistungsdetails aus:

#### Grundlegende Informationen

| Feld | Beschreibung | Erforderlich |
|------|--------------|--------------|
| **Titel** | Dienstleistungsname | ✅ Ja |
| **Untertitel** | Kurzer Slogan | ❌ Optional |
| **Beschreibung** | Vollständige Beschreibung | ✅ Ja |
| **Preis** | Dienstleistungstarif | ✅ Ja |
| **Preistyp** | Pro Stunde, Pauschal, etc. | ✅ Ja |
| **Währung** | CHF, EUR, etc. | ✅ Ja |

#### Preistypen

| Typ | Beschreibung | Beispiel |
|-----|--------------|----------|
| **Pauschal** | Festpreis | CHF 500 pro Projekt |
| **Pro Stunde** | Stundensatz | CHF 80/Stunde |
| **Pro Tag** | Tagessatz | CHF 600/Tag |
| **Pro Sitzung** | Pro Session | CHF 150/Sitzung |
| **Auf Anfrage** | Individueller Preis | Kontaktieren Sie uns |

#### Dienstleistungsdetails

| Feld | Beschreibung |
|------|--------------|
| **Kategorie** | Hauptdienstleistungstyp |
| **Dauer** | Typische Dauer |
| **Lieferart** | Vor Ort, Remote, Hybrid |
| **Max. Kapazität** | Max. Teilnehmer (falls zutreffend) |
| **Buchungsvorlauf** | Erforderliche Mindestvorankündigung |
| **Highlights** | Wichtige Verkaufsargumente |

3. Dienstleistungsstatus festlegen:

| Status | Sichtbarkeit |
|--------|--------------|
| **Aktiv** | Sichtbar auf Marktplatz |
| **Inaktiv** | Vom Marktplatz verborgen |
| **Entwurf** | Noch nicht veröffentlicht |

4. Klicken Sie auf **"Dienstleistung speichern"**

## Dienstleistungsanfragen verwalten

Verwalten Sie Dienstleistungsanfragen von Stiftungen.

### Auf Anfragen zugreifen

Klicken Sie auf **Dienstleistungsanfragen** in der Seitenleiste.

### Anfragen anzeigen

Ihre Anfragenliste zeigt:
- Stiftungsdetails
- Angeforderte Dienstleistung
- Angefragtes Datum/Uhrzeit
- Besondere Anforderungen
- Status

### Anfragendetails

Jede Anfrage enthält:
- **Stiftungsinformationen** - Name, Kontakt, Standort
- **Dienstleistungsdetails** - Welche Dienstleistung sie möchten
- **Terminplanung** - Bevorzugte Daten und Zeiten
- **Anforderungen** - Spezielle Bedürfnisse oder Präferenzen
- **Budget** - Budgetrahmen, falls angegeben
- **Dringlichkeit** - Prioritätsstufe

### Anfragestatus

| Status | Bedeutung | Ihre Aktion |
|--------|-----------|-------------|
| 🆕 **Neu** | Gerade erhalten | Prüfen und antworten |
| ⏳ **Ausstehend** | Wartet auf Ihre Antwort | Schnell antworten |
| 📅 **Geplant** | Termin festgelegt | Lieferung vorbereiten |
| 🔄 **In Bearbeitung** | Wird erbracht | Fortschritt aktualisieren |
| ✅ **Abgeschlossen** | Dienstleistung erbracht | Fertig |
| ❌ **Storniert** | Anfrage storniert | Keine Aktion |

### Auf Anfrage antworten

1. Anfrage öffnen
2. Kundenbedürfnisse prüfen
3. Auf **"Antworten"** klicken
4. Antwort wählen:
   - **Akzeptieren** - Sie können die Dienstleistung erbringen
   - **Alternative vorschlagen** - Änderungen vorschlagen
   - **Ablehnen** - Kann nicht erfüllen
5. Ihre Nachricht hinzufügen
6. Angebot beifügen, falls akzeptiert
7. Senden

## Buchungslink-Integration

Ermöglichen Sie Kunden, direkt in Ihren Kalender zu buchen.

### Warum Buchungslinks verwenden?

- Kunden können Ihre Verfügbarkeit sehen
- Automatische Terminbuchung
- Weniger E-Mail-Austausch
- Professionelles Erscheinungsbild

### Unterstützte Tools

Sie können jedes externe Planungstool verwenden:
- **Calendly** - Beliebt und einfach zu bedienen
- **Cal.com** - Open-Source-Alternative
- **Microsoft Bookings** - Office 365-Integration
- **Andere** - Jedes Planungstool mit teilbarer URL

### Ihren Buchungslink einrichten

1. Gehen Sie zu **Einstellungen** → **Kontakt und Buchung**
2. Finden Sie den Abschnitt **"Buchungslink"**
3. Fügen Sie die URL Ihres Planungstools ein
4. Fügen Sie eine optionale Beschreibung hinzu (z.B. "Beratungstermin buchen")
5. Speichern

### Wie Kunden Ihren Link verwenden

Wenn Kunden Ihr Profil oder Ihre Dienstleistung sehen:

1. Sie sehen einen Button **"Termin buchen"**
2. Klick leitet zu Ihrem Planungstool weiter
3. Sie wählen einen verfügbaren Zeitslot
4. Sie erhalten eine Bestätigung
5. Sie erhalten eine Buchungsbenachrichtigung

## Promo-Codes

### Auf Promo-Codes zugreifen

Gehen Sie zu **Einstellungen** → **Promo-Code-Manager**

### Promo-Code erstellen

1. Klicken Sie auf **"Neuen Code hinzufügen"**
2. Details ausfüllen:

```

│        Neuen Promo-Code hinzufügen              [X] │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Code *                                             │
│  ┌─────────────────────────────────────────────┐   │
│  │  SERVICE15                                   │   │
│  └─────────────────────────────────────────────┘   │
│  Der Code, den Kunden verwenden werden             │
│                                                     │
│  Rabatt *                                           │
│  ┌─────────────────────────────────────────────┐   │
│  │  15% Rabatt auf erste Dienstleistung        │   │
│  └─────────────────────────────────────────────┘   │
│  Beschreiben Sie den angebotenen Rabatt            │
│                                                     │
│  Beschreibung (Optional)                            │
│  ┌─────────────────────────────────────────────┐   │

│  │  Gültig für alle Dienstleistungen.          │   │
```

#### Promo-Code-Felder

| Feld | Beschreibung | Erforderlich |
|------|--------------|--------------|

## Rabattkündigungen

### Was sind Rabattkündigungen?

Rabattkündigungen sind Vereinbarungen, bei denen Aktionspreise oder Rabatte formell beendet werden.

### Wie es funktioniert

1. Wird über das Admin-System verwaltet
2. Vorankündigung wird gesendet
3. Standardtarife gelten wieder
4. Historie wird aufbewahrt
