## Nachrichtensystem

Kommunizieren Sie sicher mit anderen Benutzern auf der Plattform.

### Auf Nachrichten zugreifen

Klicken Sie auf **Nachrichten** in der Seitenleiste.

### Hauptfunktionen

#### Gespräche

- **Gesprächsliste** - Alle Ihre Diskussionen
- **Ungelesene Nachrichten** - Durch blauen Punkt gekennzeichnet
- **Suche** - Gespräche finden
- **Sortierung** - Nach Datum, Ungelesen zuerst

#### Nachrichten senden

1. Wählen Sie ein Gespräch oder erstellen Sie ein neues
2. Geben Sie Ihre Nachricht ein
3. Klicken Sie auf **Senden** oder drücken Sie Enter

#### Anhänge

- Klicken Sie auf 📎, um Dateien anzuhängen
- Akzeptierte Formate: PDF, DOC, JPG, PNG
- Maximale Größe: 10 MB pro Datei

#### Neue Nachricht

1. Klicken Sie auf **"Neue Nachricht"**
2. Wählen Sie den Empfänger
3. Geben Sie Ihre Nachricht ein
4. Senden

### Statusindikatoren

| Indikator | Bedeutung |
|-----------|-----------|
| ✓ | Nachricht gesendet |
| ✓✓ | Nachricht gelesen |
| ● | Ungelesene Nachricht (blau) |
| 🕐 | Wird gesendet |

### Beste Praktiken

- ✅ Antworten Sie innerhalb von 24-48 Stunden
- ✅ Seien Sie professionell und höflich
- ✅ Verwenden Sie klare Betreffzeilen
- ✅ Hängen Sie relevante Dokumente an
- ❌ Kein Spam
- ❌ Keine sensiblen Informationen teilen

## Benachrichtigungen

Bleiben Sie über wichtige Aktivitäten informiert.

### Benachrichtigungstypen

| Typ | Beschreibung |
|-----|--------------|
| 🔔 **Echtzeit** | Sofortige Benachrichtigungen in der App |
| 📧 **E-Mail** | Benachrichtigungen per E-Mail |
| 📱 **Push** | Browser-Benachrichtigungen (falls aktiviert) |

### Benachrichtigungszentrum

Klicken Sie auf das 🔔 Symbol in der Navigationsleiste, um zu sehen:
- Neue Benachrichtigungen
- Benachrichtigungsverlauf
- Markierungsoptionen (gelesen/ungelesen)

### Benachrichtigungskategorien

#### Kontobenachrichtigungen
- Passwortänderungen
- E-Mail-Änderungen
- Sicherheitswarnungen

#### Aktivitätsbenachrichtigungen
- Neue Nachrichten
- Antworten auf Ihre Anfragen
- Statusaktualisierungen
- Neue Bewerbungen (Stiftungen)
- Neue Bestellungen (Lieferanten/Dienstleister)

#### Systembenachrichtigungen
- Geplante Wartung
- Neue Funktionen
- Änderungen der Bedingungen

### Benachrichtigungen konfigurieren

1. Gehen Sie zu **Einstellungen** → **Benachrichtigungseinstellungen**
2. Aktivieren/deaktivieren Sie jeden Typ
3. Wählen Sie die Digest-Häufigkeit:
   - **Sofort** - Jede Benachrichtigung
   - **Täglich** - Tägliche Zusammenfassung
   - **Wöchentlich** - Wöchentliche Zusammenfassung
   - **Keine** - Deaktiviert

## Dateiverwaltung

Laden Sie Ihre Dokumente auf der Plattform hoch und verwalten Sie sie.

### Wo Dateien hochladen

Je nach Ihrer Rolle:
- **Profil** - Profilbild, Logo
- **Nachrichten** - Anhänge
- **Produkte/Dienstleistungen** - Bilder
- **Bewerbungen** - Lebenslauf, Zertifikate
- **Support** - Screenshots

### Akzeptierte Formate

| Typ | Formate |
|-----|---------|
| **Bilder** | JPG, PNG, GIF, WebP |
| **Dokumente** | PDF, DOC, DOCX |
| **Tabellen** | XLS, XLSX |

### Größenbeschränkungen

| Kontext | Limit |
|---------|-------|
| Profilbild | 5 MB |
| Organisationslogo | 5 MB |
| Titelbild | 10 MB |
| Nachrichtenanhänge | 10 MB pro Datei |
| Bewerbungsdokumente | 10 MB pro Datei |

### Eine Datei hochladen

1. Klicken Sie auf **"Hochladen"** oder **"Datei wählen"**
2. Wählen Sie die Datei auf Ihrem Gerät
3. Warten Sie auf den Upload
4. Überprüfen Sie die Vorschau (Bilder)
5. Speichern

### Tipps

- 📷 Verwenden Sie hochwertige Bilder
- 📄 Benennen Sie Ihre Dateien klar
- 🗜️ Komprimieren Sie große Dateien
- 🔒 Laden Sie keine unnötigen sensiblen Infos hoch

## Profilbilder und Bilder

Personalisieren Sie Ihre Präsenz auf der Plattform.

### Profilbild

#### Für Einzelpersonen (Erzieher, Eltern)
- Klares Foto Ihres Gesichts
- Neutraler Hintergrund empfohlen
- Professionell und aktuell

#### Für Organisationen (Stiftungen, Lieferanten, Dienstleister)
- Unternehmenslogo
- Quadratisches Bild
- Gute Auflösung (min 200x200px)

### Titelbild (Organisationen)

- Banner, das Ihre Aktivität darstellt
- Breites Format (empfohlen: 1200x400px)
- Foto Ihrer Einrichtung oder Ihres Teams

### Ihr Foto ändern

1. Gehen Sie zu **Einstellungen** oder **Profil**
2. Klicken Sie auf das aktuelle Foto oder "Foto ändern"
3. Wählen Sie Ihr neues Bild
4. Bei Bedarf zuschneiden
5. Speichern

### Spezifikationen

| Element | Abmessungen | Format |
|---------|-------------|--------|
| Profilbild | 200x200px min | JPG, PNG |
| Logo | 200x200px min | JPG, PNG |
| Titelbild | 1200x400px | JPG, PNG |

## Spracheinstellungen

ProCrèche Solutions ist in mehreren Sprachen verfügbar.

### Verfügbare Sprachen

| Sprache | Code |
|---------|------|
| 🇬🇧 English | EN |
| 🇫🇷 Français | FR |
| 🇩🇪 Deutsch | DE |

### Sprache wechseln

#### Methode 1: Navigationsleiste
1. Klicken Sie auf den Sprachauswähler (EN/FR/DE)
2. Wählen Sie Ihre Sprache
3. Die Seite wird automatisch neu geladen

#### Methode 2: Einstellungen
1. Gehen Sie zu **Einstellungen**
2. Finden Sie **"Sprache"**
3. Wählen Sie Ihre Präferenz
4. Speichern

### Was sich ändert

- Benutzeroberfläche
- Systemnachrichten
- Plattform-E-Mails
- Dokumentation

### Was sich nicht ändert

- Von anderen Benutzern erstellte Inhalte
- Empfangene Nachrichten
- Produkt-/Dienstleistungsbeschreibungen

## Barrierefreiheit

ProCrèche Solutions verpflichtet sich zur Barrierefreiheit.

### Enthaltene Funktionen

#### Tastaturnavigation
- Tab zum Navigieren zwischen Elementen
- Enter zum Aktivieren von Buttons/Links
- Escape zum Schließen von Modalen
- Pfeiltasten für Dropdown-Menüs

#### Screenreader-Kompatibilität
- ARIA-Labels auf interaktiven Elementen
- Semantische Seitenstruktur
- Alternativtexte auf Bildern
- Ankündigungen dynamischer Änderungen

#### Kontrast und Lesbarkeit
- Hoher Farbkontrast
- Lesbare Textgröße (Minimum)
- Klare Schriftarten
- Kein Text auf komplexen Bildern

### Browser-Einstellungen

Sie können über Ihre Browser-Einstellungen weiter anpassen:
- Seitenzoom
- Textgröße
- Hoher Kontrastmodus
- Dunkelmodus (falls unterstützt)

### Ein Barrierefreiheitsproblem melden

Wenn Sie Schwierigkeiten haben:
1. Erstellen Sie ein Support-Ticket
2. Wählen Sie "Barrierefreiheit" als Kategorie
3. Beschreiben Sie das aufgetretene Problem
4. Erwähnen Sie verwendete Hilfstechnologien
