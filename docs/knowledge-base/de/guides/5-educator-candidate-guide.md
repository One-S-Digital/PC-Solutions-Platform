## Dashboard-Übersicht

Ihr Dashboard ist Ihr zentraler Ort zur Verwaltung Ihrer Jobsuche. Greifen Sie darauf zu, indem Sie auf **Dashboard** in der Seitenleiste klicken.

### Was Sie sehen

| Bereich | Beschreibung |
|---------|--------------|
| 📋 **Bewerbungsübersicht** | Gesamte Bewerbungen und deren Status |
| 💼 **Empfohlene Stellen** | Stellen, die zu Ihrem Profil passen |
| 🔔 **Letzte Aktivitäten** | Neueste Updates zu Ihren Bewerbungen |
| ✅ **Profilvollständigkeit** | Fortschritt zu einem vollständigen Profil |

### Schnellstatistiken

- **Eingereichte Bewerbungen** - Gesamt gesendete Bewerbungen
- **Ausstehend** - Warten auf Antwort
- **In Prüfung** - Arbeitgeber prüft Ihre Bewerbung
- **Akzeptiert** - Erhaltene Angebote

## Profilverwaltung

Ihr Profil ist Ihr Online-Lebenslauf. Stiftungen schauen es an, um zu entscheiden, ob sie Sie kontaktieren.

### Auf Ihr Profil zugreifen

1. Klicken Sie auf Ihr **Profilbild** oben rechts
2. Wählen Sie **"Profil bearbeiten"**, ODER
3. Gehen Sie zu **Einstellungen** → **Mein Profil**

### Profilbereiche

#### Persönliche Informationen

| Feld | Beschreibung | Tipps |
|------|--------------|-------|
| **Vollständiger Name** | Ihr Vor- und Nachname | Verwenden Sie Ihren legalen Namen |
| **E-Mail** | Kontaktadresse | Halten Sie sie aktuell |
| **Telefon** | Kontaktnummer | Für Arbeitgeberanrufe |
| **Standort** | Kanton/Region | Wo Sie arbeiten können |
| **Stadt** | Ihre Stadt | Für lokale Stellen |

#### Berufliche Zusammenfassung

| Feld | Beschreibung | Tipps |
|------|--------------|-------|
| **Bio** | Ihre Vorstellung | Beschreiben Sie Ihre Erfahrung und Philosophie |
| **Qualifikationen** | Abschlüsse und Zertifikate | Listen Sie alle relevanten Abschlüsse auf |
| **Jahre Erfahrung** | Gesamterfahrung | Seien Sie genau |
| **Gesprochene Sprachen** | Beherrschte Sprachen | Niveaus einbeziehen |
| **Fähigkeiten** | Schlüsselkompetenzen | Relevant für frühkindliche Bildung |

#### Jobpräferenzen

| Feld | Beschreibung | Optionen |
|------|--------------|----------|
| **Vertragsart** | Vollzeit, Teilzeit, etc. | Wählen Sie Ihre Präferenzen |
| **Verfügbarkeit** | Wann Sie anfangen können | Sofort oder bestimmtes Datum |
| **Gehaltsvorstellung** | Gewünschtes Gehalt | Optional aber empfohlen |
| **Umzugsbereitschaft** | Geografische Mobilität | Ja/Nein und Bedingungen |
| **Bevorzugte Regionen** | Gewünschte Kantone | Wo Sie arbeiten möchten |

#### Profilbild

- Laden Sie ein professionelles Foto hoch
- Empfohlen: Quadratisches Bild, klar und professionell
- Vermeiden: Urlaubsfotos, verschwommene Selfies

### Tipps für ein erfolgreiches Profil

- ✅ Vervollständigen Sie Ihr Profil zu 100%
- ✅ Verwenden Sie ein professionelles Foto
- ✅ Schreiben Sie eine ansprechende Bio
- ✅ Listen Sie alle Qualifikationen auf
- ✅ Halten Sie Ihre Verfügbarkeit aktuell
- ✅ Laden Sie Ihren Lebenslauf in die Galerie hoch

## Verfügbarkeitseinstellungen

Zeigen Sie Arbeitgebern, wann Sie für Arbeit verfügbar sind.

### Auf Verfügbarkeit zugreifen

1. Gehen Sie zu **Profil** oder **Einstellungen**
2. Finden Sie den Abschnitt **"Verfügbarkeit"**

### Wochenplan

Legen Sie Ihre regelmäßige Verfügbarkeit fest:

```
┌─────────────────────────────────────────────────────┐
│               Wochenplan                             │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Montag     [✓] Verfügbar     08:00 - 18:00        │
│  Dienstag   [✓] Verfügbar     08:00 - 18:00        │
│  Mittwoch   [✓] Verfügbar     08:00 - 12:00        │
│  Donnerstag [✓] Verfügbar     08:00 - 18:00        │
│  Freitag    [✓] Verfügbar     08:00 - 17:00        │
│  Samstag    [ ] Nicht verfügbar                      │
│  Sonntag    [ ] Nicht verfügbar                      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

Für jeden Tag:
1. Aktivieren Sie, ob Sie **verfügbar** sind
2. Legen Sie die **Startzeit** fest
3. Legen Sie die **Endzeit** fest

### Datumsausnahmen

Fügen Sie Ausnahmen für bestimmte Tage hinzu:

#### Nichtverfügbarkeit hinzufügen

Um bestimmte Daten zu blockieren (Urlaub, Termine, etc.):

1. Klicken Sie auf **"Ausnahme hinzufügen"**
2. Wählen Sie das **Datum**
3. Wählen Sie **"Nicht verfügbar"**
4. Fügen Sie einen optionalen Grund hinzu
5. Speichern

#### Ausnahme-Verfügbarkeit hinzufügen

Um sich an einem normalerweise blockierten Tag verfügbar zu machen:

1. Klicken Sie auf **"Ausnahme hinzufügen"**
2. Wählen Sie das **Datum**
3. Wählen Sie **"Verfügbar"**
4. Legen Sie die Zeiten fest
5. Speichern

### Verfügbarkeitskalender

Sehen Sie Ihre Verfügbarkeit in Kalenderform:
- **Grün** - Verfügbar
- **Rot** - Nicht verfügbar
- **Gelb** - Teilweise verfügbar

## Stellenangebote durchsuchen

Finden Sie Stellenangebote, die zu Ihren Fähigkeiten und Präferenzen passen.

### Auf Stellen zugreifen

Klicken Sie auf **Stellenangebote** in der Seitenleiste.

### Stellen suchen

Verwenden Sie Filter, um Ihre Suche einzugrenzen:

| Filter | Optionen |
|--------|----------|
| **Standort** | Kanton, Stadt |
| **Vertragsart** | Unbefristet, Befristet, Teilzeit, etc. |
| **Erforderliche Erfahrung** | Junior, Bestätigt, Senior |
| **Gehaltsrahmen** | Minimum/Maximum |
| **Veröffentlichungsdatum** | Letzte 24h, Woche, Monat |
| **Textsuche** | Schlüsselwörter in Titel/Beschreibung |

### Stelleninformationen

Jede Stelle zeigt:
- **Stellentitel**
- **Name der Stiftung**
- **Standort**
- **Vertragsart**
- **Gehaltsrahmen** (falls angegeben)
- **Veröffentlichungsdatum**
- **Beschreibungsvorschau**

### Details anzeigen

Klicken Sie auf eine Stelle, um zu sehen:
- Vollständige Stellenbeschreibung
- Verantwortlichkeiten
- Erforderliche Qualifikationen
- Angebotene Leistungen
- Stiftungsprofil
- Wie man sich bewirbt

## Bewerbungen verwalten

### Auf eine Stelle bewerben

1. Finden Sie eine interessante Stelle
2. Klicken Sie auf **"Bewerben"**
3. Füllen Sie das Bewerbungsformular aus:

| Feld | Beschreibung |
|------|--------------|
| **Anschreiben** | Erklären Sie, warum Sie der richtige Kandidat sind |
| **Lebenslauf** | Aus Ihrer Galerie auswählen oder hochladen |
| **Zusätzliche Dokumente** | Zertifikate, Diplome (optional) |
| **Verfügbarkeit** | Mögliches Startdatum |
| **Gehaltsvorstellung** | Ihre Erwartungen (optional) |

4. Überprüfen Sie Ihre Informationen
5. Klicken Sie auf **"Bewerbung absenden"**

### Ihre Bewerbungen verfolgen

Greifen Sie auf **Meine Bewerbungen** in der Seitenleiste zu.

### Bewerbungsstatus

| Status | Bedeutung | Nächster Schritt |
|--------|-----------|------------------|
| 📝 **Eingereicht** | Bewerbung gesendet | Auf Antwort warten |
| 👁️ **Angesehen** | Arbeitgeber hat sie gesehen | Gutes Zeichen, Geduld |
| 🔍 **In Prüfung** | Wird bewertet | Bleiben Sie erreichbar |
| 📞 **Vorstellungsgespräch** | Sie sind eingeladen | Bereiten Sie sich vor! |
| ✅ **Akzeptiert** | Angebot erhalten | Herzlichen Glückwunsch! |
| ❌ **Abgelehnt** | Nicht ausgewählt | Suche fortsetzen |
| ⏸️ **Pausiert** | Prozess pausiert | Geduld |

### Aktionen bei Bewerbungen

- **Anzeigen** - Details Ihrer Bewerbung ansehen
- **Zurückziehen** - Bewerbung stornieren (vor Entscheidung)
- **Kontaktieren** - Nachricht an den Arbeitgeber senden

## Dateigalerie

Speichern und verwalten Sie Ihre beruflichen Dokumente.

### Auf die Galerie zugreifen

Gehen Sie zu **Profil** → **Dateigalerie** oder **Dokumente**

### Dokumenttypen

| Typ | Verwendung |
|-----|------------|
| 📄 **Lebenslauf** | Für Bewerbungen anhängen |
| 📜 **Diplome** | Qualifikationsnachweise |
| 🏆 **Zertifikate** | Zusätzliche Weiterbildungen |
| 📋 **Referenzen** | Empfehlungsschreiben |
| 🖼️ **Portfolio** | Arbeitsbeispiele (optional) |

### Dateien hochladen

1. Klicken Sie auf **"Hochladen"**
2. Wählen Sie Ihre Datei
3. Fügen Sie einen beschreibenden Titel hinzu
4. Wählen Sie die Kategorie
5. Speichern

### Akzeptierte Formate

- **Dokumente:** PDF, DOC, DOCX
- **Bilder:** JPG, PNG

### Tipps

- 📁 Halten Sie Ihren Lebenslauf aktuell
- 🏷️ Benennen Sie Dateien klar
- 📂 Nach Kategorie organisieren
- 🔒 Dateien sind standardmäßig privat

## Nachrichten

Kommunizieren Sie mit Stiftungen und Personalvermittlern.

### Auf Nachrichten zugreifen

Klicken Sie auf **Nachrichten** in der Seitenleiste.

### Nachrichten empfangen

Stiftungen können Sie kontaktieren, um:
- Fragen zu Ihrem Profil zu stellen
- Ein Vorstellungsgespräch zu planen
- Eine Gelegenheit zu besprechen

### Nachrichten senden

Sie können Stiftungen kontaktieren:
- Nach Bewerbung auf eine Stelle
- Um Fragen zu einer Position zu stellen
- Nach einer Einladung zur Kommunikation

### Beste Praktiken

- ✅ Antworten Sie schnell
- ✅ Seien Sie professionell
- ✅ Lesen Sie vor dem Senden Korrektur
- ✅ Stellen Sie relevante Fragen

Siehe [Allgemeine Funktionen - Nachrichten](./8-common-features.md#nachrichtensystem) für mehr Details.

## Einstellungen

Verwalten Sie Ihr Konto und Ihre Präferenzen.

### Auf Einstellungen zugreifen

Klicken Sie auf **Einstellungen** in der Seitenleiste.

### Einstellungsbereiche

#### Kontosicherheit
- Persönliche Informationen ändern
- Passwort ändern
- E-Mail-Adresse ändern

#### Benachrichtigungseinstellungen
- **Neue Stellen:** Benachrichtigungen für passende Stellen
- **Bewerbungs-Updates:** Statusänderungsbenachrichtigungen
- **Nachrichten:** Benachrichtigungen für neue Nachrichten
- **Digest-Häufigkeit:** Täglich, Wöchentlich oder Keine

#### Datenschutz und Daten
- **Profilsichtbarkeit:** Für Stiftungen sichtbar oder verborgen
- **Datenexport:** Ihre Informationen herunterladen
- **Kontolöschung:** DSGVO-Löschung anfordern

Siehe [Einstellungs-Guide](../reference/settings-guide.md) für detaillierte Anweisungen.

## Tipps für Ihre Jobsuche

### Profil optimieren

1. **Professionelles Foto** - Erster Eindruck zählt
2. **Ansprechende Bio** - Zeigen Sie Ihre Leidenschaft
3. **Vollständige Qualifikationen** - Alles auflisten
4. **Aktuelle Verfügbarkeit** - Rekrutierung erleichtern

### Effektiv bewerben

1. **Jede Bewerbung personalisieren** - Kein Copy-Paste
2. **Anschreiben** - Zeigen Sie Ihr Interesse
3. **Schlüsselwörter** - Begriffe aus der Stellenausschreibung verwenden
4. **Follow-up** - Proaktiv aber geduldig bleiben

### Vorstellungsgespräche vorbereiten

1. **Stiftung recherchieren** - Kennen Sie ihre Philosophie
2. **Beispiele vorbereiten** - Konkrete Situationen
3. **Fragen** - Welche Sie dem Arbeitgeber stellen möchten
4. **Präsentation** - Pünktlich und professionell sein
