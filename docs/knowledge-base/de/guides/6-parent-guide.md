## Dashboard-Übersicht

Ihr Dashboard ist Ihr zentraler Ort zur Verwaltung Ihrer Kinderbetreuungssuche.

### Auf das Dashboard zugreifen

Klicken Sie nach der Anmeldung auf **Dashboard** in der Seitenleiste.

### Was Sie sehen

| Bereich | Beschreibung |
|---------|--------------|
| 📋 **Meine Anfragen** | Übersicht Ihrer aktuellen Anfragen |
| 🔔 **Letzte Aktivitäten** | Neueste Antworten von Kitas |
| 📍 **Kitas in der Nähe** | Vorschläge basierend auf Ihrem Standort |
| ✉️ **Letzte Nachrichten** | Zuletzt erhaltene Nachrichten |

### Schnellstatistiken

- **Gesendete Anfragen** - Gesamt eingereichte Anfragen
- **Ausstehend** - Anfragen ohne Antwort
- **Erhaltene Antworten** - Kitas, die geantwortet haben
- **Aktive Gespräche** - Laufende Diskussionen

## Kita suchen

Finden Sie die ideale Kita für Ihr Kind.

### Auf die Suche zugreifen

1. Klicken Sie auf **Kita finden** in der Seitenleiste
2. Oder gehen Sie zu **Dashboard** → **Suchen**

### Suchoptionen

#### Verfügbare Filter

| Filter | Beschreibung |
|--------|--------------|
| **Kanton/Region** | Gewünschter Standort |
| **Stadt** | Bestimmte Stadt |
| **Betreuungsart** | Vollzeit, Teilzeit, gelegentlich |
| **Kindesalter** | Akzeptierte Altersgruppen |
| **Sprachen** | In der Kita gesprochene Sprachen |
| **Pädagogik** | Montessori, Reggio Emilia, etc. |

#### Namenssuche

Geben Sie den Namen einer Kita ein, wenn Sie sie bereits kennen.

### Suchergebnisse

Jede Kita zeigt:
- **Name** der Kita
- **Standort** (Stadt, Kanton)
- **Kapazität** verfügbar
- **Sprachen** gesprochen
- **Beschreibungsvorschau**
- **Profilfoto**

### Kita-Profil anzeigen

Klicken Sie auf eine Kita, um zu sehen:
- Vollständige Beschreibung
- Pädagogische Philosophie
- Angebotene Dienstleistungen
- Öffnungszeiten
- Kontaktinformationen
- Wie man eine Anfrage stellt

## Anfrage stellen

### Was ist eine Anfrage?

Eine Anfrage (oder "Lead") ist Ihr erster Kontakt mit einer Kita. Sie teilen Ihre Informationen und Bedürfnisse mit, und die Kita antwortet Ihnen.

### Anfrageformular

Wenn Sie eine Anfrage stellen, geben Sie an:

#### Informationen über das Kind

| Feld | Beschreibung | Beispiel |
|------|--------------|----------|
| **Name des Kindes** | Vorname Ihres Kindes | Emma |
| **Alter/Geburtsdatum** | Aktuelles Alter | 2 Jahre |
| **Besondere Bedürfnisse** | Allergien, Erkrankungen | Nussallergie |

#### Ihre Präferenzen

| Feld | Beschreibung | Beispiel |
|------|--------------|----------|
| **Startdatum** | Wann Sie Betreuung benötigen | 1. September 2025 |
| **Betreuungsart** | Vollzeit, Teilzeit | Vollzeit |
| **Gewünschte Tage** | Wochentage | Montag bis Freitag |
| **Uhrzeiten** | Gewünschte Stunden | 8:00 - 18:00 |

#### Kontaktinformationen

| Feld | Beschreibung |
|------|--------------|
| **Ihr Name** | Aus Ihrem Profil vorausgefüllt |
| **E-Mail** | Für Antworten |
| **Telefon** | Direkter Kontakt |
| **Standort** | Ihre Adresse/Stadt |

#### Persönliche Nachricht

Fügen Sie eine Nachricht hinzu, um sich vorzustellen und Ihre spezifischen Bedürfnisse zu erklären.

### Anfrage absenden

1. Füllen Sie alle Pflichtfelder aus
2. Fügen Sie eine persönliche Nachricht hinzu
3. Überprüfen Sie Ihre Informationen
4. Klicken Sie auf **"Anfrage senden"**

> 💡 **Tipp:** Eine persönliche Nachricht erhöht Ihre Chancen auf eine Antwort!

## Anfragen verwalten

Verfolgen Sie alle Ihre laufenden Anfragen.

### Auf Anfragen zugreifen

Klicken Sie auf **Meine Anfragen** in der Seitenleiste.

### Anfragenliste

Sehen Sie alle Ihre Anfragen mit:
- Name der Kita
- Sendedatum
- Aktueller Status
- Letzte Aktivität

### Anfragestatus

| Status | Bedeutung | Nächster Schritt |
|--------|-----------|------------------|
| 📝 **Gesendet** | Anfrage eingereicht | Auf Antwort warten |
| 👁️ **Angesehen** | Kita hat sie gesehen | Gutes Zeichen! |
| 💬 **Antwort erhalten** | Kita hat geantwortet | Lesen und antworten |
| 📞 **Kontakt läuft** | Aktive Diskussion | Austausch fortsetzen |
| ✅ **Akzeptiert** | Platz bestätigt | Herzlichen Glückwunsch! |
| ❌ **Abgelehnt** | Kein Platz verfügbar | Anderswo versuchen |

### Anfragedetails anzeigen

Klicken Sie auf eine Anfrage, um zu sehen:
- Ihre eingereichten Informationen
- Antwort der Kita
- Austauschverlauf
- Verfügbare Aktionen

### Mögliche Aktionen

| Aktion | Wann verwenden |
|--------|----------------|
| **Anzeigen** | Details ansehen |
| **Antworten** | Nachricht senden |
| **Abbrechen** | Anfrage zurückziehen |
| **Archivieren** | Aus aktiver Liste verbergen |

## Nachrichten

Kommunizieren Sie direkt mit Kitas.

### Auf Nachrichten zugreifen

Klicken Sie auf **Nachrichten** in der Seitenleiste.

### Ihre Gespräche

Die Liste zeigt:
- Name der Kita
- Letzte Nachricht
- Datum/Uhrzeit
- Ungelesene Nachrichten (Badge)

### Lesen und antworten

1. Klicken Sie auf ein Gespräch
2. Lesen Sie die Nachrichten
3. Geben Sie Ihre Antwort ein
4. Klicken Sie auf **Senden**

### Neues Gespräch starten

Sie können eine Kita kontaktieren:
1. Nach dem Senden einer Anfrage
2. Nach einer Antwort auf Ihre Anfrage
3. Über den "Kontaktieren"-Button auf ihrem Profil

### Kommunikationstipps

- ✅ Seien Sie klar und präzise
- ✅ Antworten Sie schnell
- ✅ Stellen Sie Ihre Fragen
- ✅ Erwähnen Sie Ihr Kind und seine Bedürfnisse
- ✅ Seien Sie höflich und professionell

Siehe [Allgemeine Funktionen - Nachrichten](./8-common-features.md#nachrichtensystem) für mehr Details.

## Support und Hilfe

### Support-Ticket erstellen

1. Klicken Sie auf **Support** in der Seitenleiste
2. Klicken Sie auf **"Ticket erstellen"**
3. Ausfüllen:
   - **Betreff** - Kurze Beschreibung
   - **Kategorie** - Allgemein, Technisch, Konto
   - **Nachricht** - Detaillierte Erklärung
4. Klicken Sie auf **Absenden**

### Schnelle FAQ

**F: Wie viele Anfragen kann ich senden?**
A: Es gibt keine Grenze. Sie können so viele Kitas kontaktieren, wie Sie möchten.

**F: Wie lange dauert es, eine Antwort zu erhalten?**
A: Das hängt von der Kita ab. Die meisten antworten innerhalb von 1-5 Werktagen.

**F: Kann ich eine gesendete Anfrage ändern?**
A: Sie können eine gesendete Anfrage nicht ändern, aber Sie können der Kita eine Nachricht mit aktualisierten Informationen senden.

## Tipps für die Kita-Suche

### Bereiten Sie Ihre Suche vor

1. **Definieren Sie Ihre Kriterien** - Standort, Zeiten, Pädagogik
2. **Listen Sie Ihre Bedürfnisse auf** - Allergien, besondere Bedürfnisse
3. **Legen Sie ein Budget fest** - Haben Sie eine Vorstellung, was Sie ausgeben können
4. **Bereiten Sie Fragen vor** - Was Sie wissen möchten

### Maximieren Sie Ihre Chancen

1. **Vervollständigen Sie Ihr Profil** - Mehr Infos = bessere Antworten
2. **Personalisieren Sie Ihre Nachrichten** - Vermeiden Sie generische Nachrichten
3. **Antworten Sie schnell** - Zeigen Sie Ihr Interesse
4. **Seien Sie flexibel** - Mehr Flexibilität = mehr Optionen

### Bei Besichtigungen

1. **Bereiten Sie Fragen vor** - Pädagogik, Mahlzeiten, Aktivitäten
2. **Beobachten Sie die Umgebung** - Sauberkeit, Sicherheit, Atmosphäre
3. **Treffen Sie das Personal** - Bewerten Sie die Interaktionen
4. **Vertrauen Sie Ihrem Bauchgefühl** - Die Atmosphäre zählt
