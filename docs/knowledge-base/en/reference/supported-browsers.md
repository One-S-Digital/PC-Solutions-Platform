## Troubleshooting Browser Issues

### Platform Not Working Correctly?

1. **Update your browser**
   - Check for updates in browser settings
   - Install the latest version

2. **Clear cache and cookies**
   - Windows: Ctrl+Shift+Delete
   - Mac: Cmd+Shift+Delete
   - Select "All time" and clear

3. **Try incognito/private mode**
   - This rules out extension conflicts
   - Chrome: Ctrl+Shift+N
   - Firefox: Ctrl+Shift+P

4. **Disable extensions**
   - Ad blockers can interfere
   - Try disabling temporarily

5. **Try a different browser**
   - If one browser fails, try another
   - Chrome is recommended

### Common Extension Conflicts

These extensions may cause issues:

| Extension Type | Potential Issue |
|----------------|-----------------|
| **Ad Blockers** | May block API calls |
| **Privacy Extensions** | May block cookies |
| **VPN Extensions** | May slow connections |
| **Script Blockers** | May disable features |

**Solution:** Add ProCrèche Solutions to your extension's whitelist.

## Enabling JavaScript

JavaScript is required. If disabled:

### Chrome
1. Click ⋮ menu → Settings
2. Privacy and Security → Site Settings
3. JavaScript → Enable

### Firefox
1. Type `about:config` in address bar
2. Search for `javascript.enabled`
3. Set to `true`

### Safari
1. Safari menu → Preferences
2. Security tab
3. Check "Enable JavaScript"

### Edge
1. Click ... menu → Settings
2. Cookies and site permissions
3. JavaScript → Allow

---

## Enabling Cookies

Cookies are required for login. If blocked:

### Chrome
1. Click ⋮ menu → Settings
2. Privacy and Security → Cookies
3. Allow cookies (or add site exception)

### Firefox
1. Click ☰ menu → Settings
2. Privacy & Security
3. Cookies → Accept

### Safari
1. Safari menu → Preferences
2. Privacy tab
3. Uncheck "Block all cookies"
