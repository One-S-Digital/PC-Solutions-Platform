## Messaging System

Communicate with other users directly through the platform.

### Accessing Messages

Click **Messages** in the sidebar to open the messaging center.

### Starting a New Conversation

1. Click **"+ New Message"** or **"New Conversation"**
2. Select the recipient:
   - Search by name
   - Select from recent contacts
3. Type your message
4. Click **Send**

### Individual Chats

One-on-one conversations between two users.

**Features:**
- Real-time message delivery
- Read receipts (when messages are seen)
- Message history
- File attachments

### Group Chats

Conversations with multiple participants.

**To create a group chat:**
1. Click **"New Message"**
2. Select **"Create Group"**
3. Add participants
4. Name the group (optional)
5. Start messaging

### Sending Messages

1. Open a conversation
2. Type in the message field
3. Press **Enter** or click **Send**

### Attaching Files

1. Click the attachment icon 📎
2. Select a file from your device
3. File uploads and appears in the message
4. Send

**Supported attachments:**
- Documents (PDF, Word)
- Images (JPEG, PNG)
- Spreadsheets

### Message Notifications

You'll be notified of new messages:
- 🔔 In-app notification badge
- 📧 Email notification (if enabled)
- 🔊 Browser notification (if permitted)

### Tips for Effective Communication

- ✅ Be professional and courteous
- ✅ Respond in a timely manner
- ✅ Be clear and concise
- ✅ Use attachments for important documents
- ✅ Keep records of important agreements

## Notifications

Stay informed about important updates.

### Accessing Notifications

Click the 🔔 **bell icon** in the top navigation bar.

### Notification Types

| Type | Description |
|------|-------------|
| 💬 **Messages** | New messages received |
| 📋 **Applications** | Job application updates |
| 📦 **Orders** | Order status changes |
| 📝 **Enquiries** | New leads or responses |
| ⚙️ **System** | Platform announcements |

### Notification Center


### Notification Indicators

- 🔵 **Blue dot** = Unread notification
- ⚪ **No dot** = Already read

### Managing Notifications

**Mark as Read:**
- Click on a notification to mark it read
- Click **"Mark All Read"** to clear all

**Notification Settings:**
1. Go to **Settings** → **Notifications**
2. Configure your preferences:
   - Email notifications (on/off)
   - Digest frequency (Daily/Weekly/None)
   - Specific notification types

### Notification Preferences

Customize which notifications you receive:

| Setting | Options |
|---------|---------|
| **Email Notifications** | On / Off |
| **Digest Frequency** | Daily / Weekly / None |
| **New Request Alerts** | On / Off |
| **Message Alerts** | On / Off |
| **Promo Redemption Alerts** | On / Off |

## File Management

Upload, manage, and share files on the platform.

### Uploading Files

Files can be uploaded in various places:
- Profile photos
- Cover images
- Documents (CV, catalogs)
- Message attachments

### How to Upload

1. Click the upload button or area
2. Select file from your device
3. Wait for upload to complete
4. Save changes

### Supported File Types

| Category | Formats |
|----------|---------|
| **Documents** | PDF, DOC, DOCX |
| **Images** | JPEG, JPG, PNG, GIF, WebP |
| **Spreadsheets** | XLS, XLSX, CSV |

### File Size Limits

- **Profile pictures:** Up to 5 MB
- **Cover images:** Up to 10 MB
- **Documents:** Up to 25 MB
- **Attachments:** Up to 10 MB

### Secure File Access

- Files are stored securely
- Only authorized users can access your files
- Files are scanned for security

### Document Preview

Many file types can be previewed without downloading:
- PDFs display in-browser
- Images show thumbnails
- Documents may show preview

## Profile Pictures & Images

Make your profile stand out with quality images.

### Profile Photo

Your profile photo appears:
- In the navigation bar
- On your public profile
- In messages
- In listings (for organizations)

**To upload/change:**
1. Go to your profile
2. Click on your current photo or the camera icon
3. Select a new image
4. Crop if needed
5. Save

**Best practices:**
- Use a clear, professional image
- Face should be visible (for individuals)
- Logo should be high-resolution (for organizations)
- Square format works best

### Cover Image

Cover images appear on profile pages as a banner.

**To upload/change:**
1. Go to your profile
2. Click **"Edit Cover"** or the camera icon on the banner
3. Select an image
4. Position if needed
5. Save

**Best practices:**
- Use a wide, horizontal image
- Minimum 1200 x 400 pixels
- Represents your brand or personality

### Image Cropping

When uploading, you may be asked to crop:
1. Drag to position the image
2. Resize the crop area
3. Confirm the crop
4. Save

### Recommended Image Sizes

| Image Type | Recommended Size |
|------------|------------------|
| Profile Photo | 400 x 400 px |
| Cover Image | 1200 x 400 px |
| Product Image | 800 x 800 px |
| Service Image | 800 x 600 px |

## Language Settings

Use the platform in your preferred language.

### Available Languages

- 🇬🇧 **English** (EN)
- 🇫🇷 **Français** (FR)
- 🇩🇪 **Deutsch** (DE)

### Changing Your Language

**Method 1: Language Switcher**
1. Look for the language selector in the navigation bar
2. Click it
3. Select your preferred language
4. The page refreshes in the new language

**Method 2: Settings**
1. Go to **Settings**
2. Find **Language** or **Preferences**
3. Select your language
4. Save

### What Changes

When you change languages:
- All interface text changes
- Buttons and labels update
- Dates may format differently
- User-generated content (messages, listings) stays in original language

## Accessibility Features

We strive to make the platform accessible to everyone.

### Keyboard Navigation

Navigate without a mouse:
- **Tab** - Move between elements
- **Enter** - Activate buttons/links
- **Escape** - Close modals/dropdowns
- **Arrow keys** - Navigate menus

### Screen Reader Compatibility

The platform is designed to work with screen readers:
- Proper heading structure
- Alt text for images
- ARIA labels for interactive elements
- Form labels and descriptions

### Color Contrast

- Text meets WCAG contrast requirements
- Important information isn't conveyed by color alone
- High-contrast mode compatible

### Text Sizing

- Browser zoom works correctly
- Text scales properly
- Layouts remain functional when zoomed

### Reducing Motion

If you're sensitive to motion:
1. Enable "Reduce Motion" in your device settings
2. Animations will be minimized

### Need Accessibility Help?

If you encounter accessibility issues:
1. Create a support ticket
2. Select "Accessibility" as the category
3. Describe the issue
4. We'll work to resolve it
