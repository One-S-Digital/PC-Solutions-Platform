## Organization Profile

Your company profile is visible to foundations browsing the marketplace. A complete profile builds trust and attracts customers.

### Accessing Your Profile

1. Click **Organization Profile** in the sidebar, OR
2. Go to **Settings** → **Edit Profile**

### Profile Information

#### Basic Company Details

| Field | Description | Tips |
|-------|-------------|------|
| **Company Name** | Your business name | Use your official registered name |
| **Contact Person** | Primary contact | Sales or account manager |
| **Phone Number** | Business phone | Include country code |
| **Contact Email** | Public email | For customer inquiries |
| **Canton/Region** | Your location | Where you're based |
| **City** | Your city | Helps customers find local suppliers |

#### Business Details

| Field | Description | Tips |
|-------|-------------|------|
| **Description** | About your company | Describe your products and values |
| **Product Categories** | Types of products you sell | Select all relevant categories |
| **Regions Served** | Where you deliver | Select multiple cantons |
| **Languages** | Languages you support | For customer communication |
| **VAT Number** | Tax identification | Required for invoicing |
| **Minimum Order Quantity** | Minimum order size | Set if applicable |
| **Direct Order Link** | External ordering URL | Link to your own shop (optional) |
| **Catalog URL** | Link to product catalog | PDF or online catalog |

#### Visual Elements

| Element | Recommendation |
|---------|----------------|
| **Logo** | Square image, min 200x200px |
| **Cover Image** | Wide banner, min 1200x400px |

### Uploading Your Catalog

You can upload a product catalog for customers to browse:

1. Go to **Organization Profile**
2. Scroll to **Catalog** section
3. Click **"Upload Catalog"**
4. Select your PDF file
5. Add a title and description
6. Save

> 💡 **Tip:** Keep your catalog updated with current products and pricing.

## Product Listings

Manage your products that appear in the marketplace.

### Accessing Products

Click **Product Listings** in the sidebar.

### Creating Products

1. Click **"Create Product"**
2. Fill in product details:

#### Basic Information

| Field | Description | Required |
|-------|-------------|----------|
| **Title** | Product name | ✅ Yes |
| **Subtitle** | Brief tagline | ❌ Optional |
| **Description** | Full product description | ✅ Yes |
| **Price** | Product price | ✅ Yes |
| **Currency** | CHF, EUR, etc. | ✅ Yes (default: CHF) |

#### Categorization

| Field | Description |
|-------|-------------|
| **Primary Category** | Main product category |
| **Additional Categories** | Secondary categories |
| **Tags** | Keywords for search |
| **Product Highlights** | Key selling points |

#### Inventory Details

| Field | Description |
|-------|-------------|
| **SKU** | Your internal product code |
| **Vendor SKU** | Alternative product code |
| **EAN** | Barcode number (if applicable) |
| **Unit of Measure** | Piece, Pack, Box, etc. |
| **Min Order Quantity** | Minimum units per order |
| **Max Order Quantity** | Maximum units per order |
| **Stock Status** | In Stock, Low Stock, Out of Stock |
| **Delivery Lead Time** | Days to deliver |

#### Product Image

- Click **"Upload Image"**
- Select a high-quality product photo
- Recommended: Square image, min 500x500px

3. Set product status:

| Status | Visibility |
|--------|------------|
| **Active** | Visible in marketplace |
| **Inactive** | Hidden from marketplace |
| **Draft** | Not published yet |
| **Out of Stock** | Shows as unavailable |

4. Click **"Save Product"**

### Managing Products

#### Editing a Product
1. Find the product in your list
2. Click **"Edit"** or the product name
3. Make changes
4. Click **"Save"**

#### Changing Status
1. Click the status badge on a product
2. Select new status
3. Changes save automatically

#### Deleting a Product
1. Click **"Delete"** on the product
2. Confirm deletion

> ⚠️ **Warning:** Deleting a product is permanent and cannot be undone.

### Filtering Products

Use filters to find products:
- **Status** - Active, Inactive, Draft
- **Category** - Filter by product category
- **Search** - Search by name or SKU

## Order Management

Handle orders from foundations.

### Accessing Orders

Click **Orders** in the sidebar.

### Viewing Orders

Your orders list shows:
- Order ID
- Foundation name
- Order date
- Items ordered
- Total amount
- Current status

### Order Details

Click an order to see:
- Full order information
- Customer (Foundation) details
- Line items with quantities and prices
- Delivery address
- Order notes

### Order Statuses

| Status | Meaning | Your Action |
|--------|---------|-------------|
| 📝 **Pending** | New order received | Review and accept |
| ✅ **Accepted** | You confirmed the order | Prepare for shipping |
| 🔄 **Processing** | Being prepared | Update when ready |
| 🚚 **Shipped** | Sent to customer | Provide tracking info |
| ✅ **Delivered** | Successfully delivered | Complete |
| ❌ **Cancelled** | Order cancelled | No action needed |

### Updating Order Status

1. Open the order
2. Click **"Update Status"**
3. Select the new status
4. Add notes (optional)
5. Save

### Workflow Example

```
New Order Received
       ↓
   [Accept Order]
       ↓
   Processing
       ↓
   [Mark as Shipped]
       ↓
     Shipped
       ↓
   [Mark as Delivered]
       ↓
    Delivered ✓
```

## Inquiries Management

Handle questions and requests from potential customers.

### What are Inquiries?

Inquiries are messages from foundations who want to learn more about your products before ordering.

### Inquiry Information

Each inquiry contains:
- Foundation details (name, contact)
- Subject line
- Message content
- Product of interest (if specified)
- Quantity needed
- Budget range
- Urgency level
- Contact preferences

### Inquiry Statuses

| Status | Meaning | Your Action |
|--------|---------|-------------|
| 🆕 **New** | Just received | Review and respond |
| ⏳ **Pending** | Awaiting your response | Respond soon |
| 📞 **Contacted** | You've responded | Follow up if needed |
| 💰 **Quoted** | Quote provided | Wait for decision |
| ✅ **Fulfilled** | Converted to order | Complete |
| ❌ **Declined** | You declined | No action |
| 🚫 **Cancelled** | Customer cancelled | No action |

### Responding to Inquiries

1. Open the inquiry
2. Review the customer's needs
3. Click **"Respond"**
4. Provide:
   - Your response message
   - Quote amount (if applicable)
   - Additional information
5. Update status
6. Send

## Promo Codes

Create promotional codes to attract customers. Promo codes are displayed on your public profile.

### Accessing Promo Codes

Go to **Settings** → **Promo Code Manager**

### Understanding Promo Codes

Promo codes let you offer discounts to customers. They appear on your supplier profile so foundations can see and use them.

### Creating a Promo Code

1. Click **"Add New Code"**
2. A modal window opens with these fields:

```
┌─────────────────────────────────────────────────────┐
│         Add New Promo Code                      [X] │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Code *                                             │
│  ┌─────────────────────────────────────────────┐   │
│  │  SAVE20                                      │   │
│  └─────────────────────────────────────────────┘   │
│  The code customers will use (e.g., SAVE20)        │
│                                                     │
│  Discount *                                         │
│  ┌─────────────────────────────────────────────┐   │
│  │  20% off your first order                   │   │
│  └─────────────────────────────────────────────┘   │
│  Describe the discount (e.g., 20% off, Free ship)  │
│                                                     │
│  Description (Optional)                             │
│  ┌─────────────────────────────────────────────┐   │
│  │  Valid for new customers only. Cannot be    │   │
│  │  combined with other offers.                │   │
│  └─────────────────────────────────────────────┘   │
│  Optional details about terms or conditions        │
│                                                     │
│  ☑ Show on profile                                 │
│                                                     │
│         [Cancel]              [Save]               │
└─────────────────────────────────────────────────────┘
```

#### Promo Code Fields

| Field | Description | Required | Example |
|-------|-------------|----------|---------|
| **Code** | The code customers use | ✅ Yes | `SAVE20`, `FREESHIP`, `WELCOME10` |
| **Discount** | What the customer gets | ✅ Yes | "20% off", "Free shipping", "CHF 10 off" |
| **Description** | Terms and conditions | ❌ Optional | "Valid for orders over CHF 100" |
| **Show on Profile** | Display publicly | ✅ Yes (default: on) | Toggle on/off |

3. Click **"Save"**

### Promo Code Tips

- ✅ **Code names:** Use memorable, easy-to-type codes
- ✅ **Discount text:** Be clear about what customers receive
- ✅ **Terms:** Add conditions in the description if needed
- ✅ **Hide old codes:** Turn off "Show on profile" for expired promos

### Managing Promo Codes

Your promo codes are displayed in a table:

| Code | Discount | Status | Actions |
|------|----------|--------|---------|
| `SAVE20` | 20% off first order | 🟢 Active | Edit · Delete |
| `FREESHIP` | Free shipping over CHF 100 | 🟢 Active | Edit · Delete |
| `OLDCODE` | 10% off | 🔴 Hidden | Edit · Delete |

#### Editing a Promo Code
1. Click **"Edit"** on the code
2. Modify the details
3. Click **"Save"**

#### Deleting a Promo Code
1. Click **"Delete"** on the code
2. Confirm the deletion

#### Hiding a Promo Code
1. Click **"Edit"**
2. Uncheck **"Show on profile"**
3. Save

> 💡 **Tip:** Instead of deleting old promo codes, hide them so you have a record.

### How Customers See Promo Codes

When foundations view your profile, they see your active promo codes in a dedicated section:

## Discount Terminations

### What Are Discount Terminations?

Discount terminations are special arrangements where promotional pricing or discounts are formally ended. This may occur when:

- A promotional period expires
- A special pricing agreement ends
- Volume discounts no longer apply
- Partnership terms change

### How It Works

1. Discount terminations are managed through the admin system
2. You'll be notified in advance of any termination
3. Standard pricing resumes after termination
4. Historical discount records are maintained

### What This Means for You

- Review your notification settings to receive termination notices
- Plan for price changes when promotional periods end
- Contact support if you have questions about discount arrangements
