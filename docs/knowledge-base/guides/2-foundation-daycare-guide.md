## Dashboard Overview

Your dashboard is your home base. Access it by clicking **Dashboard** in the sidebar or navigating to `/foundation/dashboard`.

### Quick Stats Cards

At the top of your dashboard, you'll see key metrics at a glance:

| Stat | What it shows |
|------|---------------|
| 📋 **Total Job Listings** | Number of jobs you've posted |
| ⏳ **Pending Applications** | Applications awaiting your review |
| 📝 **Total Applications** | All applications received |
| 🛒 **Total Orders** | Orders placed in marketplace |
| 🔧 **Service Requests** | Services you've requested |
| ✅ **Active Jobs** | Currently published job listings |

### Recent Activities Feed

Below the stats, see your latest activities:
- New parent leads received
- Job applications submitted
- Order updates
- Service request updates
- New messages

### Calendar Preview

View upcoming events:
- Interviews scheduled
- Service appointments
- Important deadlines
- Custom reminders

### Creating a Calendar Event

1. Go to **Dashboard**
2. Find the Calendar section
3. Click **"Add Event"**
4. Fill in:
   - **Title** - Name of the event
   - **Type** - Appointment, Interview, Service, Meeting, or Reminder
   - **Date & Time** - When it occurs
   - **Description** - Additional details (optional)
   - **Location** - Where it takes place (optional)
5. Click **Save**

## Organization Profile

Your organization profile is visible to parents, educators, and vendors. Keep it updated to attract the right connections.

### Accessing Your Profile

1. Click **Organization Profile** in the sidebar, OR
2. Go to **Settings** → **Edit Profile**

### Profile Information

#### Basic Information

| Field | Description | Tips |
|-------|-------------|------|
| **Organization Name** | Your daycare/foundation name | Use your official registered name |
| **Contact Person** | Primary contact name | Usually the director or manager |
| **Phone Number** | Business phone | Include country code |
| **Contact Email** | Public contact email | Can differ from login email |
| **Canton/Region** | Your location | Select from dropdown |
| **City** | Your city | Helps parents find you |
| **Address** | Street address | For service deliveries |

#### Detailed Information

| Field | Description | Tips |
|-------|-------------|------|
| **About/Description** | Describe your daycare | Highlight your philosophy and what makes you unique |
| **Capacity** | Number of children you accommodate | Enter total licensed capacity |
| **Languages Spoken** | Languages offered | Select all that apply |
| **Pedagogy** | Teaching approaches | Montessori, Reggio Emilia, etc. |
| **VAT Number** | For invoicing (if applicable) | Optional |
| **Regions Served** | Areas you serve | Select multiple cantons if applicable |

#### Visual Elements

| Element | Recommendation |
|---------|----------------|
| **Logo** | Square image, min 200x200px, shows in listings |
| **Cover Image** | Wide banner, min 1200x400px, shows on profile page |

### Saving Your Profile

1. Make your changes
2. Click **"Save Changes"** at the bottom
3. Wait for confirmation message

> 💡 **Tip:** Complete profiles appear higher in search results!

## Parent Leads (Enquiries)

Parent leads are enquiries from parents looking for daycare services. Responding quickly increases enrollment.

> 📋 **Requires:** Essential plan or higher

### Understanding the Leads Inbox

Access your leads by clicking **Leads** in the sidebar.

#### Lead Information

Each lead contains:
- **Parent Details** - Name, email, phone
- **Child Details** - Name, age
- **Preferences** - Location, start date, languages
- **Special Requirements** - Any specific needs
- **Message** - Personal note from the parent

#### Lead Statuses

| Status | Meaning | Action Needed |
|--------|---------|---------------|
| 🆕 **New** | Just received | Review and respond |
| 🔄 **Processing** | You're working on it | Continue follow-up |
| ✅ **Converted** | Successfully enrolled | Complete |

### Responding to a Lead

1. Click on a lead to open details
2. Review the parent's information and requirements
3. Click **"Respond"**
4. Choose your response:
   
   | Response | When to use |
   |----------|-------------|
   | **Interested** | You want to proceed with this family |
   | **Not Interested** | Not a good fit (full capacity, etc.) |
   | **Need More Info** | You need additional information |
   | **Enrolled** | The child has been enrolled |

5. Add a personal message (recommended)
6. Click **Send Response**

### Filtering Leads

Use filters to find specific leads:
- **Status** - New, Processing, Converted
- **Date Range** - Filter by when received
- **Search** - Search by parent name or child name

### Lead Statistics

Track your lead performance:
- Total leads received
- Response rate
- Conversion rate
- Average response time

## Marketplace Access

Browse and purchase products and services from verified vendors.

### Browsing Products

1. Click **Marketplace** in the sidebar
2. Select **Products** tab
3. Use filters:
   - **Category** - Toys, Furniture, Food, etc.
   - **Supplier** - Filter by specific vendor
   - **Search** - Find specific products
4. Click a product for details

### Browsing Services

1. In Marketplace, select **Services** tab
2. Filter by:
   - **Service Type** - Cleaning, Training, etc.
   - **Provider** - Filter by vendor
   - **Delivery Type** - On-site, Remote, Hybrid
3. Click a service for details

### Viewing Supplier/Provider Profiles

1. Click on a product or service
2. Click the vendor's name
3. View their full profile including:
   - Company information
   - All products/services
   - Contact details
   - Promo codes (if available)

### Placing Orders

To order products:

1. Find the product you want
2. Click **"Add to Cart"** or **"Order Now"**
3. Set the quantity
4. Review your cart
5. Click **"Place Order"**
6. Confirm order details
7. Complete the order

### Requesting Services

To request a service:

1. Find the service you want
2. Click **"Request Service"**
3. Fill in request details:
   - Preferred dates
   - Specific requirements
   - Additional notes
4. Submit your request
5. Wait for provider response

### Sending Inquiries

For custom requests or questions:

1. Find a supplier or provider
2. Click **"Send Inquiry"**
3. Fill in:
   - Subject
   - Message
   - Product/service of interest (optional)
   - Quantity needed
   - Budget range
   - Urgency level
4. Submit inquiry

## Recruitment Module

Post jobs and find qualified educators for your daycare.

> 📋 **Requires:** Professional plan or higher

### Accessing Recruitment

Click **Recruitment** in the sidebar.

### Creating a Job Listing

1. Go to **Recruitment** → **Job Listings**
2. Click **"Create Job Listing"**
3. Fill in job details:

| Field | Description | Tips |
|-------|-------------|------|
| **Title** | Job position name | Be specific: "Lead Toddler Teacher" |
| **Description** | Full job description | Include daily responsibilities |
| **Requirements** | What candidates need | List must-have qualifications |
| **Benefits** | What you offer | Highlight perks |
| **Responsibilities** | Day-to-day duties | Be detailed |
| **Qualifications** | Preferred background | Education, certifications |
| **Location** | Work location | City/canton |
| **Salary Range** | Compensation offered | Be transparent |
| **Contract Type** | Employment type | Full-time, Part-time, etc. |
| **Start Date** | When position begins | ASAP or specific date |

4. Choose status:
   - **Draft** - Save without publishing
   - **Published** - Make visible to candidates
5. Click **Save**

### Contract Types

| Type | Description |
|------|-------------|
| **Full-time** | Standard full-time position |
| **Part-time** | Reduced hours position |
| **CDI** | Indefinite contract (permanent) |
| **CDD** | Fixed-term contract |
| **Internship** | Training position |
| **Replacement** | Temporary replacement |
| **Temporary** | Short-term assignment |
| **Freelance** | Independent contractor |

### Managing Job Listings

- **Edit** - Click a listing to modify
- **Publish/Unpublish** - Toggle visibility
- **Close** - Stop accepting applications
- **Delete** - Remove the listing

### Viewing Applications

1. Click on a job listing
2. View **Applications** section
3. See all candidates who applied

### Application Details

Each application shows:
- Candidate profile
- Cover letter
- CV/Resume
- Application date
- Current status

### Managing Applications

Update application status:

| Status | When to use |
|--------|-------------|
| **Pending** | Just received, not yet reviewed |
| **Reviewed** | You've looked at it |
| **Accepted** | Moving forward/offering position |
| **Rejected** | Not selected |

To update:
1. Open the application
2. Click **"Update Status"**
3. Select new status
4. Add internal notes (optional)
5. Save

### Candidate Pool

Browse all available educators:
1. Go to **Recruitment** → **Candidate Pool**
2. Browse educator profiles
3. Filter by skills, location, availability
4. View full profiles
5. Contact candidates via messaging

## Support & Help

### Creating a Support Ticket

1. Click **Support** in the sidebar
2. Click **"Create Ticket"**
3. Fill in:
   - **Subject** - Brief description
   - **Category** - General, Technical, Billing, Feature Request
   - **Priority** - Low, Medium, High, Urgent
   - **Message** - Detailed explanation
   - **Attachments** - Screenshots or files (optional)
4. Click **Submit**

### Tracking Tickets

- View all your tickets
- Check status updates
- Add follow-up messages
- Close resolved tickets

### Ticket Statuses

| Status | Meaning |
|--------|---------|
| **Open** | Submitted, awaiting response |
| **In Progress** | Being worked on |
| **Resolved** | Issue fixed |
| **Closed** | Ticket complete |
