## Sécurité du compte

### Informations personnelles

Mettez à jour vos informations de base.

#### Champs disponibles

| Champ | Description | Modifiable |
|-------|-------------|------------|
| **Nom complet** | Votre nom | ✅ Oui |
| **Numéro de téléphone** | Contact | ✅ Oui |
| **Email** | Via changement d'email | Voir ci-dessous |

#### Comment modifier

1. Allez dans **Paramètres** → **Sécurité du compte**
2. Modifiez les champs souhaités
3. Cliquez sur **"Enregistrer les modifications"**

### Changer le mot de passe

```
┌─────────────────────────────────────────────────────┐
│  Changer le mot de passe                            │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Mot de passe actuel                               │
│  ┌─────────────────────────────────────────────┐   │
│  │ ••••••••••                               👁️ │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  Nouveau mot de passe                              │
│  ┌─────────────────────────────────────────────┐   │
│  │                                           👁️ │   │
│  └─────────────────────────────────────────────┘   │
│  Min. 8 caractères                                 │
│                                                     │
│  Confirmer le nouveau mot de passe                 │
│  ┌─────────────────────────────────────────────┐   │
│  │                                           👁️ │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│            [Mettre à jour le mot de passe]         │
│                                                     │
└─────────────────────────────────────────────────────┘
```

#### Étapes

1. Entrez votre **mot de passe actuel**
2. Entrez votre **nouveau mot de passe** (min. 8 caractères)
3. **Confirmez** le nouveau mot de passe
4. Cliquez sur **"Mettre à jour le mot de passe"**

#### Exigences du mot de passe

- ✅ Minimum 8 caractères
- ✅ Pas un mot de passe couramment utilisé
- ✅ Ne doit pas avoir été compromis dans une fuite de données

### Changer l'adresse email

```
┌─────────────────────────────────────────────────────┐
│  Changer l'adresse email                            │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Adresse email actuelle                            │
│  ancien.email@exemple.com                           │
│                                                     │
│  Nouvelle adresse email                            │
│  ┌─────────────────────────────────────────────┐   │
│  │ nouvel.email@exemple.com                     │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│            [Envoyer le code de vérification]       │
│                                                     │
└─────────────────────────────────────────────────────┘
```

#### Processus

1. Entrez votre **nouvelle adresse email**
2. Cliquez sur **"Envoyer le code de vérification"**
3. Vérifiez votre **nouvel email** pour le code
4. Entrez le **code à 6 chiffres**
5. Cliquez sur **"Vérifier et changer l'email"**

> ⚠️ **Important :** Votre nouvelle adresse doit être vérifiée avant que le changement ne prenne effet.

## Facturation et abonnement

*Disponible pour : Fondations, Fournisseurs, Prestataires*

### Vue d'ensemble

Voir votre plan actuel et gérer la facturation.

#### Informations affichées

| Élément | Description |
|---------|-------------|
| **Plan actuel** | Nom et niveau de votre abonnement |
| **Statut** | Actif, Inactif, En attente, Annulé |
| **Prix** | Coût mensuel ou annuel |
| **Prochain renouvellement** | Date du prochain paiement |
| **Méthode de paiement** | Carte enregistrée |

### Actions disponibles

| Action | Description |
|--------|-------------|
| **Demander un changement** | Mise à niveau ou rétrogradation |
| **Gérer le paiement** | Ajouter/modifier la carte |
| **Voir les factures** | Historique de facturation |
| **Demander l'annulation** | Annuler l'abonnement |

### Gérer le paiement

1. Cliquez sur **"Gérer le paiement"**
2. Options disponibles :
   - Ajouter une nouvelle carte
   - Modifier la carte existante
   - Supprimer une carte

### Voir l'historique

1. Cliquez sur **"Voir les factures"**
2. Voir la liste des factures
3. Téléchargez les PDF au besoin

## Gestionnaire de codes promo

*Disponible pour : Fournisseurs, Prestataires*

### Accéder aux codes promo

**Paramètres** → **Gestionnaire de codes promo**

### Créer un code promo

```
┌─────────────────────────────────────────────────────┐
│        Ajouter un nouveau code promo            [X] │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Code *                                             │
│  ┌─────────────────────────────────────────────┐   │
│  │  PROMO20                                     │   │
│  └─────────────────────────────────────────────┘   │
│  Le code que les clients utiliseront               │
│                                                     │
│  Réduction *                                        │
│  ┌─────────────────────────────────────────────┐   │
│  │  20% de réduction sur la première commande  │   │
│  └─────────────────────────────────────────────┘   │
│  Décrivez la réduction offerte                     │
│                                                     │
│  Description (Optionnel)                            │
│  ┌─────────────────────────────────────────────┐   │
│  │  Nouveaux clients uniquement.               │   │
│  │  Non cumulable avec d'autres offres.        │   │
│  └─────────────────────────────────────────────┘   │
│  Conditions et modalités                           │
│                                                     │
│  ☑ Afficher sur le profil                          │
│                                                     │
│         [Annuler]              [Enregistrer]       │
└─────────────────────────────────────────────────────┘
```

#### Champs

| Champ | Description | Obligatoire |
|-------|-------------|-------------|
| **Code** | Le code texte (ex: PROMO20) | ✅ Oui |
| **Réduction** | Ce que le client reçoit | ✅ Oui |
| **Description** | Conditions générales | ❌ Non |
| **Afficher sur le profil** | Visibilité publique | ✅ Oui (défaut) |

### Gérer les codes existants

| Action | Comment |
|--------|---------|
| **Modifier** | Cliquez sur "Modifier" sur le code |
| **Supprimer** | Cliquez sur "Supprimer" |
| **Masquer** | Décochez "Afficher sur le profil" |

## Préférences de notification

### Types de notifications

#### Emails

| Paramètre | Description | Rôles |
|-----------|-------------|-------|
| **Nouveaux messages** | Email quand vous recevez un message | Tous |
| **Nouvelles commandes** | Email pour les nouvelles commandes | Fournisseurs |
| **Nouvelles demandes** | Email pour les nouvelles demandes | Prestataires |
| **Nouveaux leads** | Email pour les nouveaux leads parents | Fondations |
| **Nouvelles candidatures** | Email pour les candidatures reçues | Fondations |
| **Mises à jour de candidatures** | Email sur vos candidatures | Éducateurs |
| **Utilisation de codes promo** | Quand vos codes sont utilisés | Fournisseurs, Prestataires |

### Fréquence du digest

Choisissez comment recevoir les résumés :

| Option | Description |
|--------|-------------|
| **Immédiat** | Notification à chaque événement |
| **Quotidien** | Résumé une fois par jour |
| **Hebdomadaire** | Résumé une fois par semaine |
| **Aucun** | Pas de digest (notifications individuelles seulement) |

### Configurer les notifications

1. Allez dans **Paramètres** → **Préférences de notification**
2. Activez/désactivez chaque type avec les interrupteurs
3. Sélectionnez la fréquence du digest
4. Les modifications s'enregistrent automatiquement

## Confidentialité et données

### Visibilité du profil

Contrôlez qui peut voir votre profil.

| Paramètre | Options | Description |
|-----------|---------|-------------|
| **Masquer publiquement** | Activé/Désactivé | Cache votre profil des recherches |

### Gestion des données

#### Exporter vos données

1. Cliquez sur **"Exporter mes données"**
2. Confirmez la demande
3. Recevez un email avec le lien de téléchargement
4. Téléchargez dans les 48 heures

Les données exportées incluent :
- Informations de profil
- Messages
- Historique d'activité
- Paramètres

#### Supprimer votre compte (RGPD)

1. Cliquez sur **"Supprimer mon compte"**
2. Lisez les informations importantes
3. Tapez "SUPPRIMER" pour confirmer
4. Confirmez définitivement

> ⚠️ **Attention :**
> - La suppression est définitive après 30 jours
> - Toutes vos données seront effacées
> - Les conversations seront conservées (messages anonymisés)

## Contact et réservation

*Disponible pour : Fournisseurs, Prestataires*

### Préférences de contact

| Paramètre | Description |
|-----------|-------------|
| **Email de contact** | Email affiché publiquement |
| **Téléphone de contact** | Numéro affiché publiquement |
| **Horaires de contact** | Quand vous êtes joignable |

### Lien de réservation

Permettez aux clients de prendre rendez-vous directement.

#### Configuration

1. Allez dans **Paramètres** → **Contact et réservation**
2. Trouvez **"Lien de réservation"**
3. Collez l'URL de votre outil de planification :
   - Calendly
   - Cal.com
   - Microsoft Bookings
   - Autre
4. Ajoutez une description (ex: "Réserver une consultation")
5. Enregistrez

#### Comment ça fonctionne

- Un bouton "Réserver" apparaît sur votre profil
- Les clients cliquent et sont redirigés vers votre calendrier
- Ils choisissent un créneau disponible
- Vous recevez une notification de réservation
