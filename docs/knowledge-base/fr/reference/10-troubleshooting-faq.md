## Problèmes de compte

### Je ne reçois pas l'email de vérification

**Causes possibles :**
- L'email est dans vos spams
- Adresse email mal saisie
- Problème de serveur temporaire

**Solutions :**
1. Vérifiez votre dossier **Spam/Indésirables**
2. Attendez 30 secondes et cliquez sur **"Renvoyer le code"**
3. Vérifiez l'orthographe de votre adresse email
4. Essayez avec une autre adresse email
5. Contactez le support si le problème persiste

---

### Mon code de vérification ne fonctionne pas

**Causes possibles :**
- Code expiré (valide ~10 minutes)
- Code mal saisi
- Vous avez demandé un nouveau code

**Solutions :**
1. Cliquez sur **"Renvoyer le code"** pour obtenir un nouveau code
2. Utilisez le code le plus récent
3. Vérifiez que vous tapez le bon code (6 chiffres)
4. Copiez-collez le code de l'email si possible

---

### Je ne peux pas me connecter

**Causes possibles :**
- Mot de passe incorrect
- Email incorrect
- Compte non vérifié

**Solutions :**
1. Vérifiez que l'adresse email est correcte
2. Utilisez **"Mot de passe oublié"** pour réinitialiser
3. Vérifiez que vous avez complété la vérification email
4. Essayez de vider le cache de votre navigateur
5. Essayez un autre navigateur

---

### J'ai oublié mon mot de passe

**Solution :**
1. Allez sur la page de connexion
2. Cliquez sur **"Mot de passe oublié"**
3. Entrez votre adresse email
4. Vérifiez votre boîte de réception
5. Cliquez sur le lien de réinitialisation
6. Créez un nouveau mot de passe
7. Connectez-vous avec le nouveau mot de passe

---

### J'ai oublié l'email de mon compte

**Solutions :**
1. Essayez toutes les adresses email que vous pourriez avoir utilisées
2. Vérifiez vos emails pour des messages de ProCrèche
3. Contactez le support avec des informations de vérification

---

### Je veux changer mon adresse email

**Étapes :**
1. Connectez-vous à votre compte
2. Allez dans **Paramètres** → **Sécurité du compte**
3. Trouvez **"Changer l'adresse email"**
4. Entrez votre nouvelle adresse email
5. Un code de vérification sera envoyé
6. Entrez le code pour confirmer

---

### Je veux supprimer mon compte

**Étapes :**
1. Allez dans **Paramètres** → **Confidentialité et données**
2. Cliquez sur **"Supprimer mon compte"**
3. Lisez les informations sur ce qui sera supprimé
4. Confirmez votre décision

> ⚠️ **Attention :** La suppression est définitive et irréversible après 30 jours.

## Problèmes de facturation

### Ma carte a été refusée

**Causes possibles :**
- Fonds insuffisants
- Carte expirée
- Limite de transaction atteinte
- Blocage par la banque

**Solutions :**
1. Vérifiez votre solde bancaire
2. Vérifiez la date d'expiration de la carte
3. Contactez votre banque pour débloquer
4. Essayez une autre carte
5. Contactez le support pour des options alternatives

---

### Je n'ai pas reçu ma facture

**Solutions :**
1. Vérifiez vos spams
2. Allez dans **Paramètres** → **Facturation** → **Historique**
3. Téléchargez la facture depuis votre compte
4. Contactez le support si elle n'apparaît pas

---

### Je veux un remboursement

**Processus :**
1. Créez un ticket de support
2. Sélectionnez "Facturation" comme catégorie
3. Expliquez votre situation
4. L'équipe examinera votre demande
5. Vous recevrez une réponse sous 2-3 jours

**Politique de remboursement :**
- Remboursements évalués au cas par cas
- Remboursement au prorata possible
- Pas de remboursement automatique

---

### Mon abonnement n'est pas actif

**Causes possibles :**
- Paiement échoué
- Abonnement expiré
- Traitement en cours

**Solutions :**
1. Vérifiez **Paramètres** → **Facturation**
2. Vérifiez le statut de paiement
3. Mettez à jour votre méthode de paiement
4. Attendez 24h pour le traitement
5. Contactez le support si le problème persiste

---

### Comment puis-je annuler mon abonnement ?

**Étapes :**
1. Allez dans **Paramètres** → **Facturation**
2. Cliquez sur **"Demander l'annulation"**
3. Sélectionnez une raison (optionnel)
4. Confirmez

**Important :**
- L'accès continue jusqu'à la fin de la période payée
- Vos données sont conservées 30 jours
- Vous pouvez réactiver à tout moment

## Questions spécifiques aux fonctionnalités

### Fondations

#### Les parents ne voient pas ma crèche

**Causes possibles :**
- Profil incomplet
- Pas d'abonnement actif
- Profil masqué

**Solutions :**
1. Complétez votre profil à 100%
2. Vérifiez votre abonnement
3. Vérifiez **Paramètres** → **Confidentialité** que vous êtes visible

---

#### Je ne reçois pas de leads parents

**Causes possibles :**
- Abonnement sans leads
- Profil non attrayant
- Zone géographique peu active

**Solutions :**
1. Vérifiez que votre plan inclut les leads
2. Améliorez votre profil (photos, description)
3. Élargissez vos régions desservies

---

### Fournisseurs/Prestataires

#### Mes produits/services ne s'affichent pas

**Causes possibles :**
- Statut "Inactif" ou "Brouillon"
- Pas d'abonnement actif
- Profil masqué

**Solutions :**
1. Vérifiez le statut de chaque produit/service
2. Assurez-vous qu'ils sont "Actifs"
3. Vérifiez votre abonnement

---

#### Je ne reçois pas de commandes/demandes

**Causes possibles :**
- Produits/services peu visibles
- Prix non compétitifs
- Profil incomplet

**Solutions :**
1. Optimisez vos titres et descriptions
2. Ajoutez des images de qualité
3. Vérifiez vos prix par rapport au marché
4. Créez des codes promo attractifs

---

### Éducateurs

#### Mon profil n'apparaît pas dans les recherches

**Causes possibles :**
- Profil incomplet
- Profil masqué
- Disponibilité non définie

**Solutions :**
1. Complétez votre profil à 100%
2. Vérifiez **Paramètres** → **Confidentialité**
3. Définissez votre disponibilité
4. Ajoutez une photo professionnelle

---

#### Je ne reçois pas de réponses à mes candidatures

**Conseils :**
1. Personnalisez chaque lettre de motivation
2. Assurez-vous que votre CV est à jour
3. Postulez à des offres correspondant à votre profil
4. Soyez patient - les réponses peuvent prendre du temps

---

### Parents

#### Je ne trouve pas de crèches dans ma région

**Solutions :**
1. Élargissez votre zone de recherche
2. Réduisez vos critères de filtrage
3. Vérifiez que des crèches sont inscrites dans votre région

---

#### Je n'ai pas eu de réponse à ma demande

**Conseils :**
1. Les crèches répondent généralement sous 1-5 jours
2. Envoyez un message de suivi poli
3. Soumettez des demandes à plusieurs crèches
4. Vérifiez vos spams pour les réponses

## Problèmes techniques

### La page ne charge pas

**Solutions :**
1. Actualisez la page (F5 ou Ctrl+R)
2. Videz le cache de votre navigateur
3. Essayez un autre navigateur
4. Vérifiez votre connexion internet
5. Attendez quelques minutes et réessayez

---

### Le site est lent

**Causes possibles :**
- Connexion internet lente
- Cache du navigateur plein
- Serveur surchargé

**Solutions :**
1. Videz le cache du navigateur
2. Vérifiez votre connexion internet
3. Fermez d'autres onglets/applications
4. Essayez à un autre moment

---

### Je vois une erreur

**Que faire :**
1. Notez le message d'erreur
2. Prenez une capture d'écran
3. Actualisez la page
4. Si le problème persiste, créez un ticket de support

**Informations à fournir :**
- Message d'erreur exact
- Page où l'erreur s'est produite
- Actions effectuées avant l'erreur
- Navigateur et système d'exploitation

---

### Les images ne s'affichent pas

**Solutions :**
1. Actualisez la page
2. Videz le cache du navigateur
3. Vérifiez votre connexion
4. Essayez un autre navigateur

---

### Le téléchargement de fichiers échoue

**Causes possibles :**
- Fichier trop volumineux
- Format non supporté
- Connexion instable

**Solutions :**
1. Vérifiez la taille du fichier (max 10 Mo)
2. Vérifiez le format (PDF, JPG, PNG, etc.)
3. Compressez le fichier si nécessaire
4. Utilisez une connexion stable
5. Réessayez

---

### La messagerie ne fonctionne pas

**Solutions :**
1. Actualisez la page
2. Vérifiez que vous êtes connecté
3. Videz le cache du navigateur
4. Essayez un autre navigateur
5. Contactez le support

## Vider le cache du navigateur

### Chrome

1. Cliquez sur ⋮ (menu)
2. Paramètres → Confidentialité → Effacer les données de navigation
3. Sélectionnez "Images et fichiers en cache"
4. Cliquez sur "Effacer les données"

### Firefox

1. Cliquez sur ☰ (menu)
2. Paramètres → Vie privée et sécurité
3. Cookies et données de sites → Effacer les données
4. Cochez "Contenu web en cache"
5. Cliquez sur "Effacer"

### Safari

1. Safari → Préférences → Avancé
2. Cochez "Afficher le menu Développement"
3. Développement → Vider les caches

### Edge

1. Cliquez sur ⋯ (menu)
2. Paramètres → Confidentialité → Effacer les données de navigation
3. Sélectionnez "Images et fichiers mis en cache"
4. Cliquez sur "Effacer maintenant"

## Contacter le support

### Créer un ticket

1. Cliquez sur **Support** dans la barre latérale
2. Cliquez sur **"Créer un ticket"**
3. Remplissez :
   - **Objet** - Brève description du problème
   - **Catégorie** - Général, Technique, Facturation
   - **Priorité** - Faible, Moyenne, Haute, Urgente
   - **Description** - Explication détaillée
4. Joignez des captures d'écran si utile
5. Cliquez sur **"Soumettre"**

### Informations utiles à inclure

- Votre rôle sur la plateforme
- Les étapes pour reproduire le problème
- Message d'erreur (si applicable)
- Navigateur et version
- Système d'exploitation
- Captures d'écran

### Temps de réponse

| Priorité | Temps de réponse |
|----------|------------------|
| Urgente | < 4 heures |
| Haute | < 24 heures |
| Moyenne | 1-2 jours |
| Faible | 2-5 jours |

## FAQ générale

### Compte

**Q : Puis-je avoir plusieurs comptes ?**
R : Non, un seul compte par adresse email est autorisé.

**Q : Puis-je changer de rôle ?**
R : Contactez le support pour discuter d'un changement de rôle.

**Q : Mes données sont-elles sécurisées ?**
R : Oui, nous utilisons le chiffrement et suivons les normes de sécurité. Voir notre politique de confidentialité.

---

### Utilisation

**Q : La plateforme est-elle disponible sur mobile ?**
R : Oui, le site est responsive et fonctionne sur tous les appareils.

**Q : Y a-t-il une application mobile ?**
R : Pas actuellement, mais le site web mobile offre une expérience similaire.

**Q : Dans quelles langues la plateforme est-elle disponible ?**
R : Anglais, Français et Allemand.

---

### Support

**Q : Quels sont les horaires du support ?**
R : Le support est disponible en jours ouvrables, généralement 9h-18h CET.

**Q : Y a-t-il un numéro de téléphone ?**
R : Le support se fait principalement par ticket. Les clients Enterprise ont accès au support téléphonique.
