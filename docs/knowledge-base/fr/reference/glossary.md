# Glossaire

Ce glossaire définit les termes courants utilisés sur ProCrèche Solutions.

---

## Termes généraux

| Terme | Définition |
|-------|------------|
| **Tableau de bord** | Page d'accueil personnalisée affichant les métriques clés et l'activité récente |
| **Profil** | Informations publiques sur un utilisateur ou une organisation |
| **Lead** | Demande de renseignements d'un parent à une crèche |
| **Marketplace** | Section où les fondations peuvent trouver et commander des produits/services |
| **Abonnement** | Plan payant permettant l'accès aux fonctionnalités premium |
| **Canton** | Division administrative de la Suisse |

---

## Rôles utilisateur

| Rôle | Description |
|------|-------------|
| **Fondation** | Organisation de garde d'enfants (crèche, garderie, jardin d'enfants) |
| **Fournisseur de produits** | Entreprise vendant des produits aux crèches |
| **Prestataire de services** | Entreprise offrant des services aux crèches |
| **Éducateur** | Professionnel de la petite enfance cherchant un emploi |
| **Parent** | Personne cherchant des services de garde pour son enfant |
| **Administrateur** | Utilisateur avec des privilèges de gestion de la plateforme |

---

## Termes Fondation

| Terme | Définition |
|-------|------------|
| **Capacité** | Nombre maximum d'enfants pouvant être accueillis |
| **Pédagogie** | Approche éducative (Montessori, Reggio Emilia, etc.) |
| **Lead parent** | Demande de placement d'un parent |
| **Offre d'emploi** | Poste ouvert pour le recrutement |
| **Candidature** | Soumission d'un éducateur pour un poste |
| **E-Learning** | Formations en ligne pour le personnel |
| **Procédures RH** | Documents et modèles de ressources humaines |
| **Politiques cantonales** | Réglementations spécifiques à chaque canton |

---

## Termes Fournisseur/Prestataire

| Terme | Définition |
|-------|------------|
| **Produit** | Article physique vendu aux crèches |
| **Service** | Prestation offerte aux crèches |
| **Commande** | Achat de produits par une fondation |
| **Demande de service** | Demande de prestation par une fondation |
| **Demande de renseignements** | Question d'une fondation sur des produits/services |
| **Code promo** | Code de réduction pour les clients |
| **SKU** | Référence produit interne |
| **Catalogue** | Liste complète des produits/services |
| **Lien de réservation** | URL permettant la prise de rendez-vous |

## Statuts

### Statuts généraux

| Statut | Description |
|--------|-------------|
| **Actif** | Élément visible et opérationnel |
| **Inactif** | Élément masqué ou désactivé |
| **Brouillon** | Élément non publié, en cours de création |
| **En attente** | En attente d'action ou de traitement |
| **Complété** | Terminé avec succès |
| **Annulé** | Annulé par l'utilisateur |

### Statuts de commande

| Statut | Description |
|--------|-------------|
| **Soumise** | Commande envoyée, en attente de confirmation |
| **Acceptée** | Confirmée par le fournisseur |
| **En traitement** | En cours de préparation |
| **Expédiée** | Envoyée au client |
| **Livrée** | Reçue par le client |

### Statuts de candidature

| Statut | Description |
|--------|-------------|
| **Soumise** | Candidature envoyée |
| **Vue** | Consultée par l'employeur |
| **En cours d'examen** | En évaluation |
| **Entretien** | Convoqué pour un entretien |
| **Acceptée** | Offre proposée ou poste obtenu |
| **Refusée** | Non retenue |

### Statuts de lead

| Statut | Description |
|--------|-------------|
| **Nouveau** | Lead venant d'être reçu |
| **En traitement** | En cours de gestion |
| **Contacté** | Réponse envoyée au parent |
| **Converti** | Enfant inscrit |
| **Fermé** | Lead terminé sans conversion |

---

## Termes de facturation

| Terme | Définition |
|-------|------------|
| **Plan** | Niveau d'abonnement (Basic, Essential, Professional, Enterprise) |
| **Cycle de facturation** | Période entre deux paiements (mensuel/annuel) |
| **Renouvellement** | Prolongation automatique de l'abonnement |
| **Au prorata** | Calcul proportionnel pour les périodes partielles |
| **Facture** | Document détaillant un paiement |
| **Méthode de paiement** | Moyen utilisé pour payer (carte, virement) |

---

## Termes techniques

| Terme | Définition |
|-------|------------|
| **Cache** | Données stockées temporairement pour accélérer le chargement |
| **Cookie** | Petit fichier stocké par le navigateur |
| **RGPD** | Règlement Général sur la Protection des Données |
| **Authentification** | Vérification de l'identité d'un utilisateur |
| **Vérification email** | Confirmation de l'accès à une adresse email |
| **OAuth** | Connexion via un service tiers (Google) |
| **API** | Interface de programmation pour intégrations |

---

## Termes de messagerie

| Terme | Définition |
|-------|------------|
| **Conversation** | Fil de messages entre utilisateurs |
| **Message** | Communication textuelle individuelle |
| **Pièce jointe** | Fichier joint à un message |
| **Notification** | Alerte informant d'un événement |
| **Digest** | Résumé périodique des notifications |

## Termes de profil

| Terme | Définition |
