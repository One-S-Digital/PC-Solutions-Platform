# Navigateurs supportés

Ce document liste les navigateurs web supportés et les exigences pour utiliser ProCrèche Solutions.

---

## Navigateurs recommandés

Pour la meilleure expérience, utilisez l'un de ces navigateurs dans leur dernière version :

| Navigateur | Version minimale | Recommandé |
|------------|------------------|------------|
| 🌐 **Google Chrome** | 90+ | ✅ Oui |
| 🦊 **Mozilla Firefox** | 88+ | ✅ Oui |
| 🧭 **Apple Safari** | 14+ | ✅ Oui |
| 🔷 **Microsoft Edge** | 90+ | ✅ Oui |

## Navigateurs non supportés

Ces navigateurs ne sont **pas** supportés :

| Navigateur | Raison |
|------------|--------|
| ❌ Internet Explorer | Obsolète, fonctionnalités manquantes |
| ❌ Opera Mini | Fonctionnalités limitées |
| ❌ Navigateurs anciens | Problèmes de sécurité et compatibilité |

## Exigences techniques

### JavaScript

- ✅ JavaScript doit être **activé**
- La plateforme ne fonctionnera pas sans JavaScript

### Cookies

- ✅ Les cookies doivent être **autorisés**
- Nécessaires pour la connexion et les préférences

### Résolution d'écran

| Appareil | Résolution minimale |
|----------|---------------------|
| Ordinateur | 1024 x 768 px |
| Tablette | 768 x 1024 px |
| Smartphone | 320 x 568 px |

### Connexion internet

- Connexion stable recommandée
- Vitesse minimale : 1 Mbps

## Comment mettre à jour votre navigateur

### Google Chrome

1. Cliquez sur **⋮** (menu)
2. **Aide** → **À propos de Google Chrome**
3. La mise à jour se fait automatiquement

### Mozilla Firefox

1. Cliquez sur **☰** (menu)
2. **Aide** → **À propos de Firefox**
3. Cliquez sur **"Mettre à jour"** si disponible

### Apple Safari

1. Ouvrez **Préférences Système** sur Mac
2. **Mise à jour de logiciels**
3. Installez les mises à jour macOS

### Microsoft Edge

1. Cliquez sur **⋯** (menu)
2. **Aide et commentaires** → **À propos de Microsoft Edge**
3. La mise à jour se fait automatiquement

## Dépannage des problèmes de navigateur

### La page ne charge pas correctement

1. Actualisez la page (F5 ou Ctrl+R)
2. Videz le cache du navigateur
3. Essayez un autre navigateur
4. Vérifiez votre connexion internet

### Problèmes d'affichage

1. Vérifiez le zoom de la page (100% recommandé)
2. Videz le cache
3. Désactivez les extensions qui pourraient interférer

### Problèmes de connexion

1. Vérifiez que les cookies sont activés
2. Videz les cookies du site
3. Essayez en navigation privée

### Extensions problématiques

Certaines extensions peuvent causer des problèmes :
- Bloqueurs de publicités
- Extensions de confidentialité strictes
- VPN ou proxys

Essayez de les désactiver temporairement si vous rencontrez des problèmes.
