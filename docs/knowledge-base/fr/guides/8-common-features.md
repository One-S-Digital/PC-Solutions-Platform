## Système de messagerie

Communiquez en toute sécurité avec les autres utilisateurs de la plateforme.

### Accéder à la messagerie

Cliquez sur **Messages** dans la barre latérale.

### Interface de messagerie

```
┌─────────────────────────────────────────────────────────────┐
│  Messages                              [Nouveau message]    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────┐  ┌───────────────────────────────────┐│
│  │ Conversations   │  │  Conversation avec Crèche ABC   ↑ ││
│  │                 │  │                                   ││
│  │ ● Crèche ABC    │  │  [10:30] Crèche ABC:            ││
│  │   Bonjour...    │  │  Bonjour ! Merci pour votre      ││
│  │                 │  │  demande. Nous avons bien reçu   ││
│  │   Crèche XYZ    │  │  votre dossier.                  ││
│  │   Merci pour... │  │                                   ││
│  │                 │  │  [10:45] Vous:                    ││
│  │   Fournisseur   │  │  Merci de votre réponse rapide.  ││
│  │   Votre comma...│  │  Avez-vous des places pour       ││
│  │                 │  │  septembre ?                      ││
│  └─────────────────┘  │                                   ││
│                       │  ┌─────────────────────────────┐  ││
│                       │  │ Tapez votre message...      │  ││
│                       │  └─────────────────────────────┘  ││
│                       │           [📎] [Envoyer]         ││
│                       └───────────────────────────────────┘│
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Fonctionnalités principales

#### Conversations

- **Liste des conversations** - Toutes vos discussions
- **Messages non lus** - Indiqués par un point bleu
- **Recherche** - Trouver des conversations
- **Tri** - Par date, non lu en premier

#### Envoyer des messages

1. Sélectionnez une conversation ou créez-en une nouvelle
2. Tapez votre message
3. Cliquez sur **Envoyer** ou appuyez sur Entrée

#### Pièces jointes

- Cliquez sur 📎 pour joindre des fichiers
- Formats acceptés : PDF, DOC, JPG, PNG
- Taille max : 10 Mo par fichier

#### Nouveau message

1. Cliquez sur **"Nouveau message"**
2. Sélectionnez le destinataire
3. Tapez votre message
4. Envoyez

### Indicateurs de statut

| Indicateur | Signification |
|------------|---------------|
| ✓ | Message envoyé |
| ✓✓ | Message lu |
| ● | Message non lu (bleu) |
| 🕐 | En attente d'envoi |

### Bonnes pratiques

- ✅ Répondez dans les 24-48 heures
- ✅ Soyez professionnel et courtois
- ✅ Utilisez des objets clairs
- ✅ Joignez les documents pertinents
- ❌ N'envoyez pas de spam
- ❌ Ne partagez pas d'informations sensibles

## Notifications

Restez informé des activités importantes.

### Types de notifications

| Type | Description |
|------|-------------|
| 🔔 **En temps réel** | Alertes immédiates dans l'app |
| 📧 **Email** | Notifications par email |
| 📱 **Push** | Notifications navigateur (si activé) |

### Centre de notifications

Cliquez sur l'icône 🔔 dans la barre de navigation pour voir :
- Nouvelles notifications
- Historique des notifications
- Options de marquage (lu/non lu)

### Catégories de notifications

#### Notifications de compte
- Changements de mot de passe
- Modifications d'email
- Alertes de sécurité

#### Notifications d'activité
- Nouveaux messages
- Réponses à vos demandes
- Mises à jour de statut
- Nouvelles candidatures (Fondations)
- Nouvelles commandes (Fournisseurs/Prestataires)

#### Notifications système
- Maintenance planifiée
- Nouvelles fonctionnalités
- Changements de conditions

### Configurer les notifications

1. Allez dans **Paramètres** → **Préférences de notification**
2. Activez/désactivez chaque type
3. Choisissez la fréquence du digest :
   - **Immédiat** - Chaque notification
   - **Quotidien** - Résumé journalier
   - **Hebdomadaire** - Résumé de la semaine
   - **Aucun** - Désactivé

### Gérer les notifications

| Action | Comment |
|--------|---------|
| Marquer comme lu | Cliquez sur la notification |
| Tout marquer comme lu | Bouton en haut de la liste |
| Supprimer | Glissez ou cliquez sur X |
| Voir les détails | Cliquez pour ouvrir |

## Gestion des fichiers

Téléchargez et gérez vos documents sur la plateforme.

### Où télécharger des fichiers

Selon votre rôle :
- **Profil** - Photo de profil, logo
- **Messages** - Pièces jointes
- **Produits/Services** - Images
- **Candidatures** - CV, certificats
- **Support** - Captures d'écran

### Formats acceptés

| Type | Formats |
|------|---------|
| **Images** | JPG, PNG, GIF, WebP |
| **Documents** | PDF, DOC, DOCX |
| **Tableurs** | XLS, XLSX |

### Limites de taille

| Contexte | Limite |
|----------|--------|
| Photo de profil | 5 Mo |
| Logo d'organisation | 5 Mo |
| Image de couverture | 10 Mo |
| Pièces jointes messages | 10 Mo par fichier |
| Documents candidature | 10 Mo par fichier |

### Télécharger un fichier

1. Cliquez sur **"Télécharger"** ou **"Choisir un fichier"**
2. Sélectionnez le fichier sur votre appareil
3. Attendez le téléchargement
4. Vérifiez l'aperçu (images)
5. Enregistrez

### Gérer les fichiers téléchargés

- **Remplacer** - Téléchargez un nouveau fichier
- **Supprimer** - Retirez le fichier
- **Télécharger** - Récupérez une copie

### Conseils

- 📷 Utilisez des images de haute qualité
- 📄 Nommez clairement vos fichiers
- 🗜️ Compressez les fichiers volumineux
- 🔒 Ne téléchargez pas d'infos sensibles inutiles

## Photos de profil et images

Personnalisez votre présence sur la plateforme.

### Photo de profil

#### Pour les individus (Éducateurs, Parents)
- Photo claire de votre visage
- Fond neutre recommandé
- Professionnelle et récente

#### Pour les organisations (Fondations, Fournisseurs, Prestataires)
- Logo de l'entreprise
- Image carrée
- Bonne résolution (min 200x200px)

### Image de couverture (Organisations)

- Bannière représentant votre activité
- Format large (recommandé : 1200x400px)
- Photo de votre établissement ou équipe

### Modifier votre photo

1. Allez dans **Paramètres** ou **Profil**
2. Cliquez sur la photo actuelle ou "Changer la photo"
3. Sélectionnez votre nouvelle image
4. Recadrez si nécessaire
5. Enregistrez

### Spécifications

| Élément | Dimensions | Format |
|---------|------------|--------|
| Photo de profil | 200x200px min | JPG, PNG |
| Logo | 200x200px min | JPG, PNG |
| Image de couverture | 1200x400px | JPG, PNG |

## Paramètres de langue

ProCrèche Solutions est disponible en plusieurs langues.

### Langues disponibles

| Langue | Code |
|--------|------|
| 🇬🇧 English | EN |
| 🇫🇷 Français | FR |
| 🇩🇪 Deutsch | DE |

### Changer la langue

#### Méthode 1 : Barre de navigation
1. Cliquez sur le sélecteur de langue (EN/FR/DE)
2. Sélectionnez votre langue
3. La page se recharge automatiquement

#### Méthode 2 : Paramètres
1. Allez dans **Paramètres**
2. Trouvez **"Langue"**
3. Sélectionnez votre préférence
4. Enregistrez

### Ce qui change

- Interface utilisateur
- Messages système
- Emails de la plateforme
- Documentation

### Ce qui ne change pas

- Contenu créé par d'autres utilisateurs
- Messages reçus
- Descriptions de produits/services

## Compatibilité mobile

Accédez à ProCrèche Solutions depuis n'importe quel appareil.

### Appareils supportés

| Appareil | Support |
|----------|---------|
| 💻 Ordinateur | ✅ Complet |
| 📱 Smartphone | ✅ Responsive |
| 📱 Tablette | ✅ Optimisé |

### Navigation mobile

- Menu hamburger (☰) pour la navigation
- Boutons et liens adaptés au tactile
- Formulaires optimisés pour mobile
- Images redimensionnées automatiquement

### Conseils pour mobile

- 📶 Utilisez une connexion stable
- 🔄 Gardez le navigateur à jour
- 📱 Utilisez le mode portrait pour les formulaires
- 🖥️ Utilisez le mode paysage pour les tableaux

### Installation (Progressive Web App)

Sur certains navigateurs, vous pouvez "installer" l'app :

1. Visitez le site sur Chrome/Safari
2. Appuyez sur "Ajouter à l'écran d'accueil"
3. L'app apparaît comme une application native
4. Accès rapide depuis l'écran d'accueil

## Fonctionnalités d'accessibilité

ProCrèche Solutions s'engage pour l'accessibilité.

### Fonctionnalités incluses

#### Navigation au clavier
- Tab pour naviguer entre les éléments
- Entrée pour activer les boutons/liens
- Échap pour fermer les modales
- Flèches pour les menus déroulants

#### Compatibilité lecteurs d'écran
- Labels ARIA sur les éléments interactifs
- Structure sémantique des pages
- Textes alternatifs sur les images
- Annonces des changements dynamiques

#### Contraste et lisibilité
- Contraste élevé des couleurs
- Texte lisible (taille minimale)
- Polices claires
- Pas de texte sur images complexes

### Paramètres de votre navigateur

Vous pouvez personnaliser davantage via les paramètres de votre navigateur :
- Zoom de la page
- Taille du texte
- Mode contraste élevé
- Mode sombre (si supporté)

### Signaler un problème d'accessibilité

Si vous rencontrez des difficultés :
1. Créez un ticket de support
2. Sélectionnez "Accessibilité" comme catégorie
3. Décrivez le problème rencontré
4. Mentionnez les technologies d'assistance utilisées
