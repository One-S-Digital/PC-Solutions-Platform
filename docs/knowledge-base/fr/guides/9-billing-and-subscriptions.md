### Changer de plan

#### Mise à niveau (Upgrade)

1. Allez dans **Paramètres** → **Facturation**
2. Cliquez sur **"Demander un changement de plan"**
3. Sélectionnez le nouveau plan souhaité
4. Confirmez votre demande
5. L'équipe vous contactera

#### Rétrogradation (Downgrade)

1. Même processus que la mise à niveau
2. Sélectionnez un plan inférieur
3. Confirmez
4. Prend effet à la fin de la période actuelle

> ⚠️ **Note :** La rétrogradation peut entraîner la perte d'accès à certaines fonctionnalités.

## Méthodes de paiement

### Méthodes acceptées

| Méthode | Supportée |
|---------|-----------|
| 💳 Visa | ✅ |
| 💳 Mastercard | ✅ |
| 💳 American Express | ✅ |
| 🏦 Virement bancaire | ✅ (Enterprise) |
| 📄 Facture | ✅ (Enterprise) |

### Ajouter une méthode de paiement

1. Allez dans **Paramètres** → **Facturation**
2. Cliquez sur **"Gérer le paiement"**
3. Cliquez sur **"Ajouter une carte"**
4. Entrez les détails de la carte :
   - Numéro de carte
   - Date d'expiration
   - CVV
   - Nom sur la carte
5. Cliquez sur **"Enregistrer"**

### Modifier la méthode de paiement

1. Accédez aux paramètres de paiement
2. Cliquez sur **"Modifier"** à côté de la carte
3. Mettez à jour les informations
4. Enregistrez

### Supprimer une méthode de paiement

1. Accédez aux paramètres de paiement
2. Cliquez sur **"Supprimer"** à côté de la carte
3. Confirmez la suppression

> ⚠️ **Note :** Vous devez avoir au moins une méthode de paiement active si vous avez un abonnement en cours.

### Sécurité des paiements

- 🔒 Chiffrement SSL/TLS
- 🏦 Conformité PCI-DSS
- 🛡️ Tokenisation des données de carte
- 📧 Alertes de paiement

## Historique de facturation

### Accéder à l'historique

1. Allez dans **Paramètres** → **Facturation**
2. Cliquez sur **"Voir les factures"**

### Informations disponibles

| Colonne | Description |
|---------|-------------|
| **Date** | Date de la facture |
| **Numéro** | Numéro de facture |
| **Description** | Plan ou service |
| **Montant** | Prix payé |
| **Statut** | Payée, En attente, Échec |
| **Actions** | Télécharger PDF |

### Télécharger une facture

1. Trouvez la facture dans la liste
2. Cliquez sur **"Télécharger"** ou l'icône PDF
3. Le fichier PDF se télécharge
4. Utilisez-le pour votre comptabilité

### Filtrer l'historique

- Par période (mois, année)
- Par statut (Payée, En attente)
- Par recherche (numéro de facture)

## Facturation annuelle vs mensuelle

### Avantages de l'annuel

| Aspect | Mensuel | Annuel |
|--------|---------|--------|
| **Paiement** | Chaque mois | Une fois par an |
| **Prix** | Prix standard | Réduction (~10-20%) |
| **Engagement** | Mois par mois | 12 mois |
| **Flexibilité** | Plus flexible | Moins flexible |

### Comment changer

1. Contactez le support ou
2. Demandez via **Paramètres** → **Facturation**
3. L'équipe vous contactera avec les options

### Ce qui se passe lors du changement

- **Mensuel → Annuel** : Calculé au prorata, différence facturée
- **Annuel → Mensuel** : Prend effet à la fin de l'année

## Demandes d'abonnement

### Comment fonctionne le système

Les changements d'abonnement passent par un système de demandes :

1. Vous soumettez une demande
2. L'équipe examine votre demande
3. Vous recevez une confirmation
4. Le changement est appliqué

### Soumettre une demande

1. Allez dans **Paramètres** → **Facturation**
2. Cliquez sur **"Demander un changement"**
3. Sélectionnez le type de demande :
   - Mise à niveau
   - Rétrogradation
   - Changement de cycle
   - Annulation
4. Ajoutez des commentaires si nécessaire
5. Soumettez

### Suivre vos demandes

- Les demandes en cours sont affichées
- Vous recevez des notifications de mise à jour
- Contactez le support pour toute question

## Annulation

### Comment annuler

1. Allez dans **Paramètres** → **Facturation**
2. Cliquez sur **"Demander l'annulation"**
3. Sélectionnez une raison (optionnel)
4. Confirmez votre demande

### Ce qui se passe après l'annulation

| Élément | Comportement |
|---------|--------------|
| **Accès** | Jusqu'à la fin de la période payée |
| **Données** | Conservées pendant 30 jours |
| **Compte** | Passe au plan gratuit |
| **Fonctionnalités** | Accès aux fonctionnalités de base |

### Réactivation

Vous pouvez réactiver votre abonnement :

1. Allez dans **Paramètres** → **Facturation**
2. Cliquez sur **"Réactiver l'abonnement"**
3. Sélectionnez un plan
4. Procédez au paiement

> 💡 **Conseil :** Si vous annulez par erreur, contactez rapidement le support.

### Remboursements

- Les remboursements sont évalués au cas par cas
- Contactez le support pour toute demande
- Les remboursements partiels au prorata sont possibles

## Questions fréquentes

### Facturation

**Q : Quand suis-je facturé ?**
R : Le même jour chaque mois (ou année) à partir de votre souscription.

**Q : Puis-je changer de plan en milieu de mois ?**
R : Oui, les changements sont calculés au prorata.

**Q : Que se passe-t-il si mon paiement échoue ?**
R : Vous recevez une notification. Vous avez généralement quelques jours pour mettre à jour votre méthode de paiement avant suspension.

### Plans

**Q : Puis-je tester un plan supérieur ?**
R : Contactez le support pour discuter d'une période d'essai.

**Q : Y a-t-il des frais cachés ?**
R : Non, les prix affichés incluent toutes les fonctionnalités du plan.

### Annulation

**Q : Puis-je annuler à tout moment ?**
R : Oui, mais l'accès continue jusqu'à la fin de la période payée.

**Q : Mes données sont-elles supprimées après annulation ?**
R : Les données sont conservées 30 jours, puis supprimées sauf demande contraire.
