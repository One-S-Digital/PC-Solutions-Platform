## Aperçu du tableau de bord

Votre tableau de bord est votre espace central pour gérer votre recherche d'emploi. Accédez-y en cliquant sur **Tableau de bord** dans la barre latérale.

### Ce que vous verrez

| Section | Description |
|---------|-------------|
| 📋 **Résumé des candidatures** | Total des candidatures et leurs statuts |
| 💼 **Offres recommandées** | Offres correspondant à votre profil |
| 🔔 **Activité récente** | Dernières mises à jour sur vos candidatures |
| ✅ **Complétude du profil** | Progression vers un profil complet |

### Statistiques rapides

- **Candidatures soumises** - Total des candidatures envoyées
- **En attente** - En attente de réponse
- **En cours d'examen** - L'employeur évalue votre candidature
- **Acceptées** - Offres reçues

## Gestion du profil

Votre profil est votre CV en ligne. Les fondations le consultent pour décider de vous contacter.

### Accéder à votre profil

1. Cliquez sur votre **photo de profil** en haut à droite
2. Sélectionnez **"Modifier le profil"**, OU
3. Allez dans **Paramètres** → **Mon profil**

### Sections du profil

#### Informations personnelles

| Champ | Description | Conseils |
|-------|-------------|----------|
| **Nom complet** | Votre prénom et nom | Utilisez votre nom légal |
| **Email** | Adresse de contact | Gardez-la à jour |
| **Téléphone** | Numéro de contact | Pour les appels des employeurs |
| **Localisation** | Canton/Région | Où vous pouvez travailler |
| **Ville** | Votre ville | Pour les offres locales |

#### Résumé professionnel

| Champ | Description | Conseils |
|-------|-------------|----------|
| **Bio** | Votre présentation | Décrivez votre expérience et philosophie |
| **Qualifications** | Diplômes et certifications | Listez tous les diplômes pertinents |
| **Années d'expérience** | Expérience totale | Soyez précis |
| **Langues parlées** | Langues maîtrisées | Incluez les niveaux |
| **Compétences** | Compétences clés | Pertinentes pour la petite enfance |

#### Préférences d'emploi

| Champ | Description | Options |
|-------|-------------|---------|
| **Type de contrat** | Temps plein, Temps partiel, etc. | Sélectionnez vos préférences |
| **Disponibilité** | Quand vous pouvez commencer | Immédiat ou date spécifique |
| **Prétentions salariales** | Salaire souhaité | Optionnel mais recommandé |
| **Volonté de déplacement** | Mobilité géographique | Oui/Non et conditions |
| **Régions préférées** | Cantons souhaités | Où vous voulez travailler |

#### Photo de profil

- Téléchargez une photo professionnelle
- Recommandé : Image carrée, claire et professionnelle
- Évitez : Photos de vacances, selfies flous

### Conseils pour un profil réussi

- ✅ Complétez 100% de votre profil
- ✅ Utilisez une photo professionnelle
- ✅ Écrivez une bio engageante
- ✅ Listez toutes vos qualifications
- ✅ Gardez vos disponibilités à jour
- ✅ Téléchargez votre CV dans la galerie

## Paramètres de disponibilité

Montrez aux employeurs quand vous êtes disponible pour travailler.

### Accéder aux disponibilités

1. Allez dans **Profil** ou **Paramètres**
2. Trouvez la section **"Disponibilité"**

### Planning hebdomadaire

Définissez votre disponibilité régulière :

```
┌─────────────────────────────────────────────────────┐
│               Planning hebdomadaire                  │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Lundi      [✓] Disponible   08:00 - 18:00        │
│  Mardi      [✓] Disponible   08:00 - 18:00        │
│  Mercredi   [✓] Disponible   08:00 - 12:00        │
│  Jeudi      [✓] Disponible   08:00 - 18:00        │
│  Vendredi   [✓] Disponible   08:00 - 17:00        │
│  Samedi     [ ] Non disponible                      │
│  Dimanche   [ ] Non disponible                      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

Pour chaque jour :
1. Cochez si vous êtes **disponible**
2. Définissez l'**heure de début**
3. Définissez l'**heure de fin**

### Exceptions de dates

Ajoutez des exceptions pour des jours spécifiques :

#### Ajouter une indisponibilité

Pour bloquer des dates spécifiques (vacances, rendez-vous, etc.) :

1. Cliquez sur **"Ajouter une exception"**
2. Sélectionnez la **date**
3. Choisissez **"Non disponible"**
4. Ajoutez une raison optionnelle
5. Enregistrez

#### Ajouter une disponibilité exceptionnelle

Pour vous rendre disponible un jour normalement bloqué :

1. Cliquez sur **"Ajouter une exception"**
2. Sélectionnez la **date**
3. Choisissez **"Disponible"**
4. Définissez les heures
5. Enregistrez

### Calendrier des disponibilités

Voyez vos disponibilités sous forme de calendrier :
- **Vert** - Disponible
- **Rouge** - Non disponible
- **Jaune** - Partiellement disponible

### Conseils

- 🗓️ Mettez à jour régulièrement votre planning
- 📅 Ajoutez les vacances prévues à l'avance
- ⏰ Soyez réaliste sur vos horaires

## Parcourir les offres d'emploi

Trouvez des opportunités d'emploi correspondant à vos compétences et préférences.

### Accéder aux offres

Cliquez sur **Offres d'emploi** dans la barre latérale.

### Rechercher des offres

Utilisez les filtres pour affiner votre recherche :

| Filtre | Options |
|--------|---------|
| **Localisation** | Canton, ville |
| **Type de contrat** | CDI, CDD, Temps partiel, etc. |
| **Expérience requise** | Junior, Confirmé, Senior |
| **Fourchette salariale** | Minimum/Maximum |
| **Date de publication** | Dernières 24h, semaine, mois |
| **Recherche texte** | Mots-clés dans le titre/description |

### Informations sur l'offre

Chaque offre affiche :
- **Titre du poste**
- **Nom de la fondation**
- **Localisation**
- **Type de contrat**
- **Fourchette salariale** (si indiqué)
- **Date de publication**
- **Aperçu de la description**

### Voir les détails

Cliquez sur une offre pour voir :
- Description complète du poste
- Responsabilités
- Qualifications requises
- Avantages proposés
- Profil de la fondation
- Comment postuler

## Gérer les candidatures

### Postuler à une offre

1. Trouvez une offre intéressante
2. Cliquez sur **"Postuler"**
3. Remplissez le formulaire de candidature :

| Champ | Description |
|-------|-------------|
| **Lettre de motivation** | Expliquez pourquoi vous êtes le bon candidat |
| **CV** | Sélectionnez depuis votre galerie ou téléchargez |
| **Documents supplémentaires** | Certificats, diplômes (optionnel) |
| **Disponibilité** | Date de début possible |
| **Prétentions salariales** | Vos attentes (optionnel) |

4. Vérifiez vos informations
5. Cliquez sur **"Soumettre la candidature"**

### Suivre vos candidatures

Accédez à **Mes candidatures** dans la barre latérale.

### Statuts des candidatures

| Statut | Signification | Prochaine étape |
|--------|---------------|-----------------|
| 📝 **Soumise** | Candidature envoyée | Attendre la réponse |
| 👁️ **Vue** | L'employeur l'a consultée | Bon signe, patientez |
| 🔍 **En cours d'examen** | Évaluation en cours | Restez disponible |
| 📞 **Entretien** | Vous êtes convoqué | Préparez-vous ! |
| ✅ **Acceptée** | Offre reçue | Félicitations ! |
| ❌ **Refusée** | Non retenu | Continuez vos recherches |
| ⏸️ **En attente** | Processus en pause | Patientez |

### Actions sur les candidatures

- **Voir** - Consultez les détails de votre candidature
- **Retirer** - Annulez votre candidature (avant décision)
- **Contacter** - Envoyez un message à l'employeur

## Galerie de fichiers

Stockez et gérez vos documents professionnels.

### Accéder à la galerie

Allez dans **Profil** → **Galerie de fichiers** ou **Documents**

### Types de documents

| Type | Utilisation |
|------|-------------|
| 📄 **CV** | À joindre aux candidatures |
| 📜 **Diplômes** | Preuves de qualifications |
| 🏆 **Certificats** | Formations complémentaires |
| 📋 **Références** | Lettres de recommandation |
| 🖼️ **Portfolio** | Exemples de travaux (optionnel) |

### Télécharger des fichiers

1. Cliquez sur **"Télécharger"**
2. Sélectionnez votre fichier
3. Ajoutez un titre descriptif
4. Choisissez la catégorie
5. Enregistrez

### Formats acceptés

- **Documents** : PDF, DOC, DOCX
- **Images** : JPG, PNG

### Conseils

- 📁 Gardez votre CV à jour
- 🏷️ Nommez clairement vos fichiers
- 📂 Organisez par catégorie
- 🔒 Les fichiers sont privés par défaut

## Messagerie

Communiquez avec les fondations et recruteurs.

### Accéder aux messages

Cliquez sur **Messages** dans la barre latérale.

### Recevoir des messages

Les fondations peuvent vous contacter pour :
- Poser des questions sur votre profil
- Planifier un entretien
- Discuter d'une opportunité

### Envoyer des messages

Vous pouvez contacter les fondations :
- Après avoir postulé à une offre
- Pour poser des questions sur un poste
- Suite à une invitation à communiquer

### Bonnes pratiques

- ✅ Répondez rapidement
- ✅ Soyez professionnel
- ✅ Relisez avant d'envoyer
- ✅ Posez des questions pertinentes

Voir [Fonctionnalités communes - Messagerie](./8-common-features.md#système-de-messagerie) pour plus de détails.

## Paramètres

Gérez votre compte et vos préférences.

### Accéder aux paramètres

Cliquez sur **Paramètres** dans la barre latérale.

### Sections des paramètres

#### Sécurité du compte
- Modifier vos informations personnelles
- Changer votre mot de passe
- Changer votre adresse email

#### Préférences de notification
- **Nouvelles offres** : Alertes pour les offres correspondant à votre profil
- **Mises à jour de candidatures** : Notifications de changement de statut
- **Messages** : Alertes pour les nouveaux messages
- **Fréquence du digest** : Quotidien, Hebdomadaire ou Aucun

#### Confidentialité et données
- **Visibilité du profil** : Visible aux fondations ou masqué
- **Export de données** : Télécharger vos informations
- **Suppression de compte** : Demander la suppression RGPD

Voir le [Guide des paramètres](../reference/settings-guide.md) pour des instructions détaillées.

## Support et aide

### Créer un ticket de support

1. Cliquez sur **Support** dans la barre latérale
2. Cliquez sur **"Créer un ticket"**
3. Remplissez :
   - **Objet** - Description brève
   - **Catégorie** - Général, Technique, Compte
   - **Message** - Explication détaillée
4. Joignez des fichiers si nécessaire
5. Soumettez

### Suivi des tickets

- Voir tous vos tickets
- Vérifier les mises à jour
- Ajouter des messages de suivi
- Fermer les tickets résolus
