## Codes Promo

### Créer un code promo

1. Cliquez sur **"Ajouter un nouveau code"**
2. Remplissez le formulaire :
   - **Code** : Le code que les clients utiliseront
   - **Réduction** : Description de la réduction ("20% de réduction", "Livraison gratuite")
   - **Description** : Termes et conditions (optionnel)
   - **Afficher sur le profil** : Activer/désactiver la visibilité

3. Cliquez sur **"Enregistrer"**
