## Profil d'organisation

Votre profil d'entreprise est visible par les fondations parcourant le marketplace. Un profil complet inspire confiance et attire les clients.

### Accéder à votre profil

1. Cliquez sur **Profil d'organisation** dans la barre latérale, OU
2. Allez dans **Paramètres** → **Modifier le profil**

### Informations du profil

#### Détails de base de l'entreprise

| Champ | Description | Conseils |
|-------|-------------|----------|
| **Nom de l'entreprise** | Nom de votre entreprise | Utilisez votre raison sociale officielle |
| **Personne de contact** | Contact principal | Commercial ou responsable de compte |
| **Numéro de téléphone** | Téléphone professionnel | Incluez l'indicatif pays |
| **Email de contact** | Email public | Pour les demandes clients |
| **Canton/Région** | Votre localisation | Où vous êtes basé |
| **Ville** | Votre ville | Aide les clients à trouver des fournisseurs locaux |

#### Détails commerciaux

| Champ | Description | Conseils |
|-------|-------------|----------|
| **Description** | À propos de votre entreprise | Décrivez vos produits et valeurs |
| **Catégories de produits** | Types de produits vendus | Sélectionnez toutes les catégories pertinentes |
| **Régions desservies** | Où vous livrez | Sélectionnez plusieurs cantons |
| **Langues** | Langues supportées | Pour la communication client |
| **Numéro TVA** | Identification fiscale | Requis pour la facturation |
| **Quantité minimum de commande** | Taille minimum de commande | Définir si applicable |
| **Lien de commande directe** | URL de commande externe | Lien vers votre propre boutique (optionnel) |
| **URL du catalogue** | Lien vers le catalogue produits | PDF ou catalogue en ligne |

#### Éléments visuels

| Élément | Recommandation |
|---------|----------------|
| **Logo** | Image carrée, min 200x200px |
| **Image de couverture** | Bannière large, min 1200x400px |

### Télécharger votre catalogue

Vous pouvez télécharger un catalogue produits que les clients peuvent parcourir :

1. Allez dans **Profil d'organisation**
2. Faites défiler jusqu'à la section **Catalogue**
3. Cliquez sur **"Télécharger le catalogue"**
4. Sélectionnez votre fichier PDF
5. Ajoutez un titre et une description
6. Enregistrez

> 💡 **Conseil :** Gardez votre catalogue à jour avec les produits et prix actuels.

## Catalogue de produits

Gérez vos produits qui apparaissent sur le marketplace.

### Accéder aux produits

Cliquez sur **Catalogue produits** dans la barre latérale.

### Création de produits

1. Cliquez sur **"Créer un produit"**
2. Remplissez les détails du produit :

#### Informations de base

| Champ | Description | Obligatoire |
|-------|-------------|-------------|
| **Titre** | Nom du produit | ✅ Oui |
| **Sous-titre** | Courte accroche | ❌ Optionnel |
| **Description** | Description complète du produit | ✅ Oui |
| **Prix** | Prix du produit | ✅ Oui |
| **Devise** | CHF, EUR, etc. | ✅ Oui (défaut : CHF) |

#### Catégorisation

| Champ | Description |
|-------|-------------|
| **Catégorie principale** | Catégorie principale du produit |
| **Catégories additionnelles** | Catégories secondaires |
| **Tags** | Mots-clés pour la recherche |
| **Points forts du produit** | Arguments de vente clés |

#### Détails d'inventaire

| Champ | Description |
|-------|-------------|
| **SKU** | Code produit interne |
| **SKU fournisseur** | Code produit alternatif |
| **EAN** | Numéro de code-barres (si applicable) |
| **Unité de mesure** | Pièce, Pack, Boîte, etc. |
| **Quantité min. commande** | Unités minimum par commande |
| **Quantité max. commande** | Unités maximum par commande |
| **Statut stock** | En stock, Stock faible, Rupture |
| **Délai de livraison** | Jours pour livrer |

#### Image du produit

- Cliquez sur **"Télécharger une image"**
- Sélectionnez une photo de produit de haute qualité
- Recommandé : Image carrée, min 500x500px

3. Définissez le statut du produit :

| Statut | Visibilité |
|--------|------------|
| **Actif** | Visible sur le marketplace |
| **Inactif** | Masqué du marketplace |
| **Brouillon** | Pas encore publié |
| **Rupture de stock** | Affiché comme indisponible |

4. Cliquez sur **"Enregistrer le produit"**

### Gérer les produits

#### Modifier un produit
1. Trouvez le produit dans votre liste
2. Cliquez sur **"Modifier"** ou sur le nom du produit
3. Effectuez les modifications
4. Cliquez sur **"Enregistrer"**

#### Changer le statut
1. Cliquez sur le badge de statut d'un produit
2. Sélectionnez le nouveau statut
3. Les modifications s'enregistrent automatiquement

#### Supprimer un produit
1. Cliquez sur **"Supprimer"** sur le produit
2. Confirmez la suppression

> ⚠️ **Attention :** Supprimer un produit est définitif et ne peut pas être annulé.

### Filtrer les produits

Utilisez les filtres pour trouver des produits :
- **Statut** - Actif, Inactif, Brouillon
- **Catégorie** - Filtrer par catégorie de produit
- **Recherche** - Rechercher par nom ou SKU

## Gestion des commandes

Gérez les commandes des fondations.

### Accéder aux commandes

Cliquez sur **Commandes** dans la barre latérale.

### Voir les commandes

Votre liste de commandes affiche :
- ID de commande
- Nom de la fondation
- Date de commande
- Articles commandés
- Montant total
- Statut actuel

### Détails de la commande

Cliquez sur une commande pour voir :
- Informations complètes de la commande
- Détails du client (Fondation)
- Lignes d'articles avec quantités et prix
- Adresse de livraison
- Notes de commande

### Statuts des commandes

| Statut | Signification | Votre action |
|--------|---------------|--------------|
| 📝 **En attente** | Nouvelle commande reçue | Examiner et accepter |
| ✅ **Acceptée** | Vous avez confirmé | Préparer pour l'expédition |
| 🔄 **En traitement** | En cours de préparation | Mettre à jour quand prêt |
| 🚚 **Expédiée** | Envoyée au client | Fournir les infos de suivi |
| ✅ **Livrée** | Livrée avec succès | Terminé |
| ❌ **Annulée** | Commande annulée | Aucune action requise |

### Mettre à jour le statut de commande

1. Ouvrez la commande
2. Cliquez sur **"Mettre à jour le statut"**
3. Sélectionnez le nouveau statut
4. Ajoutez des notes (optionnel)
5. Enregistrez

### Exemple de workflow

```
Nouvelle commande reçue
       ↓
   [Accepter la commande]
       ↓
   En traitement
       ↓
   [Marquer comme expédiée]
       ↓
     Expédiée
       ↓
   [Marquer comme livrée]
       ↓
    Livrée ✓
```

## Gestion des demandes

Gérez les questions et demandes des clients potentiels.

### Que sont les demandes ?

Les demandes sont des messages de fondations qui souhaitent en savoir plus sur vos produits avant de commander.

### Informations de la demande

Chaque demande contient :
- Détails de la fondation (nom, contact)
- Ligne d'objet
- Contenu du message
- Produit d'intérêt (si spécifié)
- Quantité nécessaire
- Fourchette de budget
- Niveau d'urgence
- Préférences de contact

### Statuts des demandes

| Statut | Signification | Votre action |
|--------|---------------|--------------|
| 🆕 **Nouvelle** | Vient d'être reçue | Examiner et répondre |
| ⏳ **En attente** | Attend votre réponse | Répondre rapidement |
| 📞 **Contacté** | Vous avez répondu | Suivi si nécessaire |
| 💰 **Devis envoyé** | Devis fourni | Attendre la décision |
| ✅ **Conclue** | Convertie en commande | Terminé |
| ❌ **Refusée** | Vous avez décliné | Aucune action |
| 🚫 **Annulée** | Client a annulé | Aucune action |

### Répondre aux demandes

1. Ouvrez la demande
2. Examinez les besoins du client
3. Cliquez sur **"Répondre"**
4. Fournissez :
   - Votre message de réponse
   - Montant du devis (si applicable)
   - Informations supplémentaires
5. Mettez à jour le statut
6. Envoyez

## Codes promo

Créez des codes promotionnels pour attirer les clients. Les codes promo s'affichent sur votre profil public.

### Accéder aux codes promo

Allez dans **Paramètres** → **Gestionnaire de codes promo**

### Comprendre les codes promo

Les codes promo vous permettent d'offrir des réductions aux clients. Ils apparaissent sur votre profil fournisseur pour que les fondations puissent les voir et les utiliser.

### Créer un code promo

1. Cliquez sur **"Ajouter un nouveau code"**
2. Une fenêtre modale s'ouvre avec ces champs :

```
┌─────────────────────────────────────────────────────┐
│         Ajouter un nouveau code promo           [X] │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Code *                                             │
│  ┌─────────────────────────────────────────────┐   │
│  │  PROMO20                                     │   │
│  └─────────────────────────────────────────────┘   │
│  Le code que les clients utiliseront (ex: PROMO20) │
│                                                     │
│  Réduction *                                        │
│  ┌─────────────────────────────────────────────┐   │
│  │  20% de réduction sur votre première commande│   │
│  └─────────────────────────────────────────────┘   │
│  Décrivez la réduction (ex: 20%, Livraison offerte)│
│                                                     │
│  Description (Optionnel)                            │
│  ┌─────────────────────────────────────────────┐   │
│  │  Valable pour les nouveaux clients          │   │
│  │  uniquement. Non cumulable.                 │   │
│  └─────────────────────────────────────────────┘   │
│  Détails optionnels sur les conditions             │
│                                                     │
│  ☑ Afficher sur le profil                          │
│                                                     │
│         [Annuler]              [Enregistrer]       │
└─────────────────────────────────────────────────────┘
```

#### Champs du code promo

| Champ | Description | Obligatoire | Exemple |
|-------|-------------|-------------|---------|
| **Code** | Le code que les clients utilisent | ✅ Oui | `PROMO20`, `LIVRAISON`, `BIENVENUE10` |
| **Réduction** | Ce que le client obtient | ✅ Oui | "20% de réduction", "Livraison offerte", "CHF 10 de réduction" |
| **Description** | Conditions générales | ❌ Optionnel | "Valable pour commandes > CHF 100" |
| **Afficher sur le profil** | Afficher publiquement | ✅ Oui (défaut : activé) | Activer/Désactiver |

3. Cliquez sur **"Enregistrer"**

### Conseils pour les codes promo

- ✅ **Noms de codes :** Utilisez des codes mémorables, faciles à taper
- ✅ **Texte de réduction :** Soyez clair sur ce que les clients reçoivent
- ✅ **Conditions :** Ajoutez les conditions dans la description si nécessaire
- ✅ **Masquer les anciens codes :** Désactivez "Afficher sur le profil" pour les promos expirées

### Gérer les codes promo

Vos codes promo sont affichés dans un tableau :

| Code | Réduction | Statut | Actions |
|------|-----------|--------|---------|
| `PROMO20` | 20% première commande | 🟢 Actif | Modifier · Supprimer |
| `LIVRAISON` | Livraison offerte > CHF 100 | 🟢 Actif | Modifier · Supprimer |
| `ANCIEN` | 10% de réduction | 🔴 Masqué | Modifier · Supprimer |

#### Modifier un code promo
1. Cliquez sur **"Modifier"** sur le code
2. Modifiez les détails
3. Cliquez sur **"Enregistrer"**

#### Supprimer un code promo
1. Cliquez sur **"Supprimer"** sur le code
2. Confirmez la suppression

#### Masquer un code promo
1. Cliquez sur **"Modifier"**
2. Décochez **"Afficher sur le profil"**
3. Enregistrez

> 💡 **Conseil :** Au lieu de supprimer les anciens codes promo, masquez-les pour garder un historique.

### Comment les clients voient les codes promo

Quand les fondations consultent votre profil, elles voient vos codes promo actifs dans une section dédiée :

```
┌─────────────────────────────────────────────────────┐
│  🏷️ Codes promo disponibles                        │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌───────────────────────────────────────────────┐ │
│  │  PROMO20                   Copier le code [📋]│ │
│  │  20% de réduction première commande           │ │
│  │  Nouveaux clients uniquement                  │ │
│  └───────────────────────────────────────────────┘ │
│                                                     │
│  ┌───────────────────────────────────────────────┐ │
│  │  LIVRAISON                 Copier le code [📋]│ │
│  │  Livraison offerte > CHF 100                  │ │
│  └───────────────────────────────────────────────┘ │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## Résiliations de remises

### Que sont les résiliations de remises ?

Les résiliations de remises sont des arrangements spéciaux où les tarifs promotionnels ou remises sont formellement terminés. Cela peut se produire quand :

- Une période promotionnelle expire
- Un accord de tarification spéciale prend fin
- Les remises de volume ne s'appliquent plus
- Les termes d'un partenariat changent

### Comment ça fonctionne

1. Les résiliations de remises sont gérées via le système admin
2. Vous serez notifié à l'avance de toute résiliation
3. Les tarifs standards reprennent après la résiliation
4. Les historiques de remises sont conservés

### Ce que cela signifie pour vous

- Vérifiez vos paramètres de notification pour recevoir les avis de résiliation
- Planifiez les changements de prix quand les périodes promotionnelles se terminent
- Contactez le support si vous avez des questions sur les arrangements de remises
