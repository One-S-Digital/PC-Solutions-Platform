## Aperçu du tableau de bord

Votre tableau de bord est votre base. Accédez-y en cliquant sur **Tableau de bord** dans la barre latérale ou en naviguant vers `/foundation/dashboard`.

### Cartes de statistiques rapides

En haut de votre tableau de bord, vous verrez les métriques clés d'un coup d'œil :

| Statistique | Ce qu'elle montre |
|-------------|-------------------|
| 📋 **Total offres d'emploi** | Nombre d'offres publiées |
| ⏳ **Candidatures en attente** | Candidatures à examiner |
| 📝 **Total candidatures** | Toutes les candidatures reçues |
| 🛒 **Total commandes** | Commandes passées sur le marketplace |
| 🔧 **Demandes de services** | Services que vous avez demandés |
| ✅ **Offres actives** | Offres d'emploi actuellement publiées |

### Flux d'activités récentes

Sous les statistiques, voyez vos dernières activités :
- Nouveaux leads parents reçus
- Candidatures soumises
- Mises à jour des commandes
- Mises à jour des demandes de services
- Nouveaux messages

### Aperçu du calendrier

Voir et gérer les événements du calendrier :
- Rendez-vous à venir
- Entretiens
- Rendez-vous de services
- Réunions
- Rappels

### Créer un événement calendrier

1. Allez au **Tableau de bord**
2. Trouvez la section Calendrier
3. Cliquez sur **"Ajouter un événement"**
4. Remplissez :
   - **Titre** - Nom de l'événement
   - **Type** - Rendez-vous, Entretien, Service, Réunion ou Rappel
   - **Date et heure** - Quand il a lieu
   - **Description** - Détails supplémentaires (optionnel)
   - **Lieu** - Où il se déroule (optionnel)
5. Cliquez sur **Enregistrer**

## Profil d'organisation

Votre profil d'organisation est visible par les parents, les éducateurs et les fournisseurs. Gardez-le à jour pour attirer les bonnes connexions.

### Accéder à votre profil

1. Cliquez sur **Profil d'organisation** dans la barre latérale, OU
2. Allez dans **Paramètres** → **Modifier le profil**

### Informations du profil

#### Informations de base

| Champ | Description | Conseils |
|-------|-------------|----------|
| **Nom de l'organisation** | Nom de votre crèche/fondation | Utilisez votre nom officiel enregistré |
| **Personne de contact** | Nom du contact principal | Généralement le directeur ou responsable |
| **Numéro de téléphone** | Téléphone professionnel | Incluez l'indicatif pays |
| **Email de contact** | Email de contact public | Peut différer de l'email de connexion |
| **Canton/Région** | Votre localisation | Sélectionnez dans la liste |
| **Ville** | Votre ville | Aide les parents à vous trouver |
| **Adresse** | Adresse postale | Pour les livraisons de services |

#### Informations détaillées

| Champ | Description | Conseils |
|-------|-------------|----------|
| **À propos/Description** | Décrivez votre crèche | Mettez en avant votre philosophie et ce qui vous rend unique |
| **Capacité** | Nombre d'enfants accueillis | Entrez la capacité autorisée totale |
| **Langues parlées** | Langues proposées | Sélectionnez toutes celles qui s'appliquent |
| **Pédagogie** | Approches pédagogiques | Montessori, Reggio Emilia, etc. |
| **Numéro TVA** | Pour la facturation (si applicable) | Optionnel |
| **Régions desservies** | Zones que vous servez | Sélectionnez plusieurs cantons si applicable |

#### Éléments visuels

| Élément | Recommandation |
|---------|----------------|
| **Logo** | Image carrée, min 200x200px, apparaît dans les listes |
| **Image de couverture** | Bannière large, min 1200x400px, apparaît sur la page de profil |

### Enregistrer votre profil

1. Effectuez vos modifications
2. Cliquez sur **"Enregistrer les modifications"** en bas
3. Attendez le message de confirmation

> 💡 **Conseil :** Les profils complets apparaissent plus haut dans les résultats de recherche !

## Leads parents (Demandes)

Les leads parents sont des demandes de parents cherchant des services de garde. Répondre rapidement augmente les inscriptions.

> 📋 **Nécessite :** Plan Essential ou supérieur

### Comprendre la boîte de réception des leads

Accédez à vos leads en cliquant sur **Leads** dans la barre latérale.

#### Informations du lead

Chaque lead contient :
- **Détails du parent** - Nom, email, téléphone
- **Détails de l'enfant** - Nom, âge
- **Préférences** - Localisation, date de début, langues
- **Besoins spéciaux** - Toute exigence particulière
- **Message** - Note personnelle du parent

#### Statuts des leads

| Statut | Signification | Action requise |
|--------|---------------|----------------|
| 🆕 **Nouveau** | Vient d'être reçu | Examiner et répondre |
| 🔄 **En traitement** | Vous y travaillez | Continuer le suivi |
| ✅ **Converti** | Inscrit avec succès | Terminé |

### Répondre à un lead

1. Cliquez sur un lead pour ouvrir les détails
2. Examinez les informations et besoins du parent
3. Cliquez sur **"Répondre"**
4. Choisissez votre réponse :
   
   | Réponse | Quand l'utiliser |
   |---------|------------------|
   | **Intéressé** | Vous souhaitez poursuivre avec cette famille |
   | **Pas intéressé** | Pas compatible (capacité pleine, etc.) |
   | **Besoin de plus d'infos** | Vous avez besoin d'informations supplémentaires |
   | **Inscrit** | L'enfant a été inscrit |

5. Ajoutez un message personnel (recommandé)
6. Cliquez sur **Envoyer la réponse**

### Filtrer les leads

Utilisez les filtres pour trouver des leads spécifiques :
- **Statut** - Nouveau, En traitement, Converti
- **Plage de dates** - Filtrer par date de réception
- **Recherche** - Rechercher par nom du parent ou de l'enfant

### Statistiques des leads

Suivez la performance de vos leads :
- Total des leads reçus
- Taux de réponse
- Taux de conversion
- Temps de réponse moyen

## Accès au Marketplace

Parcourez et achetez des produits et services auprès de fournisseurs vérifiés.

### Parcourir les produits

1. Cliquez sur **Marketplace** dans la barre latérale
2. Sélectionnez l'onglet **Produits**
3. Utilisez les filtres :
   - **Catégorie** - Jouets, Mobilier, Alimentation, etc.
   - **Fournisseur** - Filtrer par fournisseur spécifique
   - **Recherche** - Trouver des produits spécifiques
4. Cliquez sur un produit pour voir les détails

### Parcourir les services

1. Dans Marketplace, sélectionnez l'onglet **Services**
2. Filtrez par :
   - **Type de service** - Nettoyage, Formation, etc.
   - **Prestataire** - Filtrer par fournisseur
   - **Type de livraison** - Sur site, À distance, Hybride
3. Cliquez sur un service pour voir les détails

### Voir les profils Fournisseur/Prestataire

1. Cliquez sur un produit ou service
2. Cliquez sur le nom du fournisseur
3. Voir leur profil complet incluant :
   - Informations de l'entreprise
   - Tous les produits/services
   - Coordonnées
   - Codes promo (si disponibles)

### Passer des commandes

Pour commander des produits :

1. Trouvez le produit souhaité
2. Cliquez sur **"Ajouter au panier"** ou **"Commander maintenant"**
3. Définissez la quantité
4. Vérifiez votre panier
5. Cliquez sur **"Passer la commande"**
6. Confirmez les détails de la commande
7. Finalisez la commande

### Demander des services

Pour demander un service :

1. Trouvez le service souhaité
2. Cliquez sur **"Demander un service"**
3. Remplissez les détails de la demande :
   - Dates préférées
   - Exigences spécifiques
   - Notes supplémentaires
4. Soumettez votre demande
5. Attendez la réponse du prestataire

### Envoyer des demandes de renseignements

Pour des demandes personnalisées ou des questions :

1. Trouvez un fournisseur ou prestataire
2. Cliquez sur **"Envoyer une demande"**
3. Remplissez :
   - Objet
   - Message
   - Produit/service d'intérêt (optionnel)
   - Quantité nécessaire
   - Fourchette de budget
   - Niveau d'urgence
4. Soumettez la demande

## Module de recrutement

Publiez des offres d'emploi et trouvez des éducateurs qualifiés pour votre crèche.

> 📋 **Nécessite :** Plan Professional ou supérieur

### Accéder au recrutement

Cliquez sur **Recrutement** dans la barre latérale.

### Créer une offre d'emploi

1. Allez dans **Recrutement** → **Offres d'emploi**
2. Cliquez sur **"Créer une offre"**
3. Remplissez les détails :

| Champ | Description | Conseils |
|-------|-------------|----------|
| **Titre** | Nom du poste | Soyez précis : "Éducateur petite enfance référent" |
| **Description** | Description complète | Incluez les responsabilités quotidiennes |
| **Exigences** | Ce dont les candidats ont besoin | Listez les qualifications obligatoires |
| **Avantages** | Ce que vous offrez | Mettez en avant les avantages |
| **Responsabilités** | Tâches quotidiennes | Soyez détaillé |
| **Qualifications** | Parcours souhaité | Formation, certifications |
| **Localisation** | Lieu de travail | Ville/canton |
| **Fourchette salariale** | Rémunération proposée | Soyez transparent |
| **Type de contrat** | Type d'emploi | CDI, CDD, etc. |
| **Date de début** | Quand le poste commence | Immédiat ou date spécifique |

4. Choisissez le statut :
   - **Brouillon** - Enregistrer sans publier
   - **Publiée** - Rendre visible aux candidats
5. Cliquez sur **Enregistrer**

### Types de contrats

| Type | Description |
|------|-------------|
| **Temps plein** | Poste à temps plein standard |
| **Temps partiel** | Poste à horaires réduits |
| **CDI** | Contrat à durée indéterminée (permanent) |
| **CDD** | Contrat à durée déterminée |
| **Stage** | Poste de formation |
| **Remplacement** | Poste de remplacement temporaire |
| **Temporaire** | Mission courte durée |
| **Freelance** | Travailleur indépendant |

### Gérer les offres d'emploi

- **Modifier** - Cliquez sur une offre pour la modifier
- **Publier/Dépublier** - Activer/désactiver la visibilité
- **Clôturer** - Arrêter d'accepter les candidatures
- **Supprimer** - Supprimer l'offre

### Voir les candidatures

1. Cliquez sur une offre d'emploi
2. Voir la section **Candidatures**
3. Voir tous les candidats qui ont postulé

### Détails des candidatures

Chaque candidature montre :
- Profil du candidat
- Lettre de motivation
- CV
- Date de candidature
- Statut actuel

### Gérer les candidatures

Mettre à jour le statut des candidatures :

| Statut | Quand l'utiliser |
|--------|------------------|
| **En attente** | Vient d'être reçue, pas encore examinée |
| **Examinée** | Vous l'avez consultée |
| **Acceptée** | Poursuivre/proposer le poste |
| **Refusée** | Non retenue |

Pour mettre à jour :
1. Ouvrez la candidature
2. Cliquez sur **"Mettre à jour le statut"**
3. Sélectionnez le nouveau statut
4. Ajoutez des notes internes (optionnel)
5. Enregistrez

### Vivier de candidats

Parcourez tous les éducateurs disponibles :
1. Allez dans **Recrutement** → **Vivier de candidats**
2. Parcourez les profils d'éducateurs
3. Filtrez par compétences, localisation, disponibilité
4. Voir les profils complets
5. Contactez les candidats via la messagerie

## Support et aide

### Créer un ticket de support

1. Cliquez sur **Support** dans la barre latérale
2. Cliquez sur **"Créer un ticket"**
3. Remplissez :
   - **Objet** - Brève description
   - **Catégorie** - Général, Technique, Facturation, Demande de fonctionnalité
   - **Priorité** - Faible, Moyenne, Haute, Urgente
   - **Message** - Explication détaillée
   - **Pièces jointes** - Captures d'écran ou fichiers (optionnel)
4. Cliquez sur **Soumettre**

### Suivi des tickets

- Voir tous vos tickets
- Vérifier les mises à jour de statut
- Ajouter des messages de suivi
- Fermer les tickets résolus

### Statuts des tickets

| Statut | Signification |
|--------|---------------|
| **Ouvert** | Soumis, en attente de réponse |
| **En cours** | En cours de traitement |
| **Résolu** | Problème corrigé |
| **Fermé** | Ticket terminé |
