## Profil d'organisation

Votre profil d'entreprise est visible par les fondations parcourant le marketplace. Un profil complet génère plus de leads.

### Accéder à votre profil

1. Cliquez sur **Profil d'organisation** dans la barre latérale, OU
2. Allez dans **Paramètres** → **Modifier le profil**

### Informations du profil

#### Détails de base de l'entreprise

| Champ | Description | Conseils |
|-------|-------------|----------|
| **Nom de l'entreprise** | Votre raison sociale | Utilisez votre nom officiel enregistré |
| **Personne de contact** | Contact principal | Gérant ou responsable de compte |
| **Numéro de téléphone** | Téléphone professionnel | Incluez l'indicatif pays |
| **Email de contact** | Email public | Pour les demandes clients |
| **Canton/Région** | Votre localisation | Où vous êtes basé |
| **Ville** | Votre ville | Aide les clients à vous trouver |

#### Détails des services

| Champ | Description | Conseils |
|-------|-------------|----------|
| **Description** | À propos de vos services | Décrivez votre expertise |
| **Types de services** | Services proposés | Sélectionnez toutes les catégories |
| **Type de livraison** | Comment vous délivrez | Sur site, À distance, Hybride |
| **Régions desservies** | Où vous intervenez | Cantons couverts |
| **Langues** | Langues supportées | Pour la communication |
| **Numéro TVA** | Identification fiscale | Requis pour la facturation |
| **Lien de réservation** | URL de prise de rendez-vous | Lien Calendly, etc. |

#### Éléments visuels

| Élément | Recommandation |
|---------|----------------|
| **Logo** | Image carrée, min 200x200px |
| **Image de couverture** | Bannière large, min 1200x400px |

## Catalogue de services

Gérez vos offres de services qui apparaissent sur le marketplace.

### Accéder aux services

Cliquez sur **Catalogue services** dans la barre latérale.

### Création de services

1. Cliquez sur **"Créer un service"**
2. Remplissez les détails du service :

#### Informations de base

| Champ | Description | Obligatoire |
|-------|-------------|-------------|
| **Titre** | Nom du service | ✅ Oui |
| **Sous-titre** | Courte accroche | ❌ Optionnel |
| **Description** | Description complète | ✅ Oui |
| **Prix** | Tarif du service | ✅ Oui |
| **Type de prix** | Par heure, Forfait, etc. | ✅ Oui |
| **Devise** | CHF, EUR, etc. | ✅ Oui |

#### Types de tarification

| Type | Description | Exemple |
|------|-------------|---------|
| **Forfait** | Prix fixe | CHF 500 par projet |
| **À l'heure** | Tarif horaire | CHF 80/heure |
| **À la journée** | Tarif journalier | CHF 600/jour |
| **À la séance** | Par session | CHF 150/session |
| **Sur devis** | Prix personnalisé | Nous contacter |

#### Détails du service

| Champ | Description |
|-------|-------------|
| **Catégorie** | Type de service principal |
| **Durée** | Durée typique |
| **Type de livraison** | Sur site, À distance, Hybride |
| **Capacité max** | Participants max (si applicable) |
| **Délai de réservation** | Préavis minimum requis |
| **Points forts** | Arguments de vente clés |

3. Définissez le statut du service :

| Statut | Visibilité |
|--------|------------|
| **Actif** | Visible sur le marketplace |
| **Inactif** | Masqué du marketplace |
| **Brouillon** | Pas encore publié |

4. Cliquez sur **"Enregistrer le service"**

### Gérer les services

#### Modifier un service
1. Trouvez le service dans votre liste
2. Cliquez sur **"Modifier"**
3. Effectuez les modifications
4. Enregistrez

#### Changer le statut
1. Cliquez sur le badge de statut
2. Sélectionnez le nouveau statut

#### Supprimer un service
1. Cliquez sur **"Supprimer"**
2. Confirmez

> ⚠️ **Note :** Supprimer un service est définitif.

## Gestion des demandes de services

Gérez les demandes de services des fondations.

### Accéder aux demandes

Cliquez sur **Demandes de services** dans la barre latérale.

### Voir les demandes

Votre liste de demandes affiche :
- Détails de la fondation
- Service demandé
- Date/heure demandée
- Exigences spéciales
- Statut

### Détails de la demande

Chaque demande contient :
- **Informations fondation** - Nom, contact, localisation
- **Détails du service** - Quel service ils veulent
- **Planification** - Dates et heures préférées
- **Exigences** - Besoins ou préférences spécifiques
- **Budget** - Fourchette de budget si spécifiée
- **Urgence** - Niveau de priorité

### Statuts des demandes

| Statut | Signification | Votre action |
|--------|---------------|--------------|
| 🆕 **Nouvelle** | Vient d'être reçue | Examiner et répondre |
| ⏳ **En attente** | Attend votre réponse | Répondre rapidement |
| 📅 **Planifiée** | Rendez-vous fixé | Préparer la livraison |
| 🔄 **En cours** | En train de délivrer | Mettre à jour la progression |
| ✅ **Complétée** | Service livré | Terminé |
| ❌ **Annulée** | Demande annulée | Aucune action |

### Répondre à une demande

1. Ouvrez la demande
2. Examinez les besoins du client
3. Cliquez sur **"Répondre"**
4. Choisissez votre réponse :
   - **Accepter** - Vous pouvez fournir le service
   - **Proposer une alternative** - Suggérer des changements
   - **Décliner** - Impossible de remplir
5. Ajoutez votre message
6. Incluez le devis si accepté
7. Envoyez

### Planifier un service

1. Acceptez la demande
2. Cliquez sur **"Planifier"**
3. Définissez date et heure
4. Confirmez la localisation
5. Envoyez la confirmation

## Intégration du lien de réservation

Permettez aux clients de réserver directement dans votre calendrier.

### Pourquoi utiliser les liens de réservation ?

- Les clients peuvent voir votre disponibilité
- Réservation automatique de créneaux
- Réduction des échanges d'emails
- Image professionnelle

### Outils supportés

Vous pouvez utiliser n'importe quel outil de planification externe :
- **Calendly** - Populaire et facile à utiliser
- **Cal.com** - Alternative open-source
- **Microsoft Bookings** - Intégration Office 365
- **Autres** - Tout outil de planification avec URL partageable

### Configurer votre lien de réservation

1. Allez dans **Paramètres** → **Contact et réservation**
2. Trouvez la section **"Lien de réservation"**
3. Collez l'URL de votre outil de planification
4. Ajoutez une description optionnelle (ex: "Réserver une consultation")
5. Enregistrez

### Comment les clients utilisent votre lien

Quand les clients voient votre profil ou service :

1. Ils voient un bouton **"Réserver un rendez-vous"**
2. Un clic les dirige vers votre outil de planification
3. Ils sélectionnent un créneau
4. Ils reçoivent une confirmation
5. Vous recevez une notification de réservation

### Bonnes pratiques

- ✅ Gardez votre calendrier à jour
- ✅ Définissez des règles de disponibilité claires
- ✅ Ajoutez un temps tampon entre les réservations
- ✅ Incluez les types de rendez-vous pour différents services

## Codes promo

Créez des offres promotionnelles pour attirer les clients.

### Accéder aux codes promo

Allez dans **Paramètres** → **Gestionnaire de codes promo**

### Créer un code promo

1. Cliquez sur **"Ajouter un nouveau code"**
2. Remplissez les détails :

```text
┌─────────────────────────────────────────────────────┐
│        Ajouter un nouveau code promo            [X] │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Code *                                             │
│  ┌─────────────────────────────────────────────┐   │
│  │  SERVICE15                                   │   │
│  └─────────────────────────────────────────────┘   │
│  Le code que les clients utiliseront               │
│                                                     │
│  Réduction *                                        │
│  ┌─────────────────────────────────────────────┐   │
│  │  15% de réduction sur le premier service    │   │
│  └─────────────────────────────────────────────┘   │
│  Décrivez la réduction offerte                     │
│                                                     │
│  Description (Optionnel)                            │
│  ┌─────────────────────────────────────────────┐   │
│  │  Nouveaux clients uniquement.               │   │
│  │  Valable sur tous les services.             │   │
│  └─────────────────────────────────────────────┘   │
│  Conditions et modalités                           │
│                                                     │
│  ☑ Afficher sur le profil                          │
│                                                     │
│         [Annuler]              [Enregistrer]       │
└─────────────────────────────────────────────────────┘
```

#### Champs du code promo

| Champ | Description | Obligatoire |
|-------|-------------|-------------|
| **Code** | Le code à utiliser | ✅ Oui |
| **Réduction** | Ce que le client reçoit | ✅ Oui |
| **Description** | Conditions générales | ❌ Optionnel |
| **Afficher sur le profil** | Visibilité publique | ✅ Oui |

3. Cliquez sur **"Enregistrer"**

### Gérer les codes promo

- **Modifier** - Cliquez sur "Modifier" pour changer les détails
- **Supprimer** - Supprimer le code définitivement
- **Masquer** - Décochez "Afficher sur le profil"

### Comment les clients voient les codes

Sur votre profil, les clients voient :

```text
┌─────────────────────────────────────────────────────┐
│  🏷️ Codes promo disponibles                        │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌───────────────────────────────────────────────┐ │
│  │  SERVICE15                 Copier le code [📋]│ │
│  │  15% de réduction premier service             │ │
│  │  Nouveaux clients uniquement                  │ │
│  └───────────────────────────────────────────────┘ │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## Résiliations de remises

### Que sont les résiliations de remises ?

Les résiliations de remises sont des arrangements où les tarifs promotionnels ou remises sont formellement terminés.

### Comment ça fonctionne

1. Gérées via le système admin
2. Notification préalable envoyée
3. Les tarifs standards reprennent
4. L'historique est conservé

### Ce que cela signifie pour vous

- Vérifiez vos notifications
- Planifiez les changements de tarifs
- Contactez le support pour toute question
