## Aperçu du tableau de bord

Votre tableau de bord est votre espace central pour gérer votre recherche de garde d'enfants.

### Accéder au tableau de bord

Cliquez sur **Tableau de bord** dans la barre latérale après vous être connecté.

### Ce que vous verrez

| Section | Description |
|---------|-------------|
| 📋 **Mes demandes** | Résumé de vos demandes en cours |
| 🔔 **Activité récente** | Dernières réponses des crèches |
| 📍 **Crèches à proximité** | Suggestions basées sur votre localisation |
| ✉️ **Messages récents** | Derniers messages reçus |

### Statistiques rapides

- **Demandes envoyées** - Total des demandes soumises
- **En attente** - Demandes sans réponse
- **Réponses reçues** - Crèches ayant répondu
- **Conversations actives** - Discussions en cours

## Rechercher une crèche

Trouvez la crèche idéale pour votre enfant.

### Accéder à la recherche

1. Cliquez sur **Trouver une crèche** dans la barre latérale
2. Ou allez dans **Tableau de bord** → **Rechercher**

### Options de recherche

#### Filtres disponibles

| Filtre | Description |
|--------|-------------|
| **Canton/Région** | Localisation souhaitée |
| **Ville** | Ville spécifique |
| **Type de garde** | Temps plein, temps partiel, occasionnel |
| **Âge de l'enfant** | Tranches d'âge acceptées |
| **Langues** | Langues parlées à la crèche |
| **Pédagogie** | Montessori, Reggio Emilia, etc. |

#### Recherche par nom

Tapez le nom d'une crèche si vous la connaissez déjà.

### Résultats de recherche

Chaque crèche affiche :
- **Nom** de la crèche
- **Localisation** (ville, canton)
- **Capacité** disponible
- **Langues** parlées
- **Aperçu** de la description
- **Photo** de profil

### Voir le profil d'une crèche

Cliquez sur une crèche pour voir :
- Description complète
- Philosophie pédagogique
- Services proposés
- Horaires d'ouverture
- Coordonnées
- Comment soumettre une demande

## Soumettre une demande

### Qu'est-ce qu'une demande ?

Une demande (ou "lead") est votre premier contact avec une crèche. Vous partagez vos informations et besoins, et la crèche vous répond.

### Formulaire de demande

Quand vous soumettez une demande, vous fournissez :

#### Informations sur l'enfant

| Champ | Description | Exemple |
|-------|-------------|---------|
| **Nom de l'enfant** | Prénom de votre enfant | Emma |
| **Âge/Date de naissance** | Âge actuel | 2 ans |
| **Besoins spéciaux** | Allergies, conditions | Allergie aux noix |

#### Vos préférences

| Champ | Description | Exemple |
|-------|-------------|---------|
| **Date de début** | Quand vous avez besoin de la garde | 1er septembre 2025 |
| **Type de garde** | Temps plein, partiel | Temps plein |
| **Jours souhaités** | Jours de la semaine | Lundi à vendredi |
| **Horaires** | Heures souhaitées | 8h00 - 18h00 |

#### Coordonnées

| Champ | Description |
|-------|-------------|
| **Votre nom** | Pré-rempli depuis votre profil |
| **Email** | Pour les réponses |
| **Téléphone** | Contact direct |
| **Localisation** | Votre adresse/ville |

#### Message personnel

Ajoutez un message pour vous présenter et expliquer vos besoins spécifiques.

### Soumettre la demande

1. Remplissez tous les champs obligatoires
2. Ajoutez un message personnel
3. Vérifiez vos informations
4. Cliquez sur **"Envoyer la demande"**

> 💡 **Conseil :** Un message personnel augmente vos chances de réponse !

## Gérer les demandes

Suivez toutes vos demandes en cours.

### Accéder aux demandes

Cliquez sur **Mes demandes** dans la barre latérale.

### Liste des demandes

Voyez toutes vos demandes avec :
- Nom de la crèche
- Date d'envoi
- Statut actuel
- Dernière activité

### Statuts des demandes

| Statut | Signification | Prochaine étape |
|--------|---------------|-----------------|
| 📝 **Envoyée** | Demande soumise | Attendre la réponse |
| 👁️ **Vue** | La crèche l'a consultée | Bon signe ! |
| 💬 **Réponse reçue** | La crèche a répondu | Lire et répondre |
| 📞 **Contact en cours** | Discussion active | Poursuivre l'échange |
| ✅ **Acceptée** | Place confirmée | Félicitations ! |
| ❌ **Refusée** | Pas de place disponible | Essayer ailleurs |

### Voir les détails d'une demande

Cliquez sur une demande pour voir :
- Vos informations soumises
- Réponse de la crèche
- Historique des échanges
- Actions disponibles

### Actions possibles

| Action | Quand l'utiliser |
|--------|------------------|
| **Voir** | Consulter les détails |
| **Répondre** | Envoyer un message |
| **Annuler** | Retirer votre demande |
| **Archiver** | Masquer de la liste active |

## Gestion du profil

Gardez vos informations à jour pour faciliter vos demandes.

### Accéder à votre profil

1. Cliquez sur votre **photo de profil** en haut à droite
2. Sélectionnez **"Modifier le profil"**

### Informations du profil

#### Vos informations

| Champ | Description |
|-------|-------------|
| **Nom complet** | Votre prénom et nom |
| **Email** | Adresse de contact |
| **Téléphone** | Numéro de contact |
| **Localisation** | Canton/ville |
| **Adresse** | Adresse complète (optionnel) |

#### Informations sur l'enfant

| Champ | Description |
|-------|-------------|
| **Nom de l'enfant** | Prénom de votre enfant |
| **Date de naissance** | Pour calculer l'âge |
| **Besoins spéciaux** | Allergies, conditions médicales |
| **Préférences linguistiques** | Langues souhaitées |

#### Préférences de garde

| Champ | Description |
|-------|-------------|
| **Type de garde** | Temps plein, partiel, occasionnel |
| **Jours préférés** | Jours de la semaine |
| **Horaires souhaités** | Heures typiques |
| **Budget** | Fourchette de budget (optionnel) |

### Photo de profil

- Téléchargez une photo (optionnel)
- Aide les crèches à vous identifier
- Photo de famille ou individuelle

### Enregistrer les modifications

1. Effectuez vos changements
2. Cliquez sur **"Enregistrer"**
3. Attendez la confirmation

## Messagerie

Communiquez directement avec les crèches.

### Accéder aux messages

Cliquez sur **Messages** dans la barre latérale.

### Vos conversations

La liste affiche :
- Nom de la crèche
- Dernier message
- Date/heure
- Messages non lus (badge)

### Lire et répondre

1. Cliquez sur une conversation
2. Lisez les messages
3. Tapez votre réponse
4. Cliquez sur **Envoyer**

### Démarrer une nouvelle conversation

Vous pouvez contacter une crèche :
1. Après avoir soumis une demande
2. Suite à une réponse à votre demande
3. Via le bouton "Contacter" sur leur profil

### Conseils de communication

- ✅ Soyez clair et précis
- ✅ Répondez rapidement
- ✅ Posez vos questions
- ✅ Mentionnez votre enfant et ses besoins
- ✅ Soyez poli et professionnel

Voir [Fonctionnalités communes - Messagerie](./8-common-features.md#système-de-messagerie) pour plus de détails.

## Support et aide

### Créer un ticket de support

1. Cliquez sur **Support** dans la barre latérale
2. Cliquez sur **"Créer un ticket"**
3. Remplissez :
   - **Objet** - Brève description
   - **Catégorie** - Général, Technique, Compte
   - **Message** - Explication détaillée
4. Cliquez sur **Soumettre**

### FAQ rapide

**Q : Combien de demandes puis-je envoyer ?**
R : Il n'y a pas de limite. Vous pouvez contacter autant de crèches que vous le souhaitez.

**Q : Combien de temps avant d'avoir une réponse ?**
R : Cela dépend de la crèche. La plupart répondent sous 1 à 5 jours ouvrables.

**Q : Puis-je modifier une demande envoyée ?**
R : Vous ne pouvez pas modifier une demande envoyée, mais vous pouvez envoyer un message à la crèche avec des informations mises à jour.

**Q : Comment savoir si une crèche a des places ?**
R : Soumettez une demande. La crèche vous informera de sa disponibilité.

## Conseils pour trouver une crèche

### Préparez votre recherche

1. **Définissez vos critères** - Localisation, horaires, pédagogie
2. **Listez vos besoins** - Allergies, besoins spéciaux
3. **Fixez un budget** - Ayez une idée de ce que vous pouvez dépenser
4. **Préparez vos questions** - Ce que vous voulez savoir

### Maximisez vos chances

1. **Complétez votre profil** - Plus d'infos = meilleures réponses
2. **Personnalisez vos messages** - Évitez les messages génériques
3. **Répondez rapidement** - Montrez votre intérêt
4. **Soyez flexible** - Plus de flexibilité = plus d'options

### Lors des visites

1. **Préparez des questions** - Pédagogie, repas, activités
2. **Observez l'environnement** - Propreté, sécurité, ambiance
3. **Rencontrez le personnel** - Évaluez les interactions
4. **Faites confiance à votre instinct** - L'atmosphère compte
