## Aperçu de la plateforme

### Qu'est-ce que ProCrèche Solutions ?

ProCrèche Solutions est une plateforme tout-en-un connectant l'écosystème suisse de la petite enfance :

- **Crèches (Fondations)** trouvent des produits, services, personnel et gèrent les demandes des parents
- **Fournisseurs de produits** vendent des produits de puériculture aux crèches
- **Prestataires de services** offrent des services professionnels aux crèches
- **Éducateurs** trouvent des opportunités d'emploi dans les crèches
- **Parents** trouvent une garde d'enfants de qualité pour leurs enfants

### Langues supportées

La plateforme est disponible en trois langues :
- 🇬🇧 **English** (Anglais)
- 🇫🇷 **Français**
- 🇩🇪 **Deutsch** (Allemand)

Vous pouvez changer de langue à tout moment en utilisant le sélecteur de langue dans la barre de navigation.

## Création de votre compte

### Étape 1 : Accéder à la page d'inscription

1. Visitez le site web ProCrèche Solutions
2. Cliquez sur le bouton **"S'inscrire"** dans la barre de navigation supérieure
3. Vous serez redirigé vers la page d'inscription

### Étape 2 : Choisissez votre rôle

Sélectionnez le type de compte correspondant à vos besoins :

| Rôle | Pour qui ? |
|------|------------|
| 🏫 **Fondation** | Crèches, organisations de garde d'enfants |
| 📦 **Fournisseur de produits** | Entreprises vendant des produits aux crèches |
| 🔧 **Prestataire de services** | Entreprises offrant des services aux crèches |
| 👩‍🏫 **Éducateur** | Professionnels de la petite enfance cherchant un emploi |
| 👨‍👩‍👧 **Parent** | Parents cherchant une garde d'enfants |

Cliquez sur votre rôle pour accéder au formulaire d'inscription.

### Étape 3 : Remplissez vos informations

Les informations requises dépendent de votre rôle :

#### Pour les Fondations, Fournisseurs et Prestataires de services :

| Champ | Description | Obligatoire |
|-------|-------------|-------------|
| Nom de l'organisation | Nom de votre entreprise ou crèche | ✅ Oui |
| Personne de contact | Votre nom complet | ✅ Oui |
| Email | Votre adresse email (utilisée pour la connexion) | ✅ Oui |
| Mot de passe | Créez un mot de passe sécurisé (min. 8 caractères) | ✅ Oui |
| Confirmer le mot de passe | Ressaisissez votre mot de passe | ✅ Oui |
| Numéro de téléphone | Numéro de contact professionnel | ✅ Oui |
| Canton/Région | Votre localisation en Suisse | ✅ Oui |
| Capacité | Nombre d'enfants (Fondations uniquement) | ✅ Oui |
| Catégorie | Catégorie de produits (Fournisseurs uniquement) | ✅ Oui |
| Type de service | Type de service (Prestataires uniquement) | ✅ Oui |

#### Pour les Éducateurs :

| Champ | Description | Obligatoire |
|-------|-------------|-------------|
| Nom complet | Votre prénom et nom | ✅ Oui |
| Email | Votre adresse email | ✅ Oui |
| Mot de passe | Créez un mot de passe sécurisé | ✅ Oui |
| Confirmer le mot de passe | Ressaisissez votre mot de passe | ✅ Oui |

#### Pour les Parents :

| Champ | Description | Obligatoire |
|-------|-------------|-------------|
| Nom du parent | Votre nom complet | ✅ Oui |
| Email | Votre adresse email | ✅ Oui |
| Mot de passe | Créez un mot de passe sécurisé | ✅ Oui |
| Confirmer le mot de passe | Ressaisissez votre mot de passe | ✅ Oui |
| Âge de l'enfant | Âge de votre enfant | ✅ Oui |
| Date de début souhaitée | Quand vous avez besoin de la garde | ✅ Oui |

### Étape 4 : Accepter les conditions et compléter le CAPTCHA

1. ☑️ Cochez la case pour accepter les **Conditions d'utilisation**
2. Complétez la **vérification CAPTCHA** pour prouver que vous êtes humain
3. Cliquez sur **"Créer un compte"**

### Étape 5 : Vérifiez votre email

Après avoir cliqué sur "Créer un compte", vous verrez l'écran de **Vérification d'email**.

#### Ce qui se passe :

1. Un **code de vérification à 6 chiffres** est envoyé à votre adresse email
2. Vérifiez votre boîte de réception (et le dossier spam/indésirables si nécessaire)
3. L'email provient de ProCrèche Solutions avec l'objet : "Vérifiez votre email"

#### Écran de vérification d'email :

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│              ✉️ Vérifiez votre email                │
│                                                     │
│  Nous avons envoyé un code de vérification à :     │
│  votre.email@exemple.com                            │
│                                                     │
│  Veuillez entrer le code ci-dessous :              │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │              [0] [0] [0] [0] [0] [0]        │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │            [ Vérifier l'email ]              │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  Vous n'avez pas reçu le code ?                    │
│  [Renvoyer le code] (disponible dans 30s)          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

#### Pour vérifier votre email :

1. Ouvrez votre email et trouvez l'email de vérification
2. Copiez le **code à 6 chiffres** de l'email
3. Entrez le code dans le champ de vérification
4. Cliquez sur **"Vérifier l'email"**

#### Renvoyer le code :

Si vous n'avez pas reçu l'email :

1. Attendez 30 secondes (un compte à rebours s'affiche)
2. Cliquez sur **"Renvoyer le code"**
3. Vérifiez à nouveau votre email
4. Vérifiez également votre dossier **Spam/Indésirables**

#### Code expiré ?

Les codes de vérification expirent après quelques minutes. Si votre code a expiré :

1. Cliquez sur **"Renvoyer le code"** pour en obtenir un nouveau
2. Utilisez le nouveau code pour vérifier

> 💡 **Conseil :** Le code est généralement valide pendant 10 minutes. Entrez-le dès que vous le recevez.

### Étape 6 : Configuration du compte terminée !

Après une vérification réussie :

1. Votre compte est créé
2. Vous verrez un message **"Succès !"**
3. Cliquez sur **"Aller au tableau de bord"** pour accéder à votre compte

> ⏳ **Veuillez patienter :** Le système prend quelques secondes pour configurer votre compte après la vérification. Vous verrez un indicateur de progression.

## Alternative : S'inscrire avec Google

Vous pouvez également créer un compte en utilisant votre compte Google :

### Étapes :

1. Sur la page d'inscription, cliquez sur **"Continuer avec Google"**
2. Sélectionnez votre compte Google
3. Accordez l'autorisation à ProCrèche Solutions
4. Vous serez redirigé pour compléter votre profil

### Compléter votre profil d'inscription Google :

Après l'authentification Google, vous devrez :

1. **Sélectionner votre rôle** (Fondation, Fournisseur, etc.)
2. **Remplir les informations requises** pour votre rôle
3. Accepter les Conditions d'utilisation
4. Compléter le CAPTCHA
5. Cliquer sur **"Compléter le profil"**

> 📧 **Note :** Votre adresse email Google sera utilisée et ne peut pas être modifiée lors de cette étape.

## Connexion

### Connexion standard

1. Accédez à la page de connexion
2. Entrez votre **adresse email**
3. Entrez votre **mot de passe**
4. Cliquez sur **"Se connecter"**

### Connexion avec Google

1. Cliquez sur **"Continuer avec Google"**
2. Sélectionnez votre compte Google
3. Vous serez connecté automatiquement

### Rester connecté

Votre session reste active pendant que vous naviguez sur la plateforme. Vous serez automatiquement déconnecté après une période d'inactivité pour des raisons de sécurité.

## Récupération d'un mot de passe oublié

### Étape 1 : Demander la réinitialisation du mot de passe

1. Sur la page de connexion, cliquez sur **"Mot de passe oublié ?"**
2. Entrez votre adresse email
3. Cliquez sur **"Envoyer le lien de réinitialisation"**

### Étape 2 : Vérifiez votre email

1. Cherchez un email de ProCrèche Solutions
2. Objet : "Réinitialisez votre mot de passe"
3. Cliquez sur le **lien de réinitialisation** dans l'email

### Étape 3 : Créer un nouveau mot de passe

1. Entrez votre **nouveau mot de passe** (minimum 8 caractères)
2. **Confirmez** votre nouveau mot de passe
3. Cliquez sur **"Réinitialiser le mot de passe"**

### Étape 4 : Connectez-vous

1. Vous serez redirigé vers la page de connexion
2. Connectez-vous avec votre nouveau mot de passe

> 💡 **Conseil :** Les liens de réinitialisation expirent après un court moment. Demandez-en un nouveau si le vôtre a expiré.

## Sécurité du compte

### Exigences de mot de passe

Votre mot de passe doit :
- Comporter au moins **8 caractères**
- Ne pas être un mot de passe couramment utilisé
- Ne pas avoir été exposé dans une fuite de données

### Changer votre mot de passe

1. Allez dans **Paramètres** → **Sécurité du compte**
2. Entrez votre **mot de passe actuel**
3. Entrez votre **nouveau mot de passe**
4. Confirmez votre nouveau mot de passe
5. Cliquez sur **"Mettre à jour le mot de passe"**

### Changer votre adresse email

1. Allez dans **Paramètres** → **Sécurité du compte**
2. Faites défiler jusqu'à **"Changer l'adresse email"**
3. Entrez votre nouvelle adresse email
4. Cliquez sur **"Envoyer le code de vérification"**
5. Vérifiez votre **nouvel email** pour un code de vérification
6. Entrez le code à 6 chiffres
7. Cliquez sur **"Vérifier et changer l'email"**

> ⚠️ **Important :** Vous devez vérifier votre nouvelle adresse email avant que le changement ne prenne effet.

### Bonnes pratiques

- ✅ Utilisez un mot de passe unique pour votre compte
- ✅ Ne partagez pas votre mot de passe avec d'autres
- ✅ Déconnectez-vous lorsque vous utilisez des ordinateurs partagés
- ✅ Gardez votre adresse email à jour

## Compléter votre profil

Après l'inscription, nous vous recommandons de compléter votre profil pour une meilleure expérience.

### Pourquoi compléter votre profil ?

- ✅ **Fondations :** Apparaître dans les recherches des parents, attirer les candidats
- ✅ **Fournisseurs/Prestataires :** Être découvert par des clients potentiels
- ✅ **Éducateurs :** Se démarquer auprès des employeurs
- ✅ **Parents :** Obtenir de meilleures correspondances de crèches

### Comment compléter votre profil

1. Cliquez sur votre **photo de profil** (en haut à droite)
2. Sélectionnez **"Modifier le profil"**
3. Remplissez toutes les sections
4. Téléchargez une photo de profil
5. Cliquez sur **"Enregistrer"**

> 📖 **En savoir plus :** Consultez le guide spécifique à votre rôle pour des instructions détaillées sur le profil.

## Dépannage

### Problèmes courants

| Problème | Solution |
|----------|----------|
| Email de vérification non reçu | Vérifiez le dossier spam, attendez 30s et renvoyez |
| Code de vérification expiré | Cliquez sur "Renvoyer le code" pour en obtenir un nouveau |
| Impossible de se connecter | Vérifiez email/mot de passe, utilisez "Mot de passe oublié" si nécessaire |
| Profil ne charge pas | Videz le cache du navigateur, essayez d'actualiser |
| Connexion Google ne fonctionne pas | Essayez la connexion standard email/mot de passe |
